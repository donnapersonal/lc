# 979.在二叉树中分配硬币

题目链接：[979.在二叉树中分配硬币](https://leetcode.cn/problems/distribute-coins-in-binary-tree/)

## 题目大意

给你一个有 n 个结点的二叉树的根结点 root ，其中树中每个结点 node 都对应有 node.val 枚硬币。整棵树上一共有 n 枚硬币

在一次移动中，我们可以选择两个相邻的结点，然后将一枚硬币从其中一个结点移动到另一个结点。移动可以是从父结点到子结点，或者从子结点移动到父结点

返回使每个结点上 只有 一枚硬币所需的 最少 移动次数

![example](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.54ydu3ip10.webp)

限制：
- The number of nodes in the tree is n.
- 1 <= n <= 100
- 0 <= Node.val <= n
- The sum of all Node.val is n.

## 解题

给你一棵二叉树，树上每个节点有若干个金币。要求最终每个节点都刚好有 1 个金币。每次操作可以把一个金币从某个节点移动到它的父节点或子节点，移动一条边算一次操作。要求返回最少操作次数

**核心思想：`后序遍历 + 子树金币盈亏`**

这题最关键的点是：对每一棵子树来说，我们不关心内部怎么调整，只关心它最后要向父节点“多给几个金币”或“向父节点要几个金币”

**一个子树有三种情况**

假设某个子树：
- 子树总金币数 = coins
- 子树总节点数 = nodes

那么：`balance = coins - nodes`

含义是：
- `balance > 0`	这棵子树金币多了，需要向父节点送出 balance 个金币
- `balance < 0`	这棵子树金币少了，需要从父节点拿 -balance 个金币
- `balance = 0`	这棵子树刚好平衡，不需要和父节点交换金币

**为什么用后序遍历？**

因为一个节点要知道自己这棵子树的金币盈亏，必须先知道：
- 左子树盈亏
- 右子树盈亏
- 当前节点金币数

所以遍历顺序应该是：左子树 -> 右子树 -> 当前节点

这就是 `后序遍历 Postorder DFS`

**公式**

对于当前节点 node：`d = dfs(node.left) + dfs(node.right) + node.val - 1`

含义：
- `dfs(node.left)`	左子树多出来或缺少的金币数
- `dfs(node.right)`	右子树多出来或缺少的金币数
- `node.val`	当前节点本身有多少金币
- `-1`	当前节点自己最终要保留 `1` 个金币

所以：d = 当前整棵子树的金币盈亏

为什么答案加 `abs(d)`？
- 如果当前子树多了 `d` 个金币：`d > 0`，说明它要把 `d` 个金币移动到父节点
- 如果当前子树少了 `-d` 个金币：`d < 0`，说明它要从父节点拿 `-d` 个金币

无论是送出去还是拿进来，都要经过当前子树根节点和父节点之间的那条边

所以这条边上的移动次数是：`abs(d)`

```js
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = val === undefined ? 0 : val;
 *     this.left = left === undefined ? null : left;
 *     this.right = right === undefined ? null : right;
 * }
 */

/**
 * @param {TreeNode} root
 * @return {number}
 */
var distributeCoins = function(root) {
    let res = 0;

    function dfs(node) {
        if (node === null) {
            return 0;
        }

        const d = dfs(node.left) + dfs(node.right) + node.val - 1;

        res += Math.abs(d);

        return d;
    }

    dfs(root);

    return res;
};
```

```python
# Definition for a binary tree node.
# class TreeNode:

class Solution:
    def distributeCoins(self, root: Optional[TreeNode]) -> int:
        res = 0

        def dfs(node: Optional[TreeNode]) -> int:
            if node is None:
                return 0

            d = dfs(node.left) + dfs(node.right) + node.val - 1

            nonlocal res
            res += abs(d)

            return d

        dfs(root)
        return res
```
```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    private int res = 0;

    public int distributeCoins(TreeNode root) {
        dfs(root);
        return res;
    }

    private int dfs(TreeNode node) {
        if (node == null) {
            return 0;
        }

        int d = dfs(node.left) + dfs(node.right) + node.val - 1;

        res += Math.abs(d);

        return d;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 为二叉树的节点个数。每个节点都会递归恰好一次
- 空间复杂度：`O(n)`。最坏情况下，二叉树是一条链，递归需要 `O(n)` 的栈空间
