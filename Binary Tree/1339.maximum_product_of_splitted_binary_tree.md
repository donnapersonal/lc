# 1339.分裂二叉树的最大乘积

题目链接：[1339.分裂二叉树的最大乘积](https://leetcode.cn/problems/maximum-product-of-splitted-binary-tree/)

## 题目大意

给你一棵二叉树，它的根为 root 。请你删除 1 条边，使二叉树分裂成两棵子树，且它们子树和的乘积尽可能大

由于答案可能会很大，请你将结果对 10^9 + 7 取模后再返回

![example](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.67y34n96c2.webp)

限制：
- The number of nodes in the tree is in the range [2, 5 * 10^4].
- 1 <= Node.val <= 10^4

## 解题

第一次 `DFS`：计算整棵树的点权和 `total`

第二次 `DFS`：计算子树点权和 `s`，那么删除当前节点到其父节点的边后，另一部分的和就是 `total−s`，二者乘积为 `s⋅(total−s)`

用其更新答案的最大值

由于本题保证点权是非负，无需判断当前节点是根节点的特殊情况（无父节点），此时上式为 `0`，不影响答案

- 如果有负数点权，就需要跳过当前节点是根节点的情况

```js
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = val === undefined ? 0 : val;
 *     this.left = left === undefined ? null : left;
 *     this.right = right === undefined ? null : right;
 * }
 */

/**
 * @param {TreeNode} root
 * @return {number}
 */
var maxProduct = function(root) {
    const MOD = 1000000007n;

    function dfs1(node) {
        if (node === null) {
            return 0n;
        }

        return BigInt(node.val) + dfs1(node.left) + dfs1(node.right);
    }

    function dfs2(node) {
        if (node === null) {
            return 0n;
        }

        const s = BigInt(node.val) + dfs2(node.left) + dfs2(node.right);
        res = res > s * (total - s) ? res : s * (total - s);

        return s;
    }

    const total = dfs1(root);

    let res = 0n;
    dfs2(root);

    return Number(res % MOD);
};
```

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxProduct(self, root: Optional[TreeNode]) -> int:
        def dfs1(node: Optional[TreeNode]) -> int:
            if node is None:
                return 0
            return node.val + dfs1(node.left) + dfs1(node.right)

        def dfs2(node: Optional[TreeNode]) -> int:
            if node is None:
                return 0
            s = node.val + dfs2(node.left) + dfs2(node.right)
            nonlocal res
            res = max(res, s * (total - s))
            return s

        total = dfs1(root)

        res = 0
        dfs2(root)

        return res % 1_000_000_007
```
```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */

class Solution {
    private long total;
    private long res;
    private static final int MOD = 1_000_000_007;

    public int maxProduct(TreeNode root) {
        total = dfs1(root);

        res = 0;
        dfs2(root);

        return (int)(res % MOD);
    }

    private long dfs1(TreeNode node) {
        if (node == null) {
            return 0;
        }

        return node.val + dfs1(node.left) + dfs1(node.right);
    }

    private long dfs2(TreeNode node) {
        if (node == null) {
            return 0;
        }

        long s = node.val + dfs2(node.left) + dfs2(node.right);

        res = Math.max(res, s * (total - s));

        return s;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 是二叉树的节点个数
- 空间复杂度：`O(h)`，其中 `h` 是二叉树的高度。递归需要消耗 `O(h)` 的栈空间
