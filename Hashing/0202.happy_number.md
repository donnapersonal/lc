## 解题

### HashSet 循环检测

**我们需要维护什么？**

我们需要知道：我们之前见过这个数字吗？

因此，使用：`HashSet`

如果某个数字再次出现，说明找到了一个循环

有两种常见的优化方案：
- HashSet 循环检测
- Floyd 快慢指针法

在面试中，**HashSet 方案**通常是首选解法，因为它简单、清晰且易于解释


- 创建一个名为 `seen` 的集合
- 当 `n` 不等于 `1` 且未曾出现过时：
  - 将 `n` 加入 `seen`
  - 将 `n` 替换为其各位数字的平方和
- 如果 `n == 1`，返回 `true`
- 否则，返回 `false`

在每次循环迭代之前：`seen` 包含了转换序列中之前出现过的所有数字。如果当前的 `n` 已经存在于 `seen` 中，说明序列已进入循环

```python
class Solution:
    def isHappy(self, n: int) -> bool:
        seen = set()
        while n != 1 and n not in seen:
            seen.add(n)
            n = self.get_Sum(n)

        return n == 1
    
    def get_Sum(self, n: int) -> int:
        total_sum = 0
        while n > 0:
            n, digit = divmod(n, 10)
            total_sum += digit * digit
        
        return total_sum
```
```java
class Solution {
    public boolean isHappy(int n) {
        Set<Integer> seen = new HashSet<>();
        while (n != 1 && !seen.contains(n)) {
            seen.add(n);
            n = getSum(n);
        }
        return n == 1;
    }

    private int getSum(int n) {
        int totalSum = 0;
        while (n > 0) {
            int digit = n % 10;
            totalSum += digit * digit;
            n /= 10;
        }
        return totalSum;
    }
}
```

### Floyd's Cycle Detection

将每个数字视为一个节点。下一个节点为：getSum(当前数字)

于是序列如下：

```
n -> getSum(n) -> getSum(getSum(n)) -> ...
```

如果存在循环，快慢指针最终会相遇。如果快指针到达了 `1`，则该数字是快乐数

```python
class Solution:
    def isHappy(self, n: int) -> bool:
        slow, fast = n, n
        while self.get_sum(fast) != 1 and self.get_sum(self.get_sum(fast)) != 1:
            slow = self.get_sum(slow)
            fast = self.get_sum(self.get_sum(fast))
            # 若存在一个循环（即这个数不是快乐数），快指针和慢指针最终会相遇（即 slow == fast），这就像在链表中检测环一样
            if slow == fast:
                return False
        return True
    
    def get_sum(self, n: int) -> int: 
        total_sum = 0
        while n > 0:
            n, digit = divmod(n, 10)
            total_sum += digit ** 2
        
        return total_sum
```
```java
class Solution {
    public boolean isHappy(int n) {
        int slow = n;
        int fast = getSum(n);

        while (fast != 1 && slow != fast) {
            slow = getSum(slow);
            fast = getSum(getSum(fast));
        }

        return fast == 1;
    }

    private int getSum(int n) {
        int totalSum = 0;

        while (n > 0) {
            int digit = n % 10;
            totalSum += digit * digit;
            n /= 10;
        }

        return totalSum;
    }
}
```

设：d = n 的位数，每次调用 `getSum(n)` 的开销为：O(d)

**HashSet 版本**

- 时间复杂度：`O(k * d)`
  - 其中 `k` 是在达到 `1` 或进入循环之前所访问的唯一数字的数量
  - 在实际应用中，这个值很小，因为数值很快就会被限制在一定范围内

- 空间复杂度：`O(k)`，因为我们将访问过的数字存储在 HashSet 中

**Floyd 版本**
- 时间复杂度：`O(k * d)`
- 空间复杂度：`O(1)`，因为它不存储所有访问过的数字
