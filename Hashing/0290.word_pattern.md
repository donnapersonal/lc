# 290.Word Pattern

难度: Easy
标签: String、Hashing

![290](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.5trp279567.webp)

限制：
- 1 <= pattern.length <= 300
- pattern contains only lower-case English letters.
- 1 <= s.length <= 3000
- s contains only lowercase English letters and spaces ' '.
- s does not contain any leading or trailing spaces.
- All the words in s are separated by a single space.

## 解题

### 暴力

首先将 `s` 拆分为单词。然后，对于每一对位置 `i` 和 `j`：
- 如果 `pattern[i] == pattern[j]`，则 `words[i]` 必须等于 `words[j]`
- 如果 `words[i] == words[j]`，则 `pattern[i]` 必须等于 `pattern[j]`

这用于检查两个序列是否具有相同的模式

```python
class Solution:
    def wordPattern(self, pattern: str, s: str) -> bool:
        words = s.split()

        if len(pattern) != len(words):
            return False

        n = len(pattern)

        for i in range(n):
            for j in range(n):
                if pattern[i] == pattern[j] and words[i] != words[j]:
                    return False

                if words[i] == words[j] and pattern[i] != pattern[j]:
                    return False

        return True
```
设：n = pattern.length()

- 时间复杂度：`O(n²)`
- 空间复杂度：`O(n)`

因为将字符串 `s` 拆分成了单词数组，暴力解法会比较每一对位置

示例：

```
pattern = "abba"
s = "dog cat cat dog"
```

它会反复检查不同位置处的字符模式与单词模式是否一致

这种做法是不必要的

### 最优解：bidirectional HashMap

**我们需要维护什么？**

在从左向右遍历的过程中，我们只需关注以下两点：
- 该模式字符是否已映射到某个单词？
- 该单词是否已映射到某个模式字符？

因此，我们使用：

```
Map<Character, String> charToWord
Map<String, Character> wordToChar
```

将字符串 `s` 拆分为单词数组。随后，同步遍历 `pattern` 和 `words`

在每个索引位置：

```
ch = pattern[i]
word = words[i]
```

检查以下两个条件：

- 如果 `ch` 已经映射过单词，则它必须映射到当前的 `word`
- 如果 `word` 已经映射过字符，则它必须映射到当前的 `ch`

若任一方向出现冲突，则返回 `false`。否则，保存这两项映射关系

处理完索引 `i` 之后：**`pattern[0...i]`** 与 **`words[0...i]`** 之间的所有映射关系均保持一致且为一一对应

```python
class Solution:
    def wordPattern(self, pattern: str, s: str) -> bool:
        words = s.split()
        w_2_p = dict()
        p_2_w = dict()

        if len(pattern) != len(words):
            return False
        
        for ch, word in zip(pattern, words):
            if ch in p_2_w and p_2_w[ch] != word:
                return False
            
            if word in w_2_p and w_2_p[word] != ch:
                return False
            
            w_2_p[word] = ch
            p_2_w[ch] = word
        
        return True
```
```java
class Solution {
    public boolean wordPattern(String pattern, String s) {
        String[] words = s.split(" ");
        Map<String, Character> w_2_p = new HashMap<>();
        Map<Character, String> p_2_w = new HashMap<>();
        if (pattern.length() != words.length) {
            return false;
        }
        for (int i = 0; i < pattern.length(); i++) {
            char ch = pattern.charAt(i);
            String word = words[i];

            if (p_2_w.containsKey(ch) && !p_2_w.get(ch).equals(word)) {
                return false;
            }

            if (w_2_p.containsKey(word) && w_2_p.get(word) != ch) {
                return false;
            }

            w_2_p.put(word, ch);
            p_2_w.put(ch, word);
        }
        return true;
    }
}
```

- 时间复杂度：`O(L + n)`，n = pattern.length()，L = s.length()
  - 拆分“s”需要“O(L)”
  - 然后扫描单词和模式需要“O(n)”
- 空间复杂度：`O(n)`
  - 因为在最坏的情况下，每个模式字符和每个单词都是唯一的
