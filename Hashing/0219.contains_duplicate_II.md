# 219.Contains Duplicate II

难度: Easy
标签: Array、Hashing

![219](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.lwegp0378.webp)

限制：
- 1 <= nums.length <= 10^5
- -10^9 <= nums[i] <= 10^9
- 0 <= k <= 10^5

## 解题

### 暴力

检查每一对下标 i 和 j。如果 nums[i] == nums[j] 且 j - i <= k，则返回 true。否则，返回 false

暴力解法检查了许多不必要的数对。例如：

```
nums = [1, 2, 3, 1]
k = 3
```

当我们遍历到索引 `3` 处的第二个 `1` 时，我们只需要知道：上一次出现 1 是在什么位置？

如果上一次出现的索引是 `0`，那么：3 - 0 = 3 <= k

因此，我们可以返回 `true`

```python
from typing import List


class Solution:
    def containsNearbyDuplicate(self, nums: List[int], k: int) -> bool:
        n = len(nums)

        for i in range(n):
            for j in range(i + 1, n):
                if nums[i] == nums[j] and j - i <= k:
                    return True

        return False
```

设：n = nums.length

- 时间复杂度：`O(n²)`
- 空间复杂度：`O(1)`

### HashSet

**我们需要维护什么？**

我们需要维护：`数值 -> 最近一次出现的索引`

因此，我们使用 `HashMap`

**核心思路**

从左到右遍历数组 `nums`。对于每个索引 `i`：

- 令 `num = nums[i]`。
- 如果 `num` 之前出现过，获取其上一次出现的索引
- 检查是否满足：`i - lastSeen[num] <= k`
- 如果满足，返回 `true`。否则，更新 `num` 的最新索引


在处理索引 `i` 之前：last_seen 存储了迄今为止每个数字最近一次出现的索引

为什么要存储最近一次出现的索引？

- 因为最近一次出现的位置能提供最小的距离。如果连最近一次出现的位置都超出了距离 `k`，那么更早出现的位置肯定也超出了距离 `k`

```python
class Solution:
    def containsNearbyDuplicate(self, nums: List[int], k: int) -> bool:
        last_seen = {}
        for i, num in enumerate(nums):
            if num in last_seen and i - last_seen[num] <= k:
                return True
            last_seen[num] = i
        
        return False
        
# sliding window + hashset
class Solution:
    def containsNearbyDuplicate(self, nums: List[int], k: int) -> bool:
        left, right = 0, 0
        window = set()
        while right < len(nums):
            if nums[right] in window:
                return True
                
            window.add(nums[right])
            right += 1

            if right - left > k:
                window.remove(nums[left])
                left += 1
        
        return False
```
```java
class Solution {
    public boolean containsNearbyDuplicate(int[] nums, int k) {
        Map<Integer, Integer> lastSeen = new HashMap<>();
        for (int i = 0; i < nums.length; i++) {
            int num = nums[i];
            if (lastSeen.containsKey(num) && i - lastSeen.get(num) <= k) {
                return true;
            }
            lastSeen.put(num, i);
        }
        return false;
    }
}

// sliding window + hashset
class Solution {
    public boolean containsNearbyDuplicate(int[] nums, int k) {
        int left = 0;
        int right = 0;
        Set<Integer> window = new HashSet<>();
        while (right < nums.length) {
            if (window.contains(nums[right])) {
                return true;
            }
            window.add(nums[right]);
            right++;
            if (right - left > k) {
                window.remove(nums[left]);
                left++;
            } 
        }
        return false;
    }
}
```

设：n = nums.length

**HashMap 版本**

- 时间复杂度：`O(n)`。每个数字都被处理一次
- 空间复杂度：`O(n)`。最坏情况下，所有数字各不相同

**滑动窗口 HashSet 版本**

- 时间复杂度：`O(n)`。每个数字都被处理一次
- 空间复杂度：`O(min(n, k))`。窗口内最多存储 `k` 个数字
