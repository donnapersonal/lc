# 249.移位字符串分组

题目链接：[249.移位字符串分组](https://leetcode.cn/problems/group-shifted-strings/)

## 题目大意

对字符串进行 “移位” 的操作：
- 右移：将字符串中每个字母都变为其在字母表中 后续 的字母，其中用 'a' 替换 'z'。比如，"abc" 能够右移为 "bcd"，"xyz" 能够右移为 "yza"
- 左移：将字符串中每个字母都变为其在字母表中 之前 的字母，其中用 'z' 替换 'a'。比如，"bcd" 能够左移为 "abc"，"yza" 能够左移为 "xyz"
  
我们可以不断地向两个方向移动字符串，形成 无限的移位序列
- 例如，移动 "abc" 来形成序列：... <-> "abc" <-> "bcd" <-> ... <-> "xyz" <-> "yza" <-> ... <-> "zab" <-> "abc" <-> ...
  
给定一个字符串数组 strings，将属于相同移位序列的所有 strings[i] 进行分组。你可以以 任意顺序 返回答案。

```js
Example 1:
Input: strings = ["abc","bcd","acef","xyz","az","ba","a","z"]
Output: [["acef"],["a","z"],["abc","bcd","xyz"],["az","ba"]]

Example 2:
Input: strings = ["a"]
Output: [["a"]]
```

限制：
- 1 <= strings.length <= 200
- 1 <= strings[i].length <= 50
- strings[i] consists of lowercase English letters.

## 解题

**哈希表记录分组**

定义字符串中的一对相邻字符的差为：后面的字符减前面的字符得到差，如果这个差小于 0，则将这个差加上 26，这样确保差的范围在 0 到 25。例如，相邻字符 'az' 以及相邻字符 'ba' 的差都为 25

对于任意一个字符串，对该字符串进行任意次“移位”操作后，相邻字符的差不变。因此，可以将每对相邻字符的差作为字符的分组

使用一个哈希表记录每个分组包含的字符串，哈希表的每个键值对中，键为分组，值为该分组包含的字符串列表。对于给定的字符串列表中的每个字符串计算其分组，并将当前字符串加到哈希表中相应的分组的列表中

由于分组由每对相邻字符的差表示，一般想到的是用数组表示分组，但是数组并不适合作为哈希表的键，一个办法是将数组转换成字符串以后作为哈希表的键

```js
/**
 * @param {string[]} strings
 * @return {string[][]}
 */
var groupStrings = function(strings) {
    const groups = new Map();

    function getKey(s) {
        const key = [];

        for (let i = 1; i < s.length; i++) {
            const diff = (s.charCodeAt(i) - s.charCodeAt(i - 1) + 26) % 26;
            key.push(diff);
        }

        return key.join(",");
    }

    for (const s of strings) {
        const key = getKey(s);

        if (!groups.has(key)) {
            groups.set(key, []);
        }

        groups.get(key).push(s);
    }

    return Array.from(groups.values());
};
```

class Solution:
    def groupStrings(self, strings: List[str]) -> List[List[str]]:
        def get_key(s: str) -> tuple:
            key = []

            for i in range(1, len(s)):
                diff = (ord(s[i]) - ord(s[i - 1])) % 26
                key.append(diff)

            return tuple(key)

        groups = defaultdict(list)

        for s in strings:
            key = get_key(s)
            groups[key].append(s)

        return list(groups.values())

```java
class Solution {
    public List<List<String>> groupStrings(String[] strings) {
        Map<String, List<String>> groups = new HashMap<>();

        for (String s : strings) {
            String key = getKey(s);

            groups.putIfAbsent(key, new ArrayList<>());
            groups.get(key).add(s);
        }

        return new ArrayList<>(groups.values());
    }

    private String getKey(String s) {
        StringBuilder key = new StringBuilder();

        for (int i = 1; i < s.length(); i++) {
            int diff = (s.charAt(i) - s.charAt(i - 1) + 26) % 26;
            key.append(diff).append(",");
        }

        return key.toString();
    }
}
```

- 时间复杂度：`O(nk)`
  - 假设给定的字符串列表包含 `n` 个字符串，其中最长的字符串长度为 `k`
  - 字符串列表遍历一遍，对于每个字符串计算分组的操作为 `O(k)` 时间复杂度，更新哈希表的操作为 `O(1)` 时间复杂度
  - 遍历哈希表时，由于哈希表的键值对数量不超过 `n`，因此时间复杂度为 `O(n)`
- 空间复杂度：`O(n)`。哈希表中每个字符串被存储一次
