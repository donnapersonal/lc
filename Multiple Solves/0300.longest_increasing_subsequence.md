# 300.最长递增子序列

题目链接：[300.最长递增子序列](https://leetcode.cn/problems/longest-increasing-subsequence/)

## 题目大意

给你一个整数数组 nums，找到其中最长严格递增子序列的长度

子序列 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，[3,6,2,7] 是数组 [0,3,1,6,2,2,7] 的子序列

```js
Example 1:
Input: nums = [10,9,2,5,3,7,101,18]
Output: 4
Explanation: The longest increasing subsequence is [2,3,7,101], therefore the length is 4.

Example 2:
Input: nums = [0,1,0,3,2,3]
Output: 4

Example 3:
Input: nums = [7,7,7,7,7,7,7]
Output: 1
```

限制：
- 1 <= nums.length <= 2500
- -10^4 <= nums[i] <= 10^4

## 解题

### 思路 1: DP
```python
# 思路 1: DP - 写法 1
class Solution:
    # Defines the method lengthOfLIS which takes a list of integers and returns the length of the longest strictly increasing subsequence.
    def lengthOfLIS(self, nums: List[int]) -> int:
        n = len(nums)
        # Initializes a list dp of the same length as nums, with all elements set to 1.
        # dp[i] represents the length of the longest increasing subsequence ending at index i.
        # Every element is initially at least a subsequence of length 1 (by itself).
        dp = [1] * n
        
        # Outer loop: iterates over each position i in the array.
        # We are trying to compute the best dp[i] by looking at all j < i.
        for i in range(n):
            # Inner loop: looks at all previous elements before i.
            # We’re trying to find a smaller number nums[j] such that we can append nums[i] to the subsequence ending at j.
            for j in range(i):
                # if nums[i] > nums[j], then nums[i] can extend the increasing subsequence that ends at nums[j].
                if nums[i] > nums[j]:
                    # We update dp[i] by taking the maximum of:
                    # its current value dp[i]
                    # the length of the LIS ending at j, plus 1 (because we’re appending nums[i])
                    # This ensures that we always store the longest increasing subsequence ending at i.
                    dp[i] = max(dp[i], dp[j] + 1)
        
        # Initialize a variable res to keep track of the overall longest increasing subsequence found in the entire array.
        res = 0
        # After filling the dp array, we find the maximum value in it, which represents the LIS length.
        for i in range(len(dp)):
            # Return the final result — the length of the longest increasing subsequence.
            res = max(res, dp[i])
        
        return res

class Solution:
    # Defines the method lengthOfLIS, which takes an array of integers nums and returns an integer — the length of the longest increasing subsequence.
    def lengthOfLIS(self, nums: List[int]) -> int:
        n = len(nums)
        # Initializes a DP array dp[] of the same length as nums.
        # dp[i] represents the length of the LIS ending at index i.
        # Each element is initialized to 1 because the smallest possible LIS for any element is itself (length 1).
        dp = [1] * n
        # Initializes a variable res to track the maximum LIS length found so far.
        # Starts from 1 since every number on its own forms an increasing subsequence of length 1.
        res = 1
        
        # Starts iterating from the second element (index 1) to the end of the array.
        # For each i, we want to find the LIS ending at i.
        for i in range(1, len(dp)):
            # Inner loop goes through all elements j before i.
            # We're checking: can nums[i] extend any increasing subsequence that ends at j?
            for j in range(i):
                # Key condition: If nums[i] > nums[j], we can append nums[i] to the subsequence ending at j to create a new increasing subsequence.
                if nums[i] > nums[j]:
                    # Updates dp[i] by choosing the best among:
                    # Keeping dp[i] as is (maybe from an earlier j)
                    # Taking the LIS length ending at j, and extending it with nums[i]
                    # This ensures dp[i] always stores the longest subsequence ending at i.
                    dp[i] = max(dp[i], dp[j] + 1)
                
                # After possibly updating dp[i], we check if it's the largest we've seen.
                # res keeps track of the global maximum LIS length across all positions.
                # Note: This line could technically be moved outside the inner for j loop to avoid redundant checks, but the logic is still correct.
                res = max(res, dp[i])
        
        # Finally, return the global maximum found — the length of the longest increasing subsequence     
        return res
```
- 时间复杂度：`O(n^2)`
- 空间复杂度：`O(n)`

### 思路 2: 二分查找

class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        # 每个单独的元素本身就可看作是长度为 1 的递增子序列
        dp = [1] * len(nums)
        res = 1
        for i in range(1, len(dp)):
            for j in range(i):
                # 如果 nums[i] > nums[j]，表示 nums[i] 可以接在 nums[j] 之后
                if nums[i] > nums[j]:
                    # dp[i] 的值要么是 dp[i] 原来的值
                    # 要么是 dp[j] + 1 的值（即 nums[i] 接在 nums[j] 后面的最长递增子序列长度）
                    dp[i] = max(dp[i], dp[j] + 1)
                res = max(res, dp[i])
        
        return res

class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        res = [] 
        for n in nums:
            if not res or n > res[-1]:
                res.append(n)
            else:
                l, r = 0, len(res) - 1
                loc = r
                while l <= r:
                    mid = (l + r) // 2
                    if res[mid] >= n:
                        loc = mid
                        r = mid - 1
                    else:
                        l = mid + 1
                
                res[loc] = n
        
        return len(res)

```java
// 写法 3
class Solution {
    public int lengthOfLIS(int[] nums) {
        List<Integer> res = new ArrayList<>();

        for (int num : nums) {
            int left = 0;
            int right = res.size() - 1;
            int pos = res.size();

            while (left <= right) {
                int mid = left + (right - left) / 2;

                if (res.get(mid) >= num) {
                    pos = mid;
                    right = mid - 1;
                } else {
                    left = mid + 1;
                }
            }

            if (pos == res.size()) {
                res.add(num);
            } else {
                res.set(pos, num);
            }
        }

        return res.size();
    }
}
```

- 时间复杂度：`O(nlogn)`
- 空间复杂度：`O(n)`
