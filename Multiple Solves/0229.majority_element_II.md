# 229.多数元素 II

题目链接：[229.多数元素 II](https://leetcode.cn/problems/majority-element-ii/)

## 题目大意

给定一个大小为 n 的整数数组，找出其中所有出现超过 ⌊ n/3 ⌋ 次的元素

```js
Example 1:
Input: nums = [3,2,3]
Output: [3]

Example 2:
Input: nums = [1]
Output: [1]

Example 3:
Input: nums = [1,2]
Output: [1,2]
```

限制：
- 1 <= nums.length <= 5 * 10^4
- -10^9 <= nums[i] <= 10^9

## 解题

### 思路 1: 哈希计数

```python
class Solution:
    def majorityElement(self, nums: List[int]) -> List[int]:
        cnt = {}
        ans = []

        for v in nums:
            if v in cnt:
                cnt[v] += 1
            else:
                cnt[v] = 1
        for item in cnt.keys():
            if cnt[item] > len(nums)//3:
                ans.append(item)

        return ans
```

- 时间复杂度：`O(n)`，其中 `n` 为数组的长度
- 空间复杂度：`O(n)`，其中 `n` 为数组的长度，使用哈希表需要开辟额外的空间

### 思路 2: 摩尔投票

**关键观察：答案最多只有 2 个**

因为题目要求：出现次数 > n / 3

假设有 3 个不同的数都超过 n / 3，那么它们总出现次数会：> n / 3 + n / 3 + n / 3 = n

这会超过数组总长度 n，不可能

所以：`超过 n / 3 的元素最多只有 2 个`

因此我们只需要维护两个候选人。

**摩尔投票法核心思想**

这题可以理解成一个“抵消游戏”

维护两个候选人：cand1, cand2

以及它们的票数：count1, count2

遍历数组时：

| 情况            | 操作                          |
| ------------- | --------------------------- |
| 当前数等于 `cand1` | `count1 += 1`               |
| 当前数等于 `cand2` | `count2 += 1`               |
| `count1 == 0` | 当前数成为 `cand1`               |
| `count2 == 0` | 当前数成为 `cand2`               |
| 当前数和两个候选人都不同  | `count1 -= 1`，`count2 -= 1` |

**为什么可以抵消？**

因为我们要找的是出现次数超过 n / 3 的元素

当遇到三个不同的数时：候选人1、候选人2、当前数

它们可以互相抵消一次

原因是：真正超过 n / 3 的元素，即使被抵消，也不会被完全抵消掉

换句话说：如果一个数字真的出现非常多次，它最终一定有机会留下来成为候选人

**为什么需要第二次遍历？**

第一次遍历结束后，留下来的 cand1 和 cand2 只是“可能答案”

它们不一定真的超过 n / 3

例如：nums = [1, 2, 3, 4]

最后也可能留下两个候选人，但没有任何元素出现次数超过 n / 3

所以必须第二次遍历，重新统计真实次数

```js
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var majorityElement = function(nums) {
    let cand1 = null;
    let cand2 = null;
    let count1 = 0;
    let count2 = 0;

    // Phase 1: find possible candidates
    for (const num of nums) {
        if (cand1 === num) {
            count1++;
        } else if (cand2 === num) {
            count2++;
        } else if (count1 === 0) {
            cand1 = num;
            count1 = 1;
        } else if (count2 === 0) {
            cand2 = num;
            count2 = 1;
        } else {
            count1--;
            count2--;
        }
    }

    // Phase 2: verify candidates
    count1 = 0;
    count2 = 0;

    for (const num of nums) {
        if (num === cand1) {
            count1++;
        } else if (num === cand2) {
            count2++;
        }
    }

    const res = [];
    const n = nums.length;

    if (count1 > Math.floor(n / 3)) {
        res.push(cand1);
    }

    if (count2 > Math.floor(n / 3)) {
        res.push(cand2);
    }

    return res;
};
```

```java
import java.util.*;

class Solution {
    public List<Integer> majorityElement(int[] nums) {
        Integer cand1 = null;
        Integer cand2 = null;
        int count1 = 0;
        int count2 = 0;

        // Phase 1: find possible candidates
        for (int num : nums) {
            if (cand1 != null && cand1 == num) {
                count1++;
            } else if (cand2 != null && cand2 == num) {
                count2++;
            } else if (count1 == 0) {
                cand1 = num;
                count1 = 1;
            } else if (count2 == 0) {
                cand2 = num;
                count2 = 1;
            } else {
                count1--;
                count2--;
            }
        }

        // Phase 2: verify candidates
        count1 = 0;
        count2 = 0;

        for (int num : nums) {
            if (cand1 != null && cand1 == num) {
                count1++;
            } else if (cand2 != null && cand2 == num) {
                count2++;
            }
        }

        List<Integer> res = new ArrayList<>();
        int n = nums.length;

        if (count1 > n / 3) {
            res.add(cand1);
        }

        if (count2 > n / 3) {
            res.add(cand2);
        }

        return res;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 为数组的长度
- 空间复杂度：`O(1)`，只需要常数个元素用来存储关键元素和统计次数即可
