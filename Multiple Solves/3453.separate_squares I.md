# 3453.分割正方形 I

题目链接：[3453.分割正方形 I](https://leetcode.cn/problems/separate-squares-i/)

## 题目大意

给你一个二维整数数组 squares ，其中 squares[i] = [xi, yi, li] 表示一个与 x 轴平行的正方形的左下角坐标和正方形的边长

找到一个最小的 y 坐标，它对应一条水平线，该线需要满足它以上正方形的总面积 等于 该线以下正方形的总面积

答案如果与实际答案的误差在 10-5 以内，将视为正确答案

注意：正方形 可能会 重叠。重叠区域应该被 多次计数

![example2](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.5j4rvf3mn5.webp)

限制：
- 1 <= squares.length <= 5 * 10^4
- squares[i] = [xi, yi, li]
- squares[i].length == 3
- 0 <= xi, yi <= 10^9
- 1 <= li <= 10^9
- The total area of all the squares will not exceed 10^12.

## 解题

### 思路 1: 二分

```js
/**
 * @param {number[][]} squares
 * @return {number}
 */
var separateSquares = function(squares) {
    const totalArea = squares.reduce((sum, [, , l]) => sum + l * l, 0);

    const check = (y) => {
      let area = 0;
      for (const [, yi, l] of squares) {
        if (yi < y) {
          area += l * Math.min(y - yi, l);
        }
      }
      return area >= totalArea / 2;
    };

    let left = 0;
    let right = 0;
    for (const [, y, l] of squares) {
      right = Math.max(right, y + l);
    }

    for (let i = 0; i < 60; i++) {
      const mid = (left + right) / 2;
      if (check(mid)) {
        right = mid;
      } else {
        left = mid;
      }
    }

    return (left + right) / 2;
};
```
```python
class Solution:
    def separateSquares(self, squares: List[List[int]]) -> float:
        M = 100_000
        total_area = sum(l * l for _, _, l in squares)

        def check(y: float) -> bool:
            area = 0
            for _, yi, l in squares:
                if yi < y:
                    area += l * min(y - yi, l)
            return area >= total_area / 2

        left = 0
        right = max_y = max(y + l for _, y, l in squares)
        for _ in range((max_y * M).bit_length()):
            mid = (left + right) / 2
            if check(mid):
                right = mid
            else:
                left = mid
        return (left + right) / 2  # 区间中点误差小
```
```java
class Solution {
    public double separateSquares(int[][] squares) {
        long totalArea = 0;
        double left = 0.0;
        double right = 0.0;

        for (int[] sq : squares) {
            int y = sq[1];
            int l = sq[2];
            totalArea += 1L * l * l;
            right = Math.max(right, y + l);
        }

        for (int iter = 0; iter < 60; iter++) {
            double mid = left + (right - left) / 2.0;
            if (check(squares, mid, totalArea)) {
                right = mid;
            } else {
                left = mid;
            }
        }

        return (left + right) / 2.0;
    }

    private boolean check(int[][] squares, double cutY, long totalArea) {
        double area = 0.0;

        for (int[] sq : squares) {
            int y = sq[1];
            int l = sq[2];

            if (cutY > y) {
                area += l * Math.min(cutY - y, l);
            }
        }

        return area * 2.0 >= totalArea;
    }
}
```

- 时间复杂度：`O(nlog(U/ϵ))`，其中 `n` 是 `squares` 的长度，`ϵ=10^−5`，`U=max(yi + li)`
- 空间复杂度：`O(1)`

### 思路 2: 差分 + 扫描线

想象有一根水平扫描线在从下往上扫描，对于示例 2，这根扫描线依次扫过 `y=0,1,2`：
- 从 `y=0` 到 `y=1`，面积的增加量可以视作一个底边长为 `2`，高为 `1` 的矩形的面积，即 `2⋅1=2`
- 从 `y=1` 到 `y=2`，面积的增加量可以视作一个底边长为 `2+1=3`（重叠的要累加），高为 `1` 的矩形的面积，即 `3⋅1=3`
- 扫描的过程中，维护面积之和 `area`，底边长之和 `sumL`（可以用`差分维护`）

设当前 `y` 与下一个 `y′` 之差为 `y′−y`，则新增面积为 `sumL⋅(y′−y)`

如果发现 `area⋅2 ≥ totalArea`，则可以直接算出答案，计算公式和上面「方法二：整数二分（写法二）」是一样的

```python
class Solution:
    def separateSquares(self, squares: List[List[int]]) -> float:
        tot_area = 0
        diff = defaultdict(int)
        for _, y, l in squares:
            tot_area += l * l
            diff[y] += l
            diff[y + l] -= l

        area = sum_l = 0
        for y, y2 in pairwise(sorted(diff)):
            sum_l += diff[y]  # 矩形底边长度之和
            area += sum_l * (y2 - y)  # 底边长 * 高 = 新增面积
            if area * 2 >= tot_area:
                return y2 - (area * 2 - tot_area) / (sum_l * 2)
```

- 时间复杂度：`O(nlogn)`，其中 `n` 是 `squares` 的长度。瓶颈在排序/维护有序集合上
- 空间复杂度：`O(n)`
