# 465.最优账单平衡

题目链接：[465.最优账单平衡](https://leetcode.cn/problems/optimal-account-balancing/)

## 题目大意

给你一个表示交易的数组 transactions ，其中 transactions[i] = [fromi, toi, amounti] 表示 ID = fromi 的人给 ID = toi 的人共计 amounti $ 

请你计算并返回还清所有债务的最小交易笔数

```js
Example 1:
Input: transactions = [[0,1,10],[2,0,5]]
Output: 2
Explanation:
Person #0 gave person #1 $10.
Person #2 gave person #0 $5.
Two transactions are needed. One way to settle the debt is person #1 pays person #0 and #2 $5 each.

Example 2:
Input: transactions = [[0,1,10],[1,0,1],[1,2,5],[2,0,5]]
Output: 1
Explanation:
Person #0 gave person #1 $10.
Person #1 gave person #0 $1.
Person #1 gave person #2 $5.
Person #2 gave person #0 $5.
Therefore, person #1 only need to give person #0 $4, and all debt is settled.
```

限制：
- 1 <= transactions.length <= 8
- transactions[i].length == 3
- 0 <= fromi, toi < 12
- fromi != toi
- 1 <= amounti <= 100

## 解题

### 思路 1: 回溯

定义递归函数 dfs(cur)，表示结清列表中区间 [cur:] 内所有人的债务所需的最少交易数。例如，dfs(0) 表示结清所有债务所需的最少交易数。dfs(1) 表示结清除第一个人以外所有人的债务所需的最少交易数

如图所示，当前人 cur 的净余额为 3，我们从后续位置 cur+1 开始遍历每个人 nxt，尝试把 cur 的全部债务转移给 nxt。转移后，cur 的债务被清零，交易次数增加 1。随后递归处理下一个人 cur+1。换句话说，dfs(cur)=1+dfs(cur+1)

![alt text1](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.3627xhg5zf.png)

不过，如果人 cur 的余额为零，说明他的债务已经结清，可以不需要任何交易，直接调用 dfs(cur+1) 处理下一个人：dfs(cur)=dfs(cur+1)

![alt text2](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.5tro7ua1ie.webp)

还可以进一步优化算法：只尝试把 cur 的债务转移给那些债务非零且符号与 cur 的债务相反的人。例如，如果 cur 的净余额为正，则只考虑净余额为负的人，反之亦然

![alt text3](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.3ns9m2izt0.webp)

算法步骤

- 创建哈希表，存储每个人的净余额
- 将所有非零净余额收集到数组 balance_list 中
- 定义递归函数 dfs(cur)，用于清除 balance_list[0∼cur] 范围内的所有余额
- 如果 cur 的余额已经是 0，则忽略它。当 balance_list[cur]=0 时，不断将 cur 加 1，继续处理下一个人
  - 如果 cur=n，返回 0
  - 否则，将 cost 设为一个很大的整数，例如 inf
- 从 cur+1 开始遍历下标 nxt，如果 balance_list[nxt]⋅balance_list[cur]<0：
  - 将 balance_list[cur] 的余额加到 balance_list[nxt]：balance_list[nxt]+=balance_list[cur]
  - 递归调用 dfs(cur+1)，即 dfs(cur)=1+dfs(cur+1)
  - 从 nxt 中移除上一步转移的余额：balance_list[nxt]−=balance_list[cur]（回溯）
- 重复步骤 5，并持续跟踪迭代过程中遇到的最少操作数 cost=min(cost,1+dfs(cur+1))。迭代完成后返回 cost
- 返回 dfs(0)

```js
/**
 * @param {number[][]} transactions
 * @return {number}
 */
var minTransfers = function(transactions) {
    const balanceMap = new Map();

    for (const [a, b, amount] of transactions) {
        balanceMap.set(a, (balanceMap.get(a) || 0) + amount);
        balanceMap.set(b, (balanceMap.get(b) || 0) - amount);
    }

    const balanceList = [];

    for (const amount of balanceMap.values()) {
        if (amount !== 0) {
            balanceList.push(amount);
        }
    }

    const n = balanceList.length;

    function dfs(cur) {
        while (cur < n && balanceList[cur] === 0) {
            cur++;
        }

        if (cur === n) {
            return 0;
        }

        let cost = Infinity;

        for (let nxt = cur + 1; nxt < n; nxt++) {
            if (balanceList[cur] * balanceList[nxt] < 0) {
                balanceList[nxt] += balanceList[cur];

                cost = Math.min(cost, 1 + dfs(cur + 1));

                balanceList[nxt] -= balanceList[cur];
            }
        }

        return cost;
    }

    return dfs(0);
};
```

```java
class Solution {
    public int minTransfers(int[][] transactions) {
        Map<Integer, Integer> balanceMap = new HashMap<>();

        for (int[] t : transactions) {
            int a = t[0];
            int b = t[1];
            int amount = t[2];

            balanceMap.put(a, balanceMap.getOrDefault(a, 0) + amount);
            balanceMap.put(b, balanceMap.getOrDefault(b, 0) - amount);
        }

        List<Integer> balanceList = new ArrayList<>();

        for (int amount : balanceMap.values()) {
            if (amount != 0) {
                balanceList.add(amount);
            }
        }

        return dfs(0, balanceList);
    }

    private int dfs(int cur, List<Integer> balanceList) {
        int n = balanceList.size();

        while (cur < n && balanceList.get(cur) == 0) {
            cur++;
        }

        if (cur == n) {
            return 0;
        }

        int cost = Integer.MAX_VALUE;

        for (int nxt = cur + 1; nxt < n; nxt++) {
            if (balanceList.get(cur) * balanceList.get(nxt) < 0) {
                balanceList.set(
                    nxt,
                    balanceList.get(nxt) + balanceList.get(cur)
                );

                cost = Math.min(cost, 1 + dfs(cur + 1, balanceList));

                balanceList.set(
                    nxt,
                    balanceList.get(nxt) - balanceList.get(cur)
                );
            }
        }

        return cost;
    }
}
```

- 时间复杂度：O((n−1)!)，设 n 为 transactions 的长度
  - 在 dfs(0) 中，最多存在 n−1 个人可以作为可能的 nxt，每个人都会引出一次对 dfs(1) 的递归调用。因此有 dfs(0)=(n−1)⋅dfs(1)=(n−1)⋅((n−2)⋅dfs(2))=(n−1)⋅(n−2)⋅((n−3)⋅dfs(3))=...=(n−1)!⋅dfs(n−1)
  - dfs(n−1) 可以在 O(1) 时间内确定
- 空间复杂度：O(n)
  - balance_map 和 balance_list 最多都拥有 n 个净余额
  - 递归调用的空间复杂度取决于递归调用栈的最大深度，该深度等于 n。因为每次递归调用都会将 cur 加 1，每一层消耗常数空间

### 思路 2: DP

算法步骤

- 创建长度为 2^n 的数组 memo 作为记忆化数组，所有值初始化为 −1
- 将所有非零净余额收集到数组 balance_list 中
- 定义递归函数 dfs(total_mask)，用于把 total_mask 划分为尽可能多的余额和为 0 的子组
  - 如果 memo[total_mask] 不等于 −1，返回 memo[total_mask]
  - 对 total_mask 中每个为 1 的位 cur_mask，移除该位并递归调用 dfs(total_maskˆcur_mask)。持续跟踪这些子问题结果中的最大值 answer
  - 如果 total_mask 的余额和为零，则返回 answer+1。否则返回 answer
- 返回 n−dfs((1<<n)−1)

```python
class Solution:
    def minTransfers(self, transactions: List[List[int]]) -> int:
        balance_map = collections.defaultdict(int)
        for a, b, amount in transactions:
            balance_map[a] += amount
            balance_map[b] -= amount
        
        balance_list = [amount for amount in balance_map.values() if amount]
        n = len(balance_list)
        
        memo = [-1] * (1 << n)
        memo[0] = 0
        
        def dfs(total_mask):
            if memo[total_mask] != -1:
                return memo[total_mask]
            balance_sum, answer = 0, 0

            # 每次从 total_mask 中移除一个人。
            for i in range(n):
                cur_bit = 1 << i
                if total_mask & cur_bit:
                    balance_sum += balance_list[i]
                    answer = max(answer, dfs(total_mask ^ cur_bit))

            # 如果 total_mask 的总余额为 0，则将 answer 加 1。
            memo[total_mask] = answer + (balance_sum == 0)
            return memo[total_mask]
        
        return n - dfs((1 << n) - 1)
```

- 时间复杂度：`O(n⋅2^n)`，设 n 为 transactions 的长度
  - 构建数组 memo 需要 O(2^n) 空间，它等于可能状态的数量。每个状态都需要遍历 balance_list 来计算，耗时 O(n)
- 空间复杂度：`O(2^n)` 
  - memo 的长度为 2^n
  - 递归调用的空间复杂度取决于递归调用栈的最大深度，该深度为 n。因为每次递归调用都会从 total_mask 中移除一个置位，所以最多会产生 O(n) 层递归，每层消耗常数空间
