# 66.Plus One

难度: Easy
标签: Math

![66](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.3k8ozuect6.webp)

限制：
- 1 <= digits.length <= 100
- 0 <= digits[i] <= 9
- digits does not contain any leading 0's.

## 解题

```python
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        # Store the number of digits.
        n = len(digits)

        # Start from the least significant digit
        # and move from right to left.
        for i in range(n - 1, -1, -1):
            # If the current digit is smaller than 9,
            # increment it and finish.
            if digits[i] < 9:
                digits[i] += 1
                return digits

            # The current digit is 9.
            # Adding one changes it to 0 and carries 1
            # to the digit on the left.
            digits[i] = 0

        # If the loop finishes, every digit was 9.
        # For example: 999 + 1 = 1000.
        return [1] + digits
```
```java
class Solution {
    public int[] plusOne(int[] digits) {
        int n = digits.length;
        for (int i = n-1; i >= 0; i--) {
            if (digits[i] < 9) {
                digits[i]++;
                return digits;
            }
            digits[i] = 0;
        }
        int[] res = new int[n+1];
        res[0] = 1;
        return res;
    }
}
```

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`
