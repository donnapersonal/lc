# 256.粉刷房子

题目链接：[256.粉刷房子](https://leetcode.cn/problems/paint-house/)

## 题目大意

假如有一排房子，共 n 个，每个房子可以被粉刷成红色、蓝色或者绿色这三种颜色中的一种，你需要粉刷所有的房子并且使其相邻的两个房子颜色不能相同

当然，因为市场上不同颜色油漆的价格不同，所以房子粉刷成不同颜色的花费成本也是不同的。每个房子粉刷成不同颜色的花费是以一个 n x 3 的正整数矩阵 costs 来表示的

例如，costs[0][0] 表示第 0 号房子粉刷成红色的成本花费；costs[1][2] 表示第 1 号房子粉刷成绿色的花费，以此类推

请计算出粉刷完所有房子最少的花费成本

```js
Example 1:
Input: costs = [[17,2,17],[16,16,5],[14,3,19]]
Output: 10
Explanation: Paint house 0 into blue, paint house 1 into green, paint house 2 into blue.
Minimum cost: 2 + 5 + 3 = 10.

Example 2:
Input: costs = [[7,6,2]]
Output: 2
```

限制：
- costs.length == n
- costs[i].length == 3
- 1 <= n <= 100
- 1 <= costs[i][j] <= 20

## 解题

核心思路

- 只需要维护 3 个状态：
  - redCost   = 刷到当前房子，并且当前房子刷成红色的最小总成本
  - blueCost  = 刷到当前房子，并且当前房子刷成蓝色的最小总成本
  - greenCost = 刷到当前房子，并且当前房子刷成绿色的最小总成本

当处理第 i 栋房子时：
- 如果第 i 栋刷红色，那么第 i - 1 栋不能是红色，只能是蓝色或绿色：newRedCost = min(blueCost, greenCost) + costs[i][0]
- 如果第 i 栋刷蓝色，上一栋不能是蓝色：newBlueCost = min(redCost, greenCost) + costs[i][1]
- 如果第 i 栋刷绿色，上一栋不能是绿色：newGreenCost = min(redCost, blueCost) + costs[i][2]

最后答案就是：min(redCost, blueCost, greenCost)

```js
/**
 * @param {number[][]} costs
 * @return {number}
 */
var minCost = function(costs) {
    if (!costs || costs.length === 0) {
        return 0;
    }

    let [redCost, blueCost, greenCost] = costs[0];

    for (let i = 1; i < costs.length; i++) {
        const newRedCost = Math.min(blueCost, greenCost) + costs[i][0];
        const newBlueCost = Math.min(redCost, greenCost) + costs[i][1];
        const newGreenCost = Math.min(redCost, blueCost) + costs[i][2];

        redCost = newRedCost;
        blueCost = newBlueCost;
        greenCost = newGreenCost;
    }

    return Math.min(redCost, blueCost, greenCost);
};
```

```java
class Solution {
    public int minCost(int[][] costs) {
        if (costs == null || costs.length == 0) {
            return 0;
        }

        int redCost = costs[0][0];
        int blueCost = costs[0][1];
        int greenCost = costs[0][2];

        for (int i = 1; i < costs.length; i++) {
            int newRedCost = Math.min(blueCost, greenCost) + costs[i][0];
            int newBlueCost = Math.min(redCost, greenCost) + costs[i][1];
            int newGreenCost = Math.min(redCost, blueCost) + costs[i][2];

            redCost = newRedCost;
            blueCost = newBlueCost;
            greenCost = newGreenCost;
        }

        return Math.min(redCost, Math.min(blueCost, greenCost));
    }
}
```

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`
