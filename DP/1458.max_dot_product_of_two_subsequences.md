# 1458.两个子序列的最大点积

题目链接：[1458.两个子序列的最大点积](https://leetcode.cn/problems/max-dot-product-of-two-subsequences/)

## 题目大意

给你两个数组 nums1 和 nums2 

请你返回 nums1 和 nums2 中两个长度相同的 非空 子序列的最大点积

数组的非空子序列是通过删除原数组中某些元素（可能一个也不删除）后剩余数字组成的序列，但不能改变数字间相对顺序。比方说，[2,3,5] 是 [1,2,3,4,5] 的一个子序列而 [1,5,3] 不是

```js
Example 1:
Input: nums1 = [2,1,-2,5], nums2 = [3,0,-6]
Output: 18
Explanation: Take subsequence [2,-2] from nums1 and subsequence [3,-6] from nums2.
Their dot product is (2*3 + (-2)*(-6)) = 18.

Example 2:
Input: nums1 = [3,-2], nums2 = [2,-6,7]
Output: 21
Explanation: Take subsequence [3] from nums1 and subsequence [7] from nums2.
Their dot product is (3*7) = 21.

Example 3:
Input: nums1 = [-1,-1], nums2 = [1,1]
Output: -1
Explanation: Take subsequence [-1] from nums1 and subsequence [1] from nums2.
Their dot product is -1.
```

限制：
- 1 <= nums1.length, nums2.length <= 500 
- -1000 <= nums1[i], nums2[i] <= 1000

## 解题

核心思路：`动态规划`

定义：dp[i][j]

表示：从 nums1 前 i 个数、nums2 前 j 个数中，选两个非空子序列能得到的最大点积。

每一步有 3 种选择

当前看：

x = nums1[i - 1]
y = nums2[j - 1]

我们有三种选择：
- 选 x 和 y 配对：当前这两个数一起加入点积
- 不选 x：看 nums1 前面
- 不选 y：看 nums2 前面

如果选 x 和 y，有两种情况：max(dp[i-1][j-1], 0) + x * y
- 如果前面的结果是正数，就接上
- 如果前面的结果是负数，不如从当前这一对重新开始
  
为什么初始化为 -inf？
- 因为题目要求子序列必须是非空
- 不能什么都不选然后得到 0

比如：

nums1 = [-1]
nums2 = [2]

答案应该是：-2 不是 0

所以初始状态要用 -inf 表示“还没有合法选择”

```js
/**
 * @param {number[]} nums1
 * @param {number[]} nums2
 * @return {number}
 */
var maxDotProduct = function(nums1, nums2) {
    const m = nums2.length;
    const f = new Array(m + 1).fill(-Infinity);

    for (const x of nums1) {
        let pre = f[0];

        for (let j = 0; j < m; j++) {
            const y = nums2[j];
            const tmp = f[j + 1];

            f[j + 1] = Math.max(
                Math.max(pre, 0) + x * y,
                f[j + 1],
                f[j]
            );

            pre = tmp;
        }
    }

    return f[m];
};
```
```python
# 二维DP
from typing import List
from math import inf

class Solution:
    def maxDotProduct(self, nums1: List[int], nums2: List[int]) -> int:
        n, m = len(nums1), len(nums2)
        f = [[-inf] * (m + 1) for _ in range(n + 1)]

        for i, x in enumerate(nums1):
            for j, y in enumerate(nums2):
                f[i + 1][j + 1] = max(
                    max(f[i][j], 0) + x * y,
                    f[i][j + 1],
                    f[i + 1][j]
                )

        return f[n][m]

# 空间优化
```

```java
import java.util.*;

class Solution {
    public int maxDotProduct(int[] nums1, int[] nums2) {
        int m = nums2.length;
        int[] f = new int[m + 1];

        Arrays.fill(f, Integer.MIN_VALUE / 2);

        for (int x : nums1) {
            int pre = f[0];

            for (int j = 0; j < m; j++) {
                int y = nums2[j];
                int tmp = f[j + 1];

                f[j + 1] = Math.max(
                    Math.max(pre, 0) + x * y,
                    Math.max(f[j + 1], f[j])
                );

                pre = tmp;
            }
        }

        return f[m];
    }
}
```

- 时间复杂度：`O(nm)`，其中 `n` 是 `nums1` 的长度，`m` 是 `nums2` 的长度
- 空间复杂度：`O(nm)`，空间优化后是 `O(m)`
