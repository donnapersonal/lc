# 413.等差数列划分

题目链接：[413.等差数列划分](https://leetcode.cn/problems/arithmetic-slices/)

## 题目大意

如果一个数列 至少有三个元素 ，并且任意两个相邻元素之差相同，则称该数列为等差数列

- 例如，[1,3,5,7,9]、[7,7,7,7] 和 [3,-1,-5,-9] 都是等差数列
  
给你一个整数数组 nums ，返回数组 nums 中所有为等差数组的 子数组 个数

子数组 是数组中的一个连续序列

```js
Example 1:
Input: nums = [1,2,3,4]
Output: 3
Explanation: We have 3 arithmetic slices in nums: [1, 2, 3], [2, 3, 4] and [1,2,3,4] itself.

Example 2:
Input: nums = [1]
Output: 0
```

限制：
- 1 <= nums.length <= 5000
- -1000 <= nums[i] <= 1000

## 解题

类似于递归解法，定义 dp[i] 是以 A[i] 为终点的等差数列的个数

类似递归思路，有两种情况：
​- A[i] - A[i - 1] == A[i - 1] - A[i - 2]时，说明增加的A[i]能和前面构成等差数列，那么 dp[i] = dp[i - 1] + 1
A[i] - A[i - 1] != A[i - 1] - A[i - 2]时， 说明增加的 A[i]不能和前面构成等差数列，所以dp[i] = 0
动态规划的初始状态：dp[0]=0,dp[1]=0

最后，我们要求的是整个数组中的等差数列的数目，所以需要把 0<=i<=len(A−1) 的所有 dp[i] 的结果累加起来

由于 dp[i] 只和 dp[i - 1] 有关，所以可以进行状态压缩，只用一个变量 k 来表示以 A[i] 为终点的等差数列的个数

```python
# 原版
class Solution(object):
    def numberOfArithmeticSlices(self, A):
        N = len(A)
        dp = [0] * N
        for i in range(1, N - 1):
            if A[i - 1] + A[i + 1] == A[i] * 2:
                dp[i] = dp[i - 1] + 1
        return sum(dp)

# 空间优化
class Solution:
    def numberOfArithmeticSlices(self, nums: List[int]) -> int:
        count = 0
        k = 0

        for i in range(len(nums) - 2):
            if nums[i + 1] - nums[i] == nums[i + 2] - nums[i + 1]:
                k += 1
                count += k
            else:
                k = 0

        return count
```
```java
class Solution {
    public int numberOfArithmeticSlices(int[] nums) {
        int count = 0;
        int k = 0;

        for (int i = 0; i < nums.length - 2; i++) {
            if (nums[i + 1] - nums[i] == nums[i + 2] - nums[i + 1]) {
                k += 1;
                count += k;
            } else {
                k = 0;
            }
        }

        return count;
    }
}
```

- 时间复杂度：O(N)
- 空间复杂度：O(1)
