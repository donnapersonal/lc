# 2110.股票平滑下跌阶段的数目

题目链接：[2110.股票平滑下跌阶段的数目](https://leetcode.cn/problems/number-of-smooth-descent-periods-of-a-stock/)

## 题目大意

给你一个整数数组 prices ，表示一支股票的历史每日股价，其中 prices[i] 是这支股票第 i 天的价格

一个 平滑下降的阶段 定义为：对于 连续一天或者多天 ，每日股价都比 前一日股价恰好少 1 ，这个阶段第一天的股价没有限制

请你返回 平滑下降阶段 的数目

```js
Example 1:
Input: prices = [3,2,1,4]
Output: 7
Explanation: There are 7 smooth descent periods:
[3], [2], [1], [4], [3,2], [2,1], and [3,2,1]
Note that a period with one day is a smooth descent period by the definition.

Example 2:
Input: prices = [8,6,7,7]
Output: 4
Explanation: There are 4 smooth descent periods: [8], [6], [7], and [7]
Note that [8,6] is not a smooth descent period as 8 - 6 ≠ 1.

Example 3:
Input: prices = [1]
Output: 1
Explanation: There is 1 smooth descent period: [1]
```

限制:
- 1 <= prices.length <= 10^5
- 1 <= prices[i] <= 10^5

## 解题

题意：计算 prices 有多少个连续子数组是连续递减的

示例 1 的 prices=[3,2,1,4]，按照子数组的右端点下标分组，有这些连续递减子数组：
- 右端点 i=0：[3]
- 右端点 i=1：[3,2],[2]
- 右端点 i=2：[3,2,1],[2,1],[1]
- 右端点 i=3：[4]

在遍历 prices 的同时，统计当前这段连续递减的长度 dec，那么右端点为 i 的连续递减子数组个数就是 dec，加到答案中
- 如果 prices[i]=prices[i−1]−1，连续递减，dec 增加一
- 否则，连续递减中断，重新统计，dec=1

```js
/**
 * @param {number[]} prices
 * @return {number}
 */
var getDescentPeriods = function(prices) {
    let ans = 0;
    let dec = 0;

    for (let i = 0; i < prices.length; i++) {
        if (i > 0 && prices[i] === prices[i - 1] - 1) {
            dec++;
        } else {
            dec = 1;
        }

        ans += dec;
    }

    return ans;
};
```

class Solution:
    def getDescentPeriods(self, prices: List[int]) -> int:
        ans = dec = 0

        for i, price in enumerate(prices):
            if i > 0 and price == prices[i - 1] - 1:
                dec += 1
            else:
                dec = 1

            ans += dec

        return ans
```

class Solution {
    public long getDescentPeriods(int[] prices) {
        long ans = 0;
        long dec = 0;

        for (int i = 0; i < prices.length; i++) {
            if (i > 0 && prices[i] == prices[i - 1] - 1) {
                dec++;
            } else {
                dec = 1;
            }

            ans += dec;
        }

        return ans;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 是 `prices` 的长度
- 空间复杂度：`O(1)`
