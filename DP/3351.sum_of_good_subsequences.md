# 3351.好子序列的元素之和

题目链接：[3351.好子序列的元素之和](https://leetcode.cn/problems/sum-of-good-subsequences/)

## 题目大意

给你一个整数数组 nums。好子序列 的定义是：子序列中任意 两个 连续元素的绝对差 恰好 为 1

子序列 是指可以通过删除某个数组的部分元素（或不删除）得到的数组，并且不改变剩余元素的顺序

返回 nums 中所有 可能存在的 好子序列的 元素之和

因为答案可能非常大，返回结果需要对 10^9 + 7 取余

注意，长度为 1 的子序列默认为好子序列

```js
Example 1:
Input: nums = [1,2,1]
Output: 14
Explanation:
Good subsequences are: [1], [2], [1], [1,2], [2,1], [1,2,1].
The sum of elements in these subsequences is 14.

Example 2:
Input: nums = [3,4,5]
Output: 40
Explanation:
Good subsequences are: [3], [4], [5], [3,4], [4,5], [3,4,5].
The sum of elements in these subsequences is 40.
```

限制:
- 1 <= nums.length <= 10^5
- 0 <= nums[i] <= 10^5

## 解题

对于本题，定义 f[x] 表示以元素 x 结尾的子序列的元素之和，子序列的倒数第二个数是 x−1 或 x+1

那么 x 可以加在所有以 x−1 结尾的子序列之后，以及所有以 x+1 结尾的子序列之后

为此，还需要知道以 x 结尾的子序列的个数，记作 cnt[x]

从左到右遍历 nums，设 x=nums[i]，考虑是否选 x（是否在子序列中），分类讨论：
- 不选 x，元素和为 f[x]
- x 加在所有以 x−1 结尾的子序列之后，这会额外产生 cnt[x−1] 个子序列，相当于在 f[x−1] 的基础上，额外增加了 cnt[x−1] 个 x，所以这些子序列的元素总和为 f[x−1]+x⋅cnt[x−1]
- x 加在所有以 x+1 结尾的子序列之后，这会额外产生 cnt[x+1] 个子序列，相当于在 f[x+1] 的基础上，额外增加了 cnt[x+1] 个 x，所以这些子序列的元素总和为 f[x+1]+x⋅cnt[x+1]
- x 单独作为一个子序列，元素和为 x
  
所以有：f[x]=f[x]+f[x−1]+f[x+1]+x⋅(cnt[x−1]+cnt[x+1]+1)

同时，额外产生了 cnt[x−1]+cnt[x+1]+1 个以 x 结尾的子序列，所以有 cnt[x]=cnt[x]+cnt[x−1]+cnt[x+1]+1

记得取模

```js
/**
 * @param {number[]} nums
 * @return {number}
 */
var sumOfGoodSubsequences = function(nums) {
    const MOD = 1000000007n;

    const f = new Map();
    const cnt = new Map();

    function get(map, key) {
        return map.has(key) ? map.get(key) : 0n;
    }

    for (const x of nums) {
        const bx = BigInt(x);

        const c = (get(cnt, x - 1) + get(cnt, x + 1) + 1n) % MOD;

        const newSum = (
            get(f, x) +
            get(f, x - 1) +
            get(f, x + 1) +
            bx * c
        ) % MOD;

        f.set(x, newSum);
        cnt.set(x, (get(cnt, x) + c) % MOD);
    }

    let ans = 0n;

    for (const value of f.values()) {
        ans = (ans + value) % MOD;
    }

    return Number(ans);
};
```

class Solution:
    def sumOfGoodSubsequences(self, nums: List[int]) -> int:
        MOD = 1_000_000_007
        f = defaultdict(int)
        cnt = defaultdict(int)
        for x in nums:
            c = cnt[x - 1] + cnt[x + 1] + 1
            f[x] = (f[x] + f[x - 1] + f[x + 1] + x * c) % MOD
            cnt[x] = (cnt[x] + c) % MOD
        return sum(f.values()) % MOD

```java
import java.util.*;

class Solution {
    public int sumOfGoodSubsequences(int[] nums) {
        long MOD = 1_000_000_007L;

        Map<Integer, Long> f = new HashMap<>();
        Map<Integer, Long> cnt = new HashMap<>();

        for (int x : nums) {
            long leftCnt = cnt.getOrDefault(x - 1, 0L);
            long rightCnt = cnt.getOrDefault(x + 1, 0L);

            long c = (leftCnt + rightCnt + 1) % MOD;

            long leftSum = f.getOrDefault(x - 1, 0L);
            long rightSum = f.getOrDefault(x + 1, 0L);
            long oldSum = f.getOrDefault(x, 0L);

            long newSum = (oldSum + leftSum + rightSum + (long) x * c) % MOD;
            f.put(x, newSum);

            long oldCnt = cnt.getOrDefault(x, 0L);
            cnt.put(x, (oldCnt + c) % MOD);
        }

        long ans = 0;
        for (long value : f.values()) {
            ans = (ans + value) % MOD;
        }

        return (int) ans;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 是 `nums` 的长度
- 空间复杂度：`O(n)`
