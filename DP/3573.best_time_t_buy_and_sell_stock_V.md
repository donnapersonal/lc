# 3573.买卖股票的最佳时机 V

题目链接：[3573.买卖股票的最佳时机 V](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-v/)

## 题目大意

给你一个整数数组 prices，其中 prices[i] 是第 i 天股票的价格（美元），以及一个整数 k

你最多可以进行 k 笔交易，每笔交易可以是以下任一类型：
- 普通交易：在第 i 天买入，然后在之后的第 j 天卖出，其中 i < j。你的利润是 prices[j] - prices[i]
- 做空交易：在第 i 天卖出，然后在之后的第 j 天买回，其中 i < j。你的利润是 prices[i] - prices[j]

注意：你必须在开始下一笔交易之前完成当前交易。此外，你不能在已经进行买入或卖出操作的同一天再次进行买入或卖出操作

通过进行 最多 k 笔交易，返回你可以获得的最大总利润

```js
Example 1:
Input: prices = [1,7,9,8,2], k = 2
Output: 14
Explanation: We can make $14 of profit through 2 transactions:
A normal transaction: buy the stock on day 0 for $1 then sell it on day 2 for $9.
A short selling transaction: sell the stock on day 3 for $8 then buy back on day 4 for $2.

Example 2:
Input: prices = [12,16,19,19,8,1,19,13,9], k = 3
Output: 36
Explanation: We can make $36 of profit through 3 transactions:
A normal transaction: buy the stock on day 0 for $12 then sell it on day 2 for $19.
A short selling transaction: sell the stock on day 3 for $19 then buy back on day 4 for $8.
A normal transaction: buy the stock on day 5 for $1 then sell it on day 6 for $19.
```

限制：
- 2 <= prices.length <= 10^3
- 1 <= prices[i] <= 10^9
- 1 <= k <= prices.length / 2

## 解题

**状态机 DP：在 188 题的基础上增加一个状态**

若只有普通交易，那么本题就是 `188. 买卖股票的最佳时机 IV`

回顾一下，`188` 题中的状态定义为：
- `dfs(i,j,0)` 表示在 `prices[0]` 到 `prices[i]` 中完成至多 `j` 笔交易，第 `i` 天结束时未持有股票的最大利润
- `dfs(i,j,1)` 表示在 `prices[0]` 到 `prices[i]` 中完成至多 `j` 笔交易，第 `i` 天结束时持有股票的最大利润

本题可以`做空交易`，即可以`先加再减（普通交易是先减再加）`，需要额外增加一类状态：
- `dfs(i,j,2)` 表示在 `prices[0]` 到 `prices[i]` 中完成至多 `j` 笔交易，第 `i` 天结束时处于做空中（空头状态）的最大利润
- 同样地，转移方程增加一条：`dfs(i,j,2)=max(dfs(i−1,j,2),dfs(i−1,j−1,0)+prices[i])`

分别对应继续处于做空中，或者在第 `i` 天卖出股票，开始做空（开空）

此外，对于 `dfs(i,j,0)`，在计算最大值时额外考虑在第 `i` 天买回股票（平空）的情况，即 `dfs(i−1,j,2)−prices[i]`

注意本题可能算出负数，`memo` 数组可以初始化成 `∞` 或 `−∞`

**记忆化搜索**

```python
class Solution:
    def maximumProfit(self, prices: List[int], k: int) -> int:
        # 在 [0,i] 中完成至多 j 笔交易，第 i 天【结束时】的状态为 end_state 的情况下的最大收益
        # 0=未持有股票，1=持有股票，2=做空中
        @cache  # 缓存装饰器，避免重复计算 dfs 的结果（记忆化）
        def dfs(i: int, j: int, end_state: int) -> int:
            if j < 0:
                return -inf
            if i < 0:
                return -inf if end_state else 0
            p = prices[i]
            if end_state == 0:
                return max(dfs(i - 1, j, 0), dfs(i - 1, j, 1) + p, dfs(i - 1, j, 2) - p)
            if end_state == 1:
                return max(dfs(i - 1, j, 1), dfs(i - 1, j - 1, 0) - p)
            return max(dfs(i - 1, j, 2), dfs(i - 1, j - 1, 0) + p)

        res = dfs(len(prices) - 1, k, 0)
        dfs.cache_clear()  # 防止爆内存（一般来说，状态数达到 1e6 就需要写这个）
        return res
```

- 时间复杂度：`O(nk)`，其中 `n` 是 `prices` 的长度
- 空间复杂度：`O(nk)`

**1:1 翻译成递推**

```python
class Solution:
    def maximumProfit(self, prices: List[int], k: int) -> int:
        n = len(prices)
        f = [[[-inf] * 3 for _ in range(k + 2)] for _ in range(n + 1)]
        for j in range(1, k + 2):
            f[0][j][0] = 0
        for i, p in enumerate(prices):
            for j in range(1, k + 2):
                f[i + 1][j][0] = max(f[i][j][0], f[i][j][1] + p, f[i][j][2] - p)
                f[i + 1][j][1] = max(f[i][j][1], f[i][j - 1][0] - p)
                f[i + 1][j][2] = max(f[i][j][2], f[i][j - 1][0] + p)
        return f[-1][-1][0]
```

**空间优化**

```js
/**
 * @param {number[]} prices
 * @param {number} k
 * @return {number}
 */
var maximumProfit = function(prices, k) {
    const NEG_INF = Number.NEGATIVE_INFINITY;

    // dp[j][0]: max profit after using up to j transaction slots, with no position
    // dp[j][1]: max profit after using up to j transaction slots, holding a long position
    // dp[j][2]: max profit after using up to j transaction slots, holding a short position
    const dp = Array.from({ length: k + 2 }, () => Array(3).fill(NEG_INF));

    // Before processing any price, the profit is 0 when we hold no position.
    for (let j = 1; j < k + 2; j++) {
        dp[j][0] = 0;
    }

    for (const price of prices) {
        // Iterate backwards to avoid using states updated on the same day.
        for (let j = k + 1; j >= 1; j--) {
            // End today with no position:
            // 1. do nothing
            // 2. close long position by selling
            // 3. close short position by buying back
            dp[j][0] = Math.max(
                dp[j][0],
                dp[j][1] + price,
                dp[j][2] - price
            );

            // End today holding a long position:
            // 1. keep holding
            // 2. open a long position by buying
            dp[j][1] = Math.max(
                dp[j][1],
                dp[j - 1][0] - price
            );

            // End today holding a short position:
            // 1. keep shorting
            // 2. open a short position by selling first
            dp[j][2] = Math.max(
                dp[j][2],
                dp[j - 1][0] + price
            );
        }
    }

    return dp[k + 1][0];
};
```

```java
class Solution {
    public long maximumProfit(int[] prices, int k) {
        long NEG_INF = Long.MIN_VALUE / 4;

        long[][] dp = new long[k + 2][3];

        for (int j = 0; j < k + 2; j++) {
            dp[j][0] = NEG_INF;
            dp[j][1] = NEG_INF;
            dp[j][2] = NEG_INF;
        }

        // Before processing any price, profit is 0 if we hold no position.
        for (int j = 1; j < k + 2; j++) {
            dp[j][0] = 0;
        }

        for (int price : prices) {
            // Iterate backward to avoid using states updated on the same day.
            for (int j = k + 1; j >= 1; j--) {
                // End today with no position:
                // 1. do nothing
                // 2. close long by selling
                // 3. close short by buying back
                dp[j][0] = Math.max(
                    dp[j][0],
                    Math.max(dp[j][1] + price, dp[j][2] - price)
                );

                // End today holding a long position:
                // 1. keep holding
                // 2. open long by buying
                dp[j][1] = Math.max(
                    dp[j][1],
                    dp[j - 1][0] - price
                );

                // End today holding a short position:
                // 1. keep shorting
                // 2. open short by selling first
                dp[j][2] = Math.max(
                    dp[j][2],
                    dp[j - 1][0] + price
                );
            }
        }

        return dp[k + 1][0];
    }
}
```
