# 3434.子数组操作后的最大频率

题目链接：[3434.子数组操作后的最大频率](https://leetcode.cn/problems/maximum-frequency-after-subarray-operation/)

## 题目大意

给你一个长度为 n 的数组 nums ，同时给你一个整数 k 

可以对 nums 执行以下操作 一次 ：
- 选择一个子数组 nums[i..j] ，其中 0 <= i <= j <= n - 1 
- 选择一个整数 x 并将 nums[i..j] 中 所有 元素都增加 x 
  
请返回执行以上操作以后数组中 k 出现的 最大 频率

子数组 是一个数组中一段连续 非空 的元素序列

```js
Example 1:
Input: nums = [1,2,3,4,5,6], k = 1
Output: 2
Explanation:
After adding -5 to nums[2..5], 1 has a frequency of 2 in [1, 2, -2, -1, 0, 1].

Example 2:
Input: nums = [10,2,3,4,5,5,4,3,2,2], k = 10
Output: 4
Explanation:
After adding 8 to nums[1..9], 10 has a frequency of 4 in [10, 10, 11, 12, 13, 13, 12, 11, 10, 10].
```

限制：
- 1 <= n == nums.length <= 10^5
- 1 <= nums[i] <= 50
- 1 <= k <= 50

## 解题

`枚举 + 状态机 DP`

考虑把子数组中等于 target 的元素都变成 k

由于 nums 至多有 50 种不同元素，可以枚举 target=1,2,3,…,50（或者 nums 中的不同元素）

nums 可以分为三段：
- 左：被修改的子数组的左边
- 中：被修改的子数组
- 右：被修改的子数组的右边
  
用状态机 DP 计算 nums[0] 到 nums[i] 最多有多少个数可以等于 k：
- f[i+1][0] 表示左，或者说 nums[i] 在被修改的子数组的左侧，此时只能统计等于 k 的元素个数
- f[i+1][1] 表示左+中，或者说 nums[i] 在被修改的子数组中，此时只能统计等于 target 的元素个数，这些数被修改成 k。注意不能统计子数组中的 k 的个数，因为子数组中的 k 会被修改成别的数
- f[i+1][2] 表示左+中+右，或者说 nums[i] 在被修改的子数组的右侧，此时只能统计等于 k 的元素个数

`状态机 DP 优化（一次遍历）`

f0 和 f2 的定义不变

设 x=nums[i]。用 f1[x] 存储方法一中的 target=x 时的 f1 状态，转移方程为 f1[x]=max(f1[x],f0)+1

其余 f1[y] (y 不等于 x) 不变（懒更新）

额外定义 maxF1 表示 f1[x] 的最大值

那么 f2 的转移方程为 f2=max(f2, maxF1)+[x=k]

把 maxF1 和 f2 合并成一个变量 maxF12。当 x=k 的时候，把 maxF12 加一。转移方程中的 max(f2, maxF1) 无需计算，因为两个变量已经合二为一

此外，x=k 的时候不需要计算 f1[x]，因为这个状态等价于统计 k 的个数，这也是 maxF12 统计的内容

```js
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
var maxFrequency = function(nums, k) {
    let f0 = 0;
    let maxF12 = 0;
    const f1 = new Array(51).fill(0);

    for (const x of nums) {
        if (x === k) {
            maxF12 += 1;
            f0 += 1;
        } else {
            f1[x] = Math.max(f1[x], f0) + 1;
            maxF12 = Math.max(maxF12, f1[x]);
        }
    }

    return maxF12;
};

// 如果 nums[i] 范围不确定，用 Map 更安全：
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
var maxFrequency = function(nums, k) {
    let f0 = 0;
    let maxF12 = 0;
    const f1 = new Map();

    for (const x of nums) {
        if (x === k) {
            maxF12 += 1;
            f0 += 1;
        } else {
            const prev = f1.get(x) || 0;
            const cur = Math.max(prev, f0) + 1;

            f1.set(x, cur);
            maxF12 = Math.max(maxF12, cur);
        }
    }

    return maxF12;
};
```

```java
class Solution {
    public int maxFrequency(int[] nums, int k) {
        int f0 = 0;
        int maxF12 = 0;
        int[] f1 = new int[51];

        for (int x : nums) {
            if (x == k) {
                maxF12++;
                f0++;
            } else {
                f1[x] = Math.max(f1[x], f0) + 1;
                maxF12 = Math.max(maxF12, f1[x]);
            }
        }

        return maxF12;
    }
}
```

- 时间复杂度：`O(n+U)`，其中 `n` 是 `nums` 的长度，`U` 是 `nums` 中的不同元素个数，注意创建数组需要 `O(U)` 的时间
- 空间复杂度：`O(U)`
