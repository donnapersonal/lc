# 312.戳气球

题目链接：[312.戳气球](https://leetcode.cn/problems/burst-balloons/)

## 题目大意

有 n 个气球，编号为0 到 n - 1，每个气球上都标有一个数字，这些数字存在数组 nums 中

现在要求你戳破所有的气球。戳破第 i 个气球，你可以获得 nums[i - 1] * nums[i] * nums[i + 1] 枚硬币。 这里的 i - 1 和 i + 1 代表和 i 相邻的两个气球的序号。如果 i - 1或 i + 1 超出了数组的边界，那么就当它是一个数字为 1 的气球

求所能获得硬币的最大数量

```js
Example 1:
Input: nums = [3,1,5,8]
Output: 167
Explanation:
nums = [3,1,5,8] --> [3,5,8] --> [3,8] --> [8] --> []
coins =  3*1*5    +   3*5*8   +  1*3*8  + 1*8*1 = 167

Example 2:
Input: nums = [1,5]
Output: 10
```

限制：
- n == nums.length
- 1 <= n <= 300
- 0 <= nums[i] <= 100

## 解题

**核心难点**

这题不能直接贪心

比如每次戳最大数字、最小数字，或者当前收益最大的气球，都不一定对

原因是：戳掉一个气球后，它左右两边的气球会变成新的相邻关系，后面的收益会被影响

所以这题适合用 `区间 DP`

**关键思路：反过来想，不要想：第一个戳哪个气球？**

因为第一个戳气球时，它左右邻居会一直变化，很难处理

我们反过来想：在某个区间里，最后一个被戳破的气球是谁？

假设我们在区间 (left, right) 里面戳气球，最后戳的是 k

那么在戳 k 的时候，区间 (left, right) 里面其他气球都已经被戳完了

所以 k 的左右邻居一定是：left 和 right

此时获得的金币就是：val[left] * val[k] * val[right]

这样问题就被拆成了两个独立子问题：
- (left, k) 里面最多能拿多少金币
- (k, right) 里面最多能拿多少金币

**为什么要首尾加 1？**

原数组：nums = [3,1,5,8]

我们变成：val = [1,3,1,5,8,1]

首尾加两个虚拟气球 1，这样处理边界更方便

它们不会被真正戳破，只是作为左右边界使用

**DP 定义：dp[left][right]**

表示：戳破开区间 (left, right) 里面所有气球，最多可以获得多少金币。

注意是 开区间：
- left 和 right 本身不戳
- 只戳 left 和 right 中间的气球

**状态转移**

枚举 k，表示 k 是 (left, right) 中最后一个被戳破的气球

那么：

dp[left][right] = dp[left][k] + val[left] * val[k] * val[right] + dp[k][right]

取最大值：

dp[left][right] = max(
    dp[left][right],
    dp[left][k] + val[left] * val[k] * val[right] + dp[k][right]
)

**遍历顺序**

因为大区间依赖小区间：dp[left][right] 依赖 dp[left][k] 和 dp[k][right]

所以我们要先算小区间，再算大区间，也就是按照区间长度从小到大遍历

```js
/**
 * @param {number[]} nums
 * @return {number}
 */
var maxCoins = function(nums) {
    const val = [1, ...nums, 1];
    const n = val.length;

    const dp = Array.from({ length: n }, () => Array(n).fill(0));

    for (let length = 2; length < n; length++) {
        for (let left = 0; left < n - length; left++) {
            const right = left + length;

            for (let k = left + 1; k < right; k++) {
                const coins = dp[left][k] + val[left] * val[k] * val[right] + dp[k][right];
                dp[left][right] = Math.max(dp[left][right], coins);
            }
        }
    }

    return dp[0][n - 1];
};
```
```python
class Solution:
    def maxCoins(self, nums: List[int]) -> int:
        val = [1] + nums + [1]
        n = len(val)

        dp = [[0] * n for _ in range(n)]

        for length in range(2, n):
            for left in range(0, n - length):
                right = left + length

                for k in range(left + 1, right):
                    coins = dp[left][k] + val[left] * val[k] * val[right] + dp[k][right]
                    dp[left][right] = max(dp[left][right], coins)

        return dp[0][n - 1]
```
```java
class Solution {
    public int maxCoins(int[] nums) {
        int m = nums.length;
        int[] val = new int[m + 2];

        val[0] = 1;
        val[m + 1] = 1;

        for (int i = 0; i < m; i++) {
            val[i + 1] = nums[i];
        }

        int n = val.length;
        int[][] dp = new int[n][n];

        for (int length = 2; length < n; length++) {
            for (int left = 0; left < n - length; left++) {
                int right = left + length;

                for (int k = left + 1; k < right; k++) {
                    int coins = dp[left][k] 
                              + val[left] * val[k] * val[right] 
                              + dp[k][right];

                    dp[left][right] = Math.max(dp[left][right], coins);
                }
            }
        }

        return dp[0][n - 1];
    }
}
```

- 时间复杂度: `O(n^3)`，假设原始 nums 长度是 n
  - 代码里加了两个虚拟气球：val = [1] + nums + [1]，所以新数组长度是：n + 2
  - 核心是三层循环：Loop	Meaning、length	枚举区间长度、left	枚举区间左边界、k	枚举最后一个被戳破的气球，对于每个区间 (left, right)，我们都要尝试区间内所有可能的 k
  - 所以总时间复杂度是：`O(n^3)`
- 空间复杂度: `O(n^2)`
  - 我们用了一个二维 DP 表：dp = [[0] * size for _ in range(size)]
  - 其中：size = n + 2
  - 所以空间复杂度是：`O((n + 2)^2) = O(n^2)`
