# 140.单词拆分 II

题目链接：[140.单词拆分 II](https://leetcode.cn/problems/word-break-ii/)

## 题目大意

给定一个字符串 s 和一个字符串字典 wordDict ，在字符串 s 中增加空格来构建一个句子，使得句子中所有的单词都在词典中。以任意顺序 返回所有这些可能的句子

注意：词典中的同一个单词可能在分段中被重复使用多次

```js
Example 1:
Input: s = "catsanddog", wordDict = ["cat","cats","and","sand","dog"]
Output: ["cats and dog","cat sand dog"]

Example 2:
Input: s = "pineapplepenapple", wordDict = ["apple","pen","applepen","pine","pineapple"]
Output: ["pine apple pen apple","pineapple pen apple","pine applepen apple"]
Explanation: Note that you are allowed to reuse a dictionary word.

Example 3:
Input: s = "catsandog", wordDict = ["cats","dog","sand","and","cat"]
Output: []
```

限制：
- 1 <= s.length <= 20
- 1 <= wordDict.length <= 1000
- 1 <= wordDict[i].length <= 10
- s and wordDict[i] consist of only lowercase English letters.
- All the strings of wordDict are unique.
- Input is generated in a way that the length of the answer doesn't exceed 10^5.

## 解题

这题不是只判断能不能拆分，而是要返回所有拆分方案。

所以可以用：`DFS 回溯 + 记忆化搜索`

核心函数：`dfs(start)` 表示：从 s[start:] 开始，能拆出来的所有单词组合

**为什么需要记忆化？**

因为同一个后缀可能会被不同路径重复计算

例如：

s = "catsanddog"

可能有两条路径都走到 "dog"：

cat + sand + dog
cats + and + dog

如果没有记忆化，dfs(7) 可能被重复计算

所以使用：`@lru_cache(None)`，让每个 start 位置只计算一次

**状态定义**

dfs(start)

返回：List[List[str]]

含义是：从 s[start:] 开始，所有可能的单词拆分列表

例如：dfs(7)

如果：s[7:] = "dog"

返回：[["dog"]]

表示：从 index 7 开始，可以拆成一种方案：dog

**递归终止条件**

if start == len(s):
    return [[]]

这句非常关键

意思是：如果已经走到字符串末尾，说明前面的拆分方案是成功的

为什么返回 `[[]]`，而不是 `[]`？

因为 `[[]]` 表示：有 1 种合法拆法：后面什么都不用接

这样前一层才能继续拼接

例如：

word = "dog"
rest = []

组合成：[word] + rest

也就是：["dog"]

如果返回空列表 []，前一层就没有东西可以遍历，合法方案会丢失

**转移过程**

代码核心：

for end in range(start + 1, len(s) + 1):
    word = s[start:end]

    if word in wordSet:
        for rest in dfs(end):
            res.append([word] + rest)

意思是：从当前位置 start 开始，枚举所有可能的结束位置 end

比如：

s = "catsanddog"
start = 0

会尝试：

"c"
"ca"
"cat"
"cats"
"catsa"
...

如果当前切出来的 word 在字典里，就递归处理剩下部分：dfs(end)

然后把当前单词接到后面的拆分结果前面：[word] + rest

```js
/**
 * @param {string} s
 * @param {string[]} wordDict
 * @return {string[]}
 */
var wordBreak = function(s, wordDict) {
    const wordSet = new Set(wordDict);
    const memo = new Map();

    function dfs(start) {
        if (memo.has(start)) {
            return memo.get(start);
        }

        if (start === s.length) {
            return [[]];
        }

        const res = [];

        for (let end = start + 1; end <= s.length; end++) {
            const word = s.slice(start, end);

            if (wordSet.has(word)) {
                const restLists = dfs(end);

                for (const rest of restLists) {
                    res.push([word, ...rest]);
                }
            }
        }

        memo.set(start, res);
        return res;
    }

    return dfs(0).map(words => words.join(" "));
};
```
```python
from functools import lru_cache
from typing import List

class Solution:
    def wordBreak(self, s: str, wordDict: List[str]) -> List[str]:
        wordSet = set(wordDict)

        @lru_cache(None)
        def dfs(start: int) -> List[List[str]]:
            if start == len(s):
                return [[]]

            res = []

            for end in range(start + 1, len(s) + 1):
                word = s[start:end]

                if word in wordSet:
                    for rest in dfs(end):
                        res.append([word] + rest)

            return res

        return [" ".join(words) for words in dfs(0)]
```
```java
import java.util.*;

class Solution {
    public List<String> wordBreak(String s, List<String> wordDict) {
        Set<String> wordSet = new HashSet<>(wordDict);
        Map<Integer, List<List<String>>> memo = new HashMap<>();

        List<List<String>> wordLists = dfs(s, 0, wordSet, memo);

        List<String> result = new ArrayList<>();
        for (List<String> words : wordLists) {
            result.add(String.join(" ", words));
        }

        return result;
    }

    private List<List<String>> dfs(
        String s,
        int start,
        Set<String> wordSet,
        Map<Integer, List<List<String>>> memo
    ) {
        if (memo.containsKey(start)) {
            return memo.get(start);
        }

        List<List<String>> res = new ArrayList<>();

        if (start == s.length()) {
            res.add(new ArrayList<>());
            return res;
        }

        for (int end = start + 1; end <= s.length(); end++) {
            String word = s.substring(start, end);

            if (wordSet.contains(word)) {
                List<List<String>> restLists = dfs(s, end, wordSet, memo);

                for (List<String> rest : restLists) {
                    List<String> current = new ArrayList<>();
                    current.add(word);
                    current.addAll(rest);
                    res.add(current);
                }
            }
        }

        memo.put(start, res);
        return res;
    }
}
```

这题复杂度是输出敏感的

记忆化搜索可以保证每个 start 位置只计算一次，每个 start 枚举所有 end，所以搜索框架大约是 O(n²)，如果考虑 Python 字符串切片，可能是 O(n³)

但是题目要求返回所有合法句子，而合法句子的数量最坏情况下可能是指数级。每个句子长度最多 O(n)，所以总输出规模最坏是 O(n · 2^n)

因此时间复杂度可以说是 O(n² + total output size)，最坏情况下是 O(n · 2^n)

空间复杂度是 O(n + total output size)，最坏也是 O(n · 2^n)，因为 memo 和最终结果都要存储大量拆分方案
