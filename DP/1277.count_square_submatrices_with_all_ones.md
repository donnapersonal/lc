# 1277.统计全为 1 的正方形子矩阵

题目链接：[1277.统计全为 1 的正方形子矩阵](https://leetcode.cn/problems/count-square-submatrices-with-all-ones/)

## 题目大意

给你一个 m * n 的矩阵，矩阵中的元素不是 0 就是 1，请你统计并返回其中完全由 1 组成的 正方形 子矩阵的个数

```js
Example 1:
Input: matrix =
[
  [0,1,1,1],
  [1,1,1,1],
  [0,1,1,1]
]
Output: 15
Explanation: 
There are 10 squares of side 1.
There are 4 squares of side 2.
There is  1 square of side 3.
Total number of squares = 10 + 4 + 1 = 15.

Example 2:
Input: matrix = 
[
  [1,0,1],
  [1,1,0],
  [1,1,0]
]
Output: 7
Explanation: 
There are 6 squares of side 1.  
There is 1 square of side 2. 
Total number of squares = 6 + 1 = 7.
```

限制：
- 1 <= arr.length <= 300
- 1 <= arr[0].length <= 300
- 0 <= arr[i][j] <= 1

## 解题

题目理解：我们要数出矩阵中所有由 1 组成的正方形，注意是 正方形，不是矩形。

例如：

matrix =
[
  [1, 1],
  [1, 1]
]

里面有：

4 个 1x1 正方形
1 个 2x2 正方形

所以答案是：5

**核心思想**

我们定义一个 DP 数组：`f[i][j]` 表示：以 `matrix[i][j]` 作为右下角，能够形成的最大正方形边长

比如：f[i][j] = 3

说明以 (i, j) 为右下角，可以形成：

1x1 正方形
2x2 正方形
3x3 正方形

一共贡献：3 个正方形

所以最后答案就是：`所有 f[i][j] 的总和`

**为什么 f[i][j] 可以代表贡献数量？**

如果某个位置：`f[i][j] = k` 说明以这个位置为右下角，最大可以形成边长为 `k` 的全 `1` 正方形

那么它一定也可以形成更小的正方形：

边长 1
边长 2
...
边长 k

所以它贡献了：`k 个正方形`

例如：f[i][j] = 2 代表：以 (i,j) 为右下角，可以形成 1x1 和 2x2 两个正方形

所以要：res += f[i][j]

**状态转移怎么来的？**

如果：matrix[i][j] == 0，那么这个位置不能作为任何全 1 正方形的右下角

所以：f[i][j] = 0

如果：matrix[i][j] == 1，那么至少可以形成一个 1x1 正方形

但如果想形成更大的正方形，比如 2x2、3x3，就必须看它的三个方向：

左边：      f[i][j - 1]
上边：      f[i - 1][j]
左上角：    f[i - 1][j - 1]

**为什么看这三个？**

因为一个以 (i, j) 为右下角的大正方形，需要：
- 左边区域也能撑住
- 上边区域也能撑住
- 左上区域也能撑住

三个方向中最短的那个，决定了当前能扩展到多大

所以：

f[i][j] = min(
    f[i][j - 1],
    f[i - 1][j],
    f[i - 1][j - 1]
) + 1

```js
/**
 * @param {number[][]} matrix
 * @return {number}
 */
var countSquares = function(matrix) {
    const m = matrix.length;
    const n = matrix[0].length;

    const f = Array.from({ length: m }, () => Array(n).fill(0));
    let res = 0;

    for (let i = 0; i < m; i++) {
        for (let j = 0; j < n; j++) {
            if (i === 0 || j === 0) {
                f[i][j] = matrix[i][j];
            } else if (matrix[i][j] === 0) {
                f[i][j] = 0;
            } else {
                f[i][j] = Math.min(
                    f[i][j - 1],
                    f[i - 1][j],
                    f[i - 1][j - 1]
                ) + 1;
            }

            res += f[i][j];
        }
    }

    return res;
};
```

```java
class Solution {
    public int countSquares(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;

        int[][] f = new int[m][n];
        int res = 0;

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (i == 0 || j == 0) {
                    f[i][j] = matrix[i][j];
                } else if (matrix[i][j] == 0) {
                    f[i][j] = 0;
                } else {
                    f[i][j] = Math.min(
                        Math.min(f[i][j - 1], f[i - 1][j]),
                        f[i - 1][j - 1]
                    ) + 1;
                }

                res += f[i][j];
            }
        }

        return res;
    }
}
```

- 时间复杂度：`O(m×n)`，其中 `m` 和 `n` 是输入 `matrix` 的行和列
- 空间复杂度：`O(m×n)`
