# 714.买卖股票的最佳时机含手续费

题目链接：[714.买卖股票的最佳时机含手续费](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-with-transaction-fee/)

## 题目大意

给定一个整数数组 prices，其中 prices[i]表示第 i 天的股票价格 ；整数 fee 代表了交易股票的手续费用

你可以无限次地完成交易，但是你每笔交易都需要付手续费。如果你已经购买了一个股票，在卖出它之前你就不能再继续购买股票了

返回获得利润的最大值

注意：这里的一笔交易指买入持有并卖出股票的整个过程，每笔交易你只需要为支付一次手续费

```js
Example 1:
Input: prices = [1,3,2,8,4,9], fee = 2
Output: 8
Explanation: The maximum profit can be achieved by:
- Buying at prices[0] = 1
- Selling at prices[3] = 8
- Buying at prices[4] = 4
- Selling at prices[5] = 9
The total profit is ((8 - 1) - 2) + ((9 - 4) - 2) = 8.

Example 2:
Input: prices = [1,3,7,5,10,3], fee = 3
Output: 6
```

限制：
- 1 <= prices.length <= 5 * 10^4
- 1 <= prices[i] < 5 * 10^4
- 0 <= fee < 5 * 10^4

## 解题

每天结束时，我们的状态其实只有两种：
- `cash`：手里没有股票时，最多能赚多少钱
- `hold`：手里持有一股股票时，最多能赚多少钱

所以这题可以用 `状态机 DP` 来做

状态定义
- 定义：`cash = 当前不持股时的最大利润`, `hold = 当前持股时的最大利润`
- 一开始：`cash = 0, hold = -prices[0]`
  - `cash = 0`: 第一天如果不买股票，利润就是 `0`
  - `hold = -prices[0]`: 第一天如果买入股票，相当于花了 `prices[0]` 的钱，所以利润是：`-prices[0]`

状态转移
- 假设今天股票价格是：`price`
- 更新 `cash`。今天结束后，如果手里没有股票，有两种可能：
  - 情况 1：昨天就没有股票，今天什么都不做 -> 利润还是：`cash`
  - 情况 2：昨天持有股票，今天卖出 -> 卖出可以获得 `price`，但要支付手续费 `fee`
    - 利润变成：`hold + price - fee`
  - 所以：`cash = max(cash, hold + price - fee)`
- 更新 `hold`。今天结束后，如果手里持有股票，也有两种可能：
  - 情况 1：昨天就持有股票，今天什么都不做 -> 利润还是：`hold`
  - 情况 2：昨天没有股票，今天买入 -> 买入要花 `price`，所以利润变成：`cash - price`
  - 所以：`hold = max(hold, cash - price)`
  - 不过要注意，更新 `hold` 时应该用“更新前的 cash”

为什么最后返回 `cash`？
- 最后一天结束时，最大利润一定是不持股状态
- 因为：
  - 如果手里还持有股票，说明钱还压在股票里
  - 没有卖出，就没有真正实现利润
  - 最终答案应该是已经落袋为安的钱。
- 所以返回：`cash`

```js
/**
 * @param {number[]} prices
 * @param {number} fee
 * @return {number}
 */
var maxProfit = function(prices, fee) {
    let cash = 0;              // max profit when not holding a stock
    let hold = -prices[0];     // max profit when holding one stock

    for (let i = 1; i < prices.length; i++) {
        const price = prices[i];
        const prevCash = cash;

        // either keep not holding, or sell today and pay the fee
        cash = Math.max(cash, hold + price - fee);

        // either keep holding, or buy today
        hold = Math.max(hold, prevCash - price);
    }

    return cash;
};
```

```java
class Solution {
    public int maxProfit(int[] prices, int fee) {
        int cash = 0;              // max profit when not holding a stock
        int hold = -prices[0];     // max profit when holding one stock

        for (int i = 1; i < prices.length; i++) {
            int price = prices[i];
            int prevCash = cash;

            // either keep not holding, or sell today and pay the fee
            cash = Math.max(cash, hold + price - fee);

            // either keep holding, or buy today
            hold = Math.max(hold, prevCash - price);
        }

        return cash;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 为数组的长度。一共有 `2n` 个状态，每次状态转移的时间复杂度为 `O(1)`，因此时间复杂度为 `O(2n)=O(n)`
- 空间复杂度：`O(1)`，因为只用了两个变量：`cash`、`hold`，没有使用额外数组
