# 2188.完成比赛的最少时间

题目链接：[2188.完成比赛的最少时间](https://leetcode.cn/problems/minimum-time-to-finish-the-race/)

## 题目大意

给你一个下标从 0 开始的二维整数数组 tires ，其中 tires[i] = [fi, ri] 表示第 i 种轮胎如果连续使用，第 x 圈需要耗时 fi * ri(x-1) 秒

- 比方说，如果 fi = 3 且 ri = 2 ，且一直使用这种类型的同一条轮胎，那么该轮胎完成第 1 圈赛道耗时 3 秒，完成第 2 圈耗时 3 * 2 = 6 秒，完成第 3 圈耗时 3 * 22 = 12 秒，依次类推
  
同时给你一个整数 changeTime 和一个整数 numLaps 

比赛总共包含 numLaps 圈，你可以选择 任意 一种轮胎开始比赛。每一种轮胎都有 无数条 。每一圈后，你可以选择耗费 changeTime 秒 换成 任意一种轮胎（也可以换成当前种类的新轮胎）

请你返回完成比赛需要耗费的 最少 时间

```js
Example 1:
Input: tires = [[2,3],[3,4]], changeTime = 5, numLaps = 4
Output: 21
Explanation: 
Lap 1: Start with tire 0 and finish the lap in 2 seconds.
Lap 2: Continue with tire 0 and finish the lap in 2 * 3 = 6 seconds.
Lap 3: Change tires to a new tire 0 for 5 seconds and then finish the lap in another 2 seconds.
Lap 4: Continue with tire 0 and finish the lap in 2 * 3 = 6 seconds.
Total time = 2 + 6 + 5 + 2 + 6 = 21 seconds.
The minimum time to complete the race is 21 seconds.

Example 2:
Input: tires = [[1,10],[2,2],[3,4]], changeTime = 6, numLaps = 5
Output: 25
Explanation: 
Lap 1: Start with tire 1 and finish the lap in 2 seconds.
Lap 2: Continue with tire 1 and finish the lap in 2 * 2 = 4 seconds.
Lap 3: Change tires to a new tire 1 for 6 seconds and then finish the lap in another 2 seconds.
Lap 4: Continue with tire 1 and finish the lap in 2 * 2 = 4 seconds.
Lap 5: Change tires to tire 0 for 6 seconds then finish the lap in another 1 second.
Total time = 2 + 4 + 6 + 2 + 4 + 6 + 1 = 25 seconds.
The minimum time to complete the race is 25 seconds. 
```

限制：
- 1 <= tires.length <= 10^5
- tires[i].length == 2
- 1 <= fi, changeTime <= 10^5
- 2 <= ri <= 10^5
- 1 <= numLaps <= 1000

## 解题

**提示 1**

连续使用同一个轮胎时，耗费的时间是指数增长的

**提示 2**

连续使用同一个轮胎 i 跑 x 圈，第 x 圈的耗时不应超过 changeTime+fi，否则直接更换轮胎会更优。结合提示 1，连续使用同一个轮胎的次数不会很大

**提示 3**

考虑线性 DP

**算法**

首先预处理出连续使用同一个轮胎跑 x 圈的最小耗时，记作 minSec[x]，这可以通过遍历每个轮胎计算出来

然后定义 f[i] 表示跑 i 圈的最小耗时。为方便计算，初始值 f[0]=−changeTime

考虑最后一个轮胎连续跑了 j 圈，我们可以从 f[i−j] 转移过来，因此有转移方程

![function](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.3uvh91yyxe.webp)

最后答案为 f[numLaps]

```js
/**
 * @param {number[][]} tires
 * @param {number} changeTime
 * @param {number} numLaps
 * @return {number}
 */
var minimumFinishTime = function(tires, changeTime, numLaps) {
    const MAX = 17;
    const INF = Number.MAX_SAFE_INTEGER;

    const minSec = new Array(MAX + 1).fill(INF);

    for (const [firstLap, ratio] of tires) {
        let lapTime = firstLap;
        let totalTime = 0;

        for (let lapCount = 1; lapCount <= MAX; lapCount++) {
            if (lapTime > changeTime + firstLap) {
                break;
            }

            totalTime += lapTime;
            minSec[lapCount] = Math.min(minSec[lapCount], totalTime);

            lapTime *= ratio;
        }
    }

    const dp = new Array(numLaps + 1).fill(INF);
    dp[0] = -changeTime;

    for (let laps = 1; laps <= numLaps; laps++) {
        for (let consecutive = 1; consecutive <= Math.min(MAX, laps); consecutive++) {
            dp[laps] = Math.min(
                dp[laps],
                dp[laps - consecutive] + changeTime + minSec[consecutive]
            );
        }
    }

    return dp[numLaps];
};
```
```python
class Solution:
    def minimumFinishTime(self, tires: List[List[int]], changeTime: int, numLaps: int):
        min_sec = [inf] * 18

        for f, r in tires:
            x, time, s = 1, f, 0
            while time <= changeTime + f:
                s += time
                min_sec[x] = min(min_sec[x], s)
                time *= r
                x += 1

        f = [0] * (numLaps + 1)
        f[0] = -changeTime

        for i in range(1, numLaps + 1):
            f[i] = changeTime + min(
                f[i - j] + min_sec[j]
                for j in range(1, min(18, i + 1))
            )

        return f[numLaps]
```
```java
import java.util.*;

class Solution {
    public int minimumFinishTime(int[][] tires, int changeTime, int numLaps) {
        int MAX = 17;
        int INF = 1_000_000_000;

        int[] minSec = new int[MAX + 1];
        Arrays.fill(minSec, INF);

        for (int[] tire : tires) {
            int firstLap = tire[0];
            int ratio = tire[1];

            long lapTime = firstLap;
            int totalTime = 0;

            for (int lapCount = 1; lapCount <= MAX; lapCount++) {
                if (lapTime > changeTime + firstLap) {
                    break;
                }

                totalTime += lapTime;
                minSec[lapCount] = Math.min(minSec[lapCount], totalTime);

                lapTime *= ratio;
            }
        }

        int[] dp = new int[numLaps + 1];
        Arrays.fill(dp, INF);

        dp[0] = -changeTime;

        for (int laps = 1; laps <= numLaps; laps++) {
            for (int consecutive = 1; consecutive <= Math.min(MAX, laps); consecutive++) {
                dp[laps] = Math.min(
                    dp[laps],
                    dp[laps - consecutive] + changeTime + minSec[consecutive]
                );
            }
        }

        return dp[numLaps];
    }
}
```

- 时间复杂度：`O((n+numLaps)k)`，其中 `n` 是 `tires` 的长度，`k=17`
- 空间复杂度：`O(numLaps+k)`
