# 712.两个字符串的最小ASCII删除和

题目链接：[712.两个字符串的最小ASCII删除和](https://leetcode.cn/problems/minimum-ascii-delete-sum-for-two-strings/)

## 题目大意

给定两个字符串s1 和 s2，返回 使两个字符串相等所需删除字符的 ASCII 值的最小和 

```js
Example 1:
Input: s1 = "sea", s2 = "eat"
Output: 231
Explanation: Deleting "s" from "sea" adds the ASCII value of "s" (115) to the sum.
Deleting "t" from "eat" adds 116 to the sum.
At the end, both strings are equal, and 115 + 116 = 231 is the minimum sum possible to achieve this.

Example 2:
Input: s1 = "delete", s2 = "leet"
Output: 403
Explanation: Deleting "dee" from "delete" to turn the string into "let",
adds 100[d] + 101[e] + 101[e] to the sum.
Deleting "e" from "leet" adds 101[e] to the sum.
At the end, both strings are equal to "let", and the answer is 100+101+101+101 = 403.
If instead we turned both strings into "lee" or "eet", we would get answers of 433 or 417, which are higher.
```

限制：
- 1 <= s1.length, s2.length <= 1000
- s1 and s2 consist of lowercase English letters.

## 解题

**Memozation**

```python
class Solution:
    def minimumDeleteSum(self, s1: str, s2: str) -> int:
        m, n = len(s1), len(s2)
        # -1 代表未曾计算
        self.memo = [[-1] * n for _ in range(m)]
        return self.dp(s1, 0, s2, 0)
    
    # 定义：将 s1[i..] 和 s2[j..] 删除成相同字符串
    # 最小的 ASCII 码之和为 dp(s1, i, s2, j)。
    def dp(self, s1: str, i: int, s2: str, j: int) -> int:
        res = 0
        # 如果 s1 到头了，那么 s2 剩下的都得删除
        if i == len(s1):
            for k in range(j, len(s2)):
                res += ord(s2[k])
            return res
        
        # 如果 s2 到头了，那么 s1 剩下的都得删除
        if j == len(s2):
            for k in range(i, len(s1)):
                res += ord(s1[k])
            return res
        
        if self.memo[i][j] != -1:
            return self.memo[i][j]
        
        # s1[i] 和 s2[j] 都是在 lcs 中的，不用删除
        if s1[i] == s2[j]:
            self.memo[i][j] = self.dp(s1, i + 1, s2, j + 1)
        # s1[i] 和 s2[j] 至少有一个不在 lcs 中，删一个
        else:
            self.memo[i][j] = min(ord(s1[i]) + self.dp(s1, i + 1, s2, j), ord(s2[j]) + self.dp(s1, i, s2, j + 1))
        
        return self.memo[i][j]
```

这道题和 `LCS` 很像，但不是求最长公共子序列长度，而是求：**让两个字符串相等，需要删除的最小 `ASCII` 代价**

可以直接用`动态规划`

DP 定义
- 先看二维 `DP` 的定义，更容易理解。
- 令：`dp[i][j] = 让 s1 的前 i 个字符 和 s2 的前 j 个字符变相等，所需要删除的最小 ASCII 和`，即：`s1[:i]`，`s2[:j]`
- 最终答案是：`dp[n][m]`，其中：`n = len(s1)`，`m = len(s2)`

状态转移
- 比较两个前缀的最后一个字符：`s1[i - 1]` 和 `s2[j - 1]`
- 情况 1：两个字符相同 `s1[i - 1] == s2[j - 1]`
  - 说明这两个字符可以一起保留，不需要删除
  - 所以：`dp[i][j] = dp[i - 1][j - 1]`
  - 例如：s1 前缀 = "se"，s2 前缀 = "e"，如果最后字符相同，就可以从前一个状态直接继承
- 情况 2：两个字符不同 `s1[i - 1] != s2[j - 1]`
  - 这两个字符不能同时保留，必须至少删除一个
  - 有两个选择：
    - 选择 1：删除 `s1[i - 1]`
      - 那么问题变成：`s1[:i - 1]` 和 `s2[:j]` 变相等
      - 代价是：`dp[i - 1][j] + ASCII(s1[i - 1])`
    - 选择 2：删除 `s2[j - 1]`
      - 那么问题变成：`s1[:i]` 和 `s2[:j - 1]` 变相等
      - 代价是：`dp[i][j - 1] + ASCII(s2[j - 1])`
- 所以转移方程是：`dp[i][j] = min(dp[i - 1][j] + ASCII(s1[i - 1]), dp[i][j - 1] + ASCII(s2[j - 1]))`

初始化
- 如果 `s1` 为空，要让它和 `s2[:j]` 相等，只能删除 `s2` 的前 `j` 个字符
  - 所以：`dp[0][j] = dp[0][j - 1] + ASCII(s2[j - 1])`
- 如果 `s2` 为空，要让 `s1[:i]` 和它相等，只能删除 `s1` 的前 `i` 个字符
  - 所以：`dp[i][0] = dp[i - 1][0] + ASCII(s1[i - 1])`

**为什么可以优化成一维 DP？**

二维转移中：`dp[i][j]` 只依赖三个位置 `dp[i - 1][j - 1] 左上角`、`dp[i - 1][j] 上方`、`dp[i][j - 1] 左边`

如果按行更新，就可以用一个一维数组 `dp[j]` 表示当前行的状态

在更新过程中：
- `dp[j]`  在更新前表示 `dp[i - 1][j]`
- `dp[j - 1]` 在更新后表示 `dp[i][j - 1]`
- `buffer` 保存旧的 `dp[i - 1][j - 1]`

所以需要一个额外变量 `buffer` 来保存左上角旧值

```js
/**
 * @param {string} s1
 * @param {string} s2
 * @return {number}
 */
var minimumDeleteSum = function(s1, s2) {
    const n = s1.length;
    const m = s2.length;

    const dp = new Array(m + 1).fill(0);

    // Base case:
    // s1 is empty, delete all characters from s2[0:j]
    for (let j = 1; j <= m; j++) {
        dp[j] = dp[j - 1] + s2.charCodeAt(j - 1);
    }

    for (let i = 1; i <= n; i++) {
        // buffer stores old dp[j - 1], which is dp[i - 1][j - 1]
        let buffer = dp[0];

        // s2 is empty, delete all characters from s1[0:i]
        dp[0] += s1.charCodeAt(i - 1);

        for (let j = 1; j <= m; j++) {
            // temp stores old dp[j], which is dp[i - 1][j]
            const temp = dp[j];

            if (s1[i - 1] === s2[j - 1]) {
                dp[j] = buffer;
            } else {
                const deleteFromS1 = dp[j] + s1.charCodeAt(i - 1);
                const deleteFromS2 = dp[j - 1] + s2.charCodeAt(j - 1);
                dp[j] = Math.min(deleteFromS1, deleteFromS2);
            }

            // Move diagonal value forward
            buffer = temp;
        }
    }

    return dp[m];
};
```

class Solution:
    def minimumDeleteSum(self, s1: str, s2: str) -> int:
        n, m = len(s1), len(s2)

        dp = [0] * (m + 1)

        # Base case:
        # s1 is empty, delete all characters from s2[:j]
        for j in range(1, m + 1):
            dp[j] = dp[j - 1] + ord(s2[j - 1])

        for i in range(1, n + 1):
            # buffer stores old dp[j - 1], which is dp[i - 1][j - 1]
            buffer = dp[0]

            # s2 is empty, delete all characters from s1[:i]
            dp[0] += ord(s1[i - 1])

                # temp stores old dp[j], which is dp[i - 1][j]
                temp = dp[j]

                    delete_from_s1 = dp[j] + ord(s1[i - 1])

```java
class Solution {
    public int minimumDeleteSum(String s1, String s2) {
        int n=s1.length(), m=s2.length();
        char[] c1 = s1.toCharArray();
        char[] c2 = s2.toCharArray();
        int[] dp = new int[m+1];
        for(int i = 0; i < m; i++) {
            dp[i+1] = dp[i]+c2[i];
        }
        for(int i = 1; i <= n; i++) {
            int buffer = dp[0];
            dp[0] += c1[i-1];
            for(int j = 1; j <= m; j++) {
                int res = 0;
                if(c1[i-1] == c2[j-1]) res = buffer;
                else res = Math.min(dp[j]+c1[i-1], dp[j-1]+c2[j-1]);
                buffer = dp[j];
                dp[j] = res ; 
            }
        }
        return dp[m];
    }
}
```

- 时间复杂度: `O(n * m)`，`n = len(s1)`，`m = len(s2)`
- 空间复杂度: `O(m)`
