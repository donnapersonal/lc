# 818.赛车

题目链接：[818.赛车](https://leetcode.cn/problems/race-car/)

## 题目大意

你的赛车可以从位置 0 开始，并且速度为 +1 ，在一条无限长的数轴上行驶。赛车也可以向负方向行驶。赛车可以按照由加速指令 'A' 和倒车指令 'R' 组成的指令序列自动行驶

- 当收到指令 'A' 时，赛车这样行驶：
  - position += speed
  - speed *= 2
- 当收到指令 'R' 时，赛车这样行驶：
  - 如果速度为正数，那么speed = -1
  - 否则 speed = 1
  当前所处位置不变
  
例如，在执行指令 "AAR" 后，赛车位置变化为 0 --> 1 --> 3 --> 3 ，速度变化为 1 --> 2 --> 4 --> -1 

给你一个目标位置 target ，返回能到达目标位置的最短指令序列的长度

```js
Example 1:
Input: target = 3
Output: 2
Explanation: 
The shortest instruction sequence is "AA".
Your position goes from 0 --> 1 --> 3.

Example 2:
Input: target = 6
Output: 5
Explanation: 
The shortest instruction sequence is "AAARA".
Your position goes from 0 --> 1 --> 3 --> 7 --> 7 --> 6.
```

限制：
- 1 <= target <= 10^4
  
## 解题

可以使用动态规划来找到最短的指令长度

假设需要到达位置 x，且 2^k−1 ≤ x < 2^k，我们用 dp[x] 表示到达位置 x 的最短指令长度。如果 t=2^k−1
 ，那么我们只需要用 A^k 即可。否则我们需要考虑两种情况：

- 我们首先用 A^k−1 到达位置 2^k−1 − 1，随后折返并使用 A^j，这样我们到达了位置 2^k−1 −2^j，使用的指令为 A^k−1RA^kR，长度为 k−1+j−2，剩余的距离为 x−(2^k−1 − 2^j) < x

- 我们首先用 A^k 到达位置 2^k − 1，随后仅使用折返指令，此时我们已经超过了终点并且速度方向朝向终点，使用的指令为 A^kR，长度为 k+1，剩余的距离为 x−(2^k)−1 < x

```js
/**
 * @param {number} target
 * @return {number}
 */
var racecar = function(target) {
    const dp = new Array(target + 1).fill(0);

    for (let t = 1; t <= target; t++) {
        const k = t.toString(2).length;

        if (t === (1 << k) - 1) {
            dp[t] = k;
            continue;
        }

        dp[t] = Infinity;

        const over = (1 << k) - 1;
        dp[t] = Math.min(dp[t], k + 1 + dp[over - t]);

        const forward = (1 << (k - 1)) - 1;

        for (let j = 0; j < k - 1; j++) {
            const backward = (1 << j) - 1;
            const remaining = t - forward + backward;

            dp[t] = Math.min(
                dp[t],
                (k - 1) + 1 + j + 1 + dp[remaining]
            );
        }
    }

    return dp[target];
};
```
```python
from typing import List

class Solution:
    def racecar(self, target: int) -> int:
        dp = [0] * (target + 1)

        for t in range(1, target + 1):
            k = t.bit_length()

            # Case 1: exactly reach t by pressing A k times
            if t == (1 << k) - 1:
                dp[t] = k
                continue

            dp[t] = float("inf")

            # Case 2: go past t, reverse, then solve the remaining distance
            over = (1 << k) - 1
            dp[t] = min(dp[t], k + 1 + dp[over - t])

            # Case 3: go forward k-1 times, reverse, go back j times,
            # reverse again, then solve the remaining distance
            forward = (1 << (k - 1)) - 1
            for j in range(k - 1):
                backward = (1 << j) - 1
                remaining = t - forward + backward
                dp[t] = min(dp[t], (k - 1) + 1 + j + 1 + dp[remaining])

        return dp[target]
```

class Solution {
    public int racecar(int target) {
        int[] dp = new int[target + 1];

        for (int t = 1; t <= target; t++) {
            int k = Integer.toBinaryString(t).length();

            if (t == (1 << k) - 1) {
                dp[t] = k;
                continue;
            }

            dp[t] = Integer.MAX_VALUE;

            int over = (1 << k) - 1;
            dp[t] = Math.min(dp[t], k + 1 + dp[over - t]);

            int forward = (1 << (k - 1)) - 1;

            for (int j = 0; j < k - 1; j++) {
                int backward = (1 << j) - 1;
                int remaining = t - forward + backward;

                dp[t] = Math.min(
                    dp[t],
                    (k - 1) + 1 + j + 1 + dp[remaining]
                );
            }
        }

        return dp[target];
    }
