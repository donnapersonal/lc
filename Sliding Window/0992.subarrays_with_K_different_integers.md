# 992.K 个不同整数的子数组

题目链接：[992.K 个不同整数的子数组](https://leetcode.cn/problems/subarrays-with-k-different-integers/)

## 题目大意

给定一个正整数数组 nums和一个整数 k，返回 nums 中 「好子数组」 的数目

如果 nums 的某个子数组中不同整数的个数恰好为 k，则称 nums 的这个连续、不一定不同的子数组为 「好子数组 」

例如，[1,2,3,1,2] 中有 3 个不同的整数：1，2，以及 3

子数组 是数组的 连续 部分

```js
Example 1:
Input: nums = [1,2,1,2,3], k = 2
Output: 7
Explanation: Subarrays formed with exactly 2 different integers: [1,2], [2,1], [1,2], [2,3], [1,2,1], [2,1,2], [1,2,1,2]

Example 2:
Input: nums = [1,2,1,3,4], k = 3
Output: 3
Explanation: Subarrays formed with exactly 3 different integers: [1,2,1,3], [2,1,3], [1,3,4].
```

限制：
- 1 <= nums.length <= 2 * 10^4
- 1 <= nums[i], k <= nums.length

## 解题

**题目怎么理解？**

比如：
nums = [1, 2, 1, 2, 3]
k = 2

我们要找所有 连续子数组，并且这个子数组里面 不同数字的种类数刚好是 `2`

例如：

[1, 2]      不同数字：1,2      合法
[2, 1]      不同数字：2,1      合法
[1, 2, 1]   不同数字：1,2      合法
[2, 1, 2]   不同数字：1,2      合法
[1, 2, 1, 2] 不同数字：1,2     合法
[2, 3]      不同数字：2,3      合法

最终答案是：7

**核心技巧：恰好 K = 最多 K - 最多 K-1**

这题最重要的转换是：`恰好有 K 个不同整数的子数组数量 = 最多有 K 个不同整数的子数组数量 - 最多有 K - 1 个不同整数的子数组数量`

因为：
- 最多 K 个不同整数
  - 包含：1 种、2 种、3 种、...、K 种
- 最多 K-1 个不同整数
  - 包含：1 种、2 种、3 种、...、K-1 种
- 两者相减
- 剩下的就是：刚好 K 种

所以只需要写一个函数：`atMostK(nums, k)`，它返回：`最多包含 k 个不同整数的子数组数量`

最后答案就是：`atMostK(nums, k) - atMostK(nums, k - 1)`

**怎么求「最多 K 个不同整数」？**
- 用`滑动窗口`
- 我们维护一个窗口：`[left, right]` 并且保证窗口里面不同数字数量：`distinct <= k`
- 当 `right` 向右扩展时，把 `nums[right]` 加进窗口
- 如果窗口里的不同数字数量超过了 `k`，就移动 `left`，直到窗口重新合法

**为什么每次加 right - left + 1？**

这是这题最关键的地方

假设当前窗口 `[left, right]` 是合法的，也就是最多有 `k` 个不同数字

那么所有 `以 right` 结尾，且起点在 `left` 到 `right` 之间的子数组，都合法

也就是：
nums[left:right+1]
nums[left+1:right+1]
nums[left+2:right+1]
...
nums[right:right+1]

这些子数组数量是：`right - left + 1`

所以每次窗口合法后，我们都把这个数量加到答案里

```js
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
var subarraysWithKDistinct = function(nums, k) {
    return atMostK(nums, k) - atMostK(nums, k - 1);
};

function atMostK(nums, k) {
    if (k < 0) {
        return 0;
    }

    const freq = new Map();
    let left = 0;
    let distinct = 0;
    let res = 0;

    for (let right = 0; right < nums.length; right++) {
        const rightNum = nums[right];

        freq.set(rightNum, (freq.get(rightNum) || 0) + 1);

        if (freq.get(rightNum) === 1) {
            distinct++;
        }

        while (distinct > k) {
            const leftNum = nums[left];

            freq.set(leftNum, freq.get(leftNum) - 1);

            if (freq.get(leftNum) === 0) {
                distinct--;
            }

            left++;
        }

        res += right - left + 1;
    }

    return res;
}
```
```python
from collections import defaultdict
from typing import List

class Solution:
    def subarraysWithKDistinct(self, nums: List[int], k: int) -> int:
        return self.atMostK(nums, k) - self.atMostK(nums, k - 1)

    def atMostK(self, nums: List[int], k: int) -> int:
        if k < 0:
            return 0

        freq = defaultdict(int)
        left = 0
        distinct = 0
        res = 0

        for right in range(len(nums)):
            if freq[nums[right]] == 0:
                distinct += 1
            freq[nums[right]] += 1

            while distinct > k:
                freq[nums[left]] -= 1
                if freq[nums[left]] == 0:
                    distinct -= 1
                left += 1

            res += right - left + 1

        return res
```
```java
import java.util.*;

class Solution {
    public int subarraysWithKDistinct(int[] nums, int k) {
        return atMostK(nums, k) - atMostK(nums, k - 1);
    }

    private int atMostK(int[] nums, int k) {
        if (k < 0) {
            return 0;
        }

        Map<Integer, Integer> freq = new HashMap<>();
        int left = 0;
        int distinct = 0;
        int res = 0;

        for (int right = 0; right < nums.length; right++) {
            int rightNum = nums[right];

            freq.put(rightNum, freq.getOrDefault(rightNum, 0) + 1);

            if (freq.get(rightNum) == 1) {
                distinct++;
            }

            while (distinct > k) {
                int leftNum = nums[left];

                freq.put(leftNum, freq.get(leftNum) - 1);

                if (freq.get(leftNum) == 0) {
                    distinct--;
                }

                left++;
            }

            res += right - left + 1;
        }

        return res;
    }
}
```

- 时间复杂度：`O(n)`，这里 `n` 是输入数组的长度
- 空间复杂度：`O(n)`，使用了常数个变量、频数数组的长度为 `n+1`
