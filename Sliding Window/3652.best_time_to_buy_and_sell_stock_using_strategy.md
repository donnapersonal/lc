# 3652.按策略买卖股票的最佳时机

题目链接：[3652.按策略买卖股票的最佳时机](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-using-strategy/)

## 题目大意

给你两个整数数组 prices 和 strategy，其中：
- prices[i] 表示第 i 天某股票的价格
- strategy[i] 表示第 i 天的交易策略，其中：
  - -1 表示买入一单位股票
  - 0 表示持有股票
  - 1 表示卖出一单位股票
  
同时给你一个 偶数 整数 k，你可以对 strategy 进行 最多一次 修改。一次修改包括：
- 选择 strategy 中恰好 k 个 连续 元素
- 将前 k / 2 个元素设为 0（持有）
- 将后 k / 2 个元素设为 1（卖出）
  
利润 定义为所有天数中 strategy[i] * prices[i] 的 总和 

返回你可以获得的 最大 可能利润

注意： 没有预算或股票持有数量的限制，因此所有买入和卖出操作均可行，无需考虑过去的操作

- 1 <= prices[i] <= 10^5
- -1 <= strategy[i] <= 1
- 2 <= k <= prices.length
- k is even

## 解题

![详细题解](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-using-strategy/solutions/3755330/liang-chong-fang-fa-qian-zhui-he-ding-ch-uq98/)

```js
/**
 * Sliding Gain Version — O(1) extra space
 * @param {number[]} prices
 * @param {number[]} strategy
 * @param {number} k
 * @return {number}
 */
var maxProfit = function(prices, strategy, k) {
    let total = 0;
    let gain = 0;
    let maxGain = 0;

    for (let i = 0; i < prices.length; i++) {
        const price = prices[i];
        const st = strategy[i];

        // Original total profit
        total += price * st;

        // Current element enters the right half:
        // strategy changes from st to 1
        gain += price * (1 - st);

        if (i < k - 1) {
            // Before the first full window forms,
            // keep only k / 2 elements in the right half.
            if (i >= Math.floor(k / 2) - 1) {
                gain -= prices[i - Math.floor(k / 2) + 1];
            }
            continue;
        }

        // Current length-k window is valid
        maxGain = Math.max(maxGain, gain);

        // Prepare gain for the next window:
        // 1. one element moves from right half to left half
        // 2. one element leaves the modified window and restores original strategy
        gain -= prices[i - Math.floor(k / 2) + 1]
              - prices[i - k + 1] * strategy[i - k + 1];
    }

    return total + maxGain;
};
```
```python
class Solution:
    def maxProfit(self, prices: List[int], strategy: List[int], k: int) -> int:
        total = max_s = s = 0
        for i, (p, st) in enumerate(zip(prices, strategy)):
            total += p * st  # 不修改时的最大利润

            # 1. 下标为 i 的元素入右半，交易策略从 st 变成 1
            s += p * (1 - st)  # 修改带来的额外利润

            if i < k - 1:  # 尚未形成第一个窗口
                # 在下一轮循环中，下标为 i-k/2+1 的元素从右半移到左半，交易策略从 1 变成 0
                if i >= k // 2 - 1:  
                    s -= prices[i - k // 2 + 1]
                continue

            # 2. 更新
            max_s = max(max_s, s)  # 修改带来的最大额外利润

            # 3. 出，为下一个窗口做准备
            # 下标为 i-k/2+1 的元素从右半移到左半，交易策略从 1 变成 0
            # 下标为 i-k+1 的元素从左半离开窗口，交易策略从 0 恢复为 strategy[i-k+1]
            s -= prices[i - k // 2 + 1] - prices[i - k + 1] * strategy[i - k + 1]

        return total + max_s
```
```java
class Solution {
    public long maxProfit(int[] prices, int[] strategy, int k) {
        int n = prices.length;

        long total = 0;
        long gain = 0;
        long maxGain = 0;

        for (int i = 0; i < n; i++) {
            long p = prices[i];
            int st = strategy[i];

            // Original total profit
            total += p * st;

            // Current element enters the right half:
            // strategy changes from st to 1
            gain += p * (1 - st);

            if (i < k - 1) {
                // The first full window has not formed yet.
                // Move one element from right half to left half
                // once right half exceeds k / 2 elements.
                if (i >= k / 2 - 1) {
                    gain -= prices[i - k / 2 + 1];
                }
                continue;
            }

            // Current length-k window is valid.
            maxGain = Math.max(maxGain, gain);

            // Prepare for next window:
            // 1. middle element moves from right half to left half
            // 2. leftmost element leaves window and restores original strategy
            gain -= prices[i - k / 2 + 1]
                    - (long) prices[i - k + 1] * strategy[i - k + 1];
        }

        return total + maxGain;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 是 `prices` 的长度
- 空间复杂度：`O(1)`
