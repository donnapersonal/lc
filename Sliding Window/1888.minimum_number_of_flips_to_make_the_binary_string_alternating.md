# 1888.使二进制字符串字符交替的最少反转次数

题目链接：[1888.使二进制字符串字符交替的最少反转次数](https://leetcode.cn/problems/minimum-number-of-flips-to-make-the-binary-string-alternating/)

## 题目大意

给你一个二进制字符串 s 。你可以按任意顺序执行以下两种操作任意次：

- 类型 1 ：删除 字符串 s 的第一个字符并将它 添加 到字符串结尾
- 类型 2 ：选择 字符串 s 中任意一个字符并将该字符 反转 ，也就是如果值为 '0' ，则反转得到 '1' ，反之亦然
  
请你返回使 s 变成 交替 字符串的前提下， 类型 2 的 最少 操作次数 

我们称一个字符串是 交替 的，需要满足任意相邻字符都不同

比方说，字符串 "010" 和 "1010" 都是交替的，但是字符串 "0100" 不是

```js
Example 1:
Input: s = "111000"
Output: 2
Explanation: Use the first operation two times to make s = "100011".
Then, use the second operation on the third and sixth elements to make s = "101010".

Example 2:
Input: s = "010"
Output: 0
Explanation: The string is already alternating.

Example 3:
Input: s = "1110"
Output: 1
Explanation: Use the second operation on the second element to make s = "1010".
```

限制：
- 1 <= s.length <= 10^5
- s[i] is either '0' or '1'.

## 解题

```js
/**
 * @param {string} s
 * @return {number}
 */
/**
 * @param {string} s
 * @return {number}
 */
var minFlips = function(s) {
    const n = s.length;
    const doubled = s + s;

    let diff = 0;
    let res = n;

    for (let right = 0; right < 2 * n - 1; right++) {
        const expected = right % 2 === 0 ? '0' : '1';

        if (doubled[right] !== expected) {
            diff++;
        }

        const left = right - n + 1;

        if (left < 0) {
            continue;
        }

        res = Math.min(res, diff, n - diff);

        const expectedLeft = left % 2 === 0 ? '0' : '1';

        if (doubled[left] !== expectedLeft) {
            diff--;
        }
    }

    return res;
};
```

```java
class Solution {
    public int minFlips(String s) {
        int n = s.length();
        String doubled = s + s;

        int diff = 0;
        int res = n;

        for (int right = 0; right < 2 * n - 1; right++) {
            char expected = (right % 2 == 0) ? '0' : '1';

            if (doubled.charAt(right) != expected) {
                diff++;
            }

            int left = right - n + 1;

            if (left < 0) {
                continue;
            }

            res = Math.min(res, Math.min(diff, n - diff));

            char expectedLeft = (left % 2 == 0) ? '0' : '1';

            if (doubled.charAt(left) != expectedLeft) {
                diff--;
            }
        }

        return res;
    }
}
```
