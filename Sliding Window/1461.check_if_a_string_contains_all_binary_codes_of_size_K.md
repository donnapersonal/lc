# 1461.检查一个字符串是否包含所有长度为 K 的二进制子串

题目链接：[1461.检查一个字符串是否包含所有长度为 K 的二进制子串](https://leetcode.cn/problems/check-if-a-string-contains-all-binary-codes-of-size-k/)

## 题目大意

给你一个二进制字符串 s 和一个整数 k 。如果所有长度为 k 的二进制字符串都是 s 的子串，请返回 true ，否则请返回 false 

```js
Example 1:
Input: s = "00110110", k = 2
Output: true
Explanation: The binary codes of length 2 are "00", "01", "10" and "11". They can be all found as substrings at indices 0, 1, 3 and 2 respectively.

Example 2:
Input: s = "0110", k = 1
Output: true
Explanation: The binary codes of length 1 are "0" and "1", it is clear that both exist as a substring. 

Example 3:
Input: s = "0110", k = 2
Output: false
Explanation: The binary code "00" is of length 2 and does not exist in the array.
```

限制：
- 1 <= s.length <= 5 * 10^5
- s[i] is either '0' or '1'.
- 1 <= k <= 20

## 解题

**位运算滑窗**

把子串转成整数，保存到哈希集合或者布尔数组中

小优化：如果循环过程中发现已经找到 `2^k` 个不同的二进制数，可以提前返回 `true`

```js
/**
 * @param {string} s
 * @param {number} k
 * @return {boolean}
 */
var hasAllCodes = function(s, k) {
    const mask = (1 << k) - 1;
    const has = new Array(1 << k).fill(false);
    let cnt = 0;
    let x = 0;

    for (let i = 0; i < s.length; i++) {
        const bit = s.charCodeAt(i) - 48;
        x = ((x << 1) & mask) | bit;

        if (i < k - 1 || has[x]) {
            continue;
        }

        has[x] = true;
        cnt++;

        if (cnt === (1 << k)) {
            return true;
        }
    }

    return false;
};
```
```python
class Solution:
    def hasAllCodes(self, s: str, k: int) -> bool:
        MASK = (1 << k) - 1
        has = [False] * (1 << k)
        cnt = x = 0

        for i, ch in enumerate(s):
            x = (x << 1 & MASK) | int(ch)

            if i < k - 1 or has[x]:
                continue

            has[x] = True
            cnt += 1

            if cnt == 1 << k:
                return True

        return False
```
```java
class Solution {
    public boolean hasAllCodes(String s, int k) {
        int mask = (1 << k) - 1;
        boolean[] has = new boolean[1 << k];
        int cnt = 0;
        int x = 0;

        for (int i = 0; i < s.length(); i++) {
            int bit = s.charAt(i) - '0';
            x = ((x << 1) & mask) | bit;

            if (i < k - 1 || has[x]) {
                continue;
            }

            has[x] = true;
            cnt++;

            if (cnt == (1 << k)) {
                return true;
            }
        }

        return false;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 是 `s` 的长度
- 空间复杂度：`O(n−k)` 或 `O(2^k)`，取决于实现
