# 382.链表随机节点

题目链接：[382.链表随机节点](https://leetcode.cn/problems/linked-list-random-node/)

## 题目大意

给你一个单链表，随机选择链表的一个节点，并返回相应的节点值。每个节点 被选中的概率一样 。

实现 Solution 类：
- Solution(ListNode head) 使用整数数组初始化对象
- int getRandom() 从链表中随机选择一个节点并返回该节点的值。链表中所有节点被选中的概率相等

![alt text](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.2a5qj91hv8.webp)

限制：
- The number of nodes in the linked list will be in the range [1, 10^4].
- -10^4 <= Node.val <= 10^4
- At most 10^4 calls will be made to getRandom.

## 解题

因为链表长度可能不知道，而且不能像数组一样直接随机访问，所以可以用 蓄水池抽样

**核心思想**

遍历链表时，假设当前是第 `i` 个节点。用概率：`1 / i` 来决定是否把当前节点的值更新为答案

也就是说：

if randrange(i) == 0:
    ans = cur.val

其中：`randrange(i)` 会随机生成 `[0, i - 1]` 之间的整数

所以它等于 `0` 的概率就是：`1 / i`


**为什么这样每个节点概率相等？**

假设链表有 `n` 个节点。第 `1` 个节点一开始会被选中，概率是：`1`，之后它不能被后面的节点替换。第 `2` 个节点不替换它的概率是：`1 - 1/2 = 1/2`，第 `3` 个节点不替换它的概率是：`1 - 1/3 = 2/3`... 一直到第 `n` 个节点：`1 - 1/n = (n - 1) / n`

所以第 `1` 个节点最终留下来的概率是：`1 * 1/2 * 2/3 * 3/4 * ... * (n - 1)/n = 1/n`

第 `k` 个节点被选中的概率是：`1/k`，之后它需要不被第 `k+1` 到第 `n` 个节点替换：`k/(k+1) * (k+1)/(k+2) * ... * (n-1)/n
`
所以最终概率是：`1/k * k/(k+1) * (k+1)/(k+2) * ... * (n-1)/n = 1/n`

因此，每个节点最终被返回的概率都是：`1/n`

```js
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
class Solution {
    constructor(head) {
        this.head = head;
    }

    getRandom() {
        let cur = this.head;
        let i = 1;
        let ans = null;

        while (cur !== null) {
            if (Math.floor(Math.random() * i) === 0) {
                ans = cur.val;
            }

            cur = cur.next;
            i++;
        }

        return ans;
    }
}
```

```python
# Definition for singly-linked list.

from random import randrange
from typing import Optional

class Solution:
    def __init__(self, head: Optional[ListNode]):
        self.head = head

    def getRandom(self) -> int:
        cur = self.head
        i = 1
        ans = None

        while cur:
            if randrange(i) == 0:
                ans = cur.val

            cur = cur.next
            i += 1

        return ans
```

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
import java.util.Random;

class Solution {
    private ListNode head;
    private Random random;

    public Solution(ListNode head) {
        this.head = head;
        this.random = new Random();
    }

    public int getRandom() {
        ListNode cur = head;
        int i = 1;
        int ans = 0;

        while (cur != null) {
            if (random.nextInt(i) == 0) {
                ans = cur.val;
            }

            cur = cur.next;
            i++;
        }

        return ans;
    }
}
```

- 时间复杂度：`O(n)`
  - 每次调用 `getRandom()` 都需要遍历整个链表一次，其中 `n` 是链表节点数量
- 空间复杂度：`O(1)`，只用了几个变量：`cur, i, ans`，没有额外存储整个链表，所以空间复杂度是：`O(1)`
