# 60.排列序列

题目链接：[60.排列序列](https://leetcode.cn/problems/permutation-sequence/)

## 题目大意

给出集合 [1,2,3,...,n]，其所有元素共有 n! 种排列

按大小顺序列出所有排列情况，并一一标记，当 n = 3 时, 所有排列如下：

"123"
"132"
"213"
"231"
"312"
"321"

给定 n 和 k，返回第 k 个排列

```js
Example 1:
Input: n = 3, k = 3
Output: "213"

Example 2:
Input: n = 4, k = 9
Output: "2314"

Example 3:
Input: n = 3, k = 1
Output: "123"
```

限制:
- 1 <= n <= 9
- 1 <= k <= n!

## 解题

```js
/**
 * @param {number} n
 * @param {number} k
 * @return {string}
 */
var getPermutation = function(n, k) {
    const factorial = new Array(n).fill(1);

    for (let i = 1; i < n; i++) {
        factorial[i] = factorial[i - 1] * i;
    }

    const nums = [];

    for (let i = 1; i <= n; i++) {
        nums.push(String(i));
    }

    k--;

    const res = [];

    for (let i = n; i >= 1; i--) {
        const blockSize = factorial[i - 1];
        const index = Math.floor(k / blockSize);

        res.push(nums.splice(index, 1)[0]);
        k %= blockSize;
    }

    return res.join("");
};
```

class Solution:
    def getPermutation(self, n: int, k: int) -> str:
        factorial = [1] * n

        for i in range(1, n):
            factorial[i] = factorial[i - 1] * i

        nums = [str(i) for i in range(1, n + 1)]

        k -= 1
        res = []

        for i in range(n, 0, -1):
            block_size = factorial[i - 1]
            index = k // block_size

            res.append(nums.pop(index))
            k %= block_size

        return "".join(res)

```java
import java.util.*;

class Solution {
    public String getPermutation(int n, int k) {
        int[] factorial = new int[n];

        factorial[0] = 1;
        for (int i = 1; i < n; i++) {
            factorial[i] = factorial[i - 1] * i;
        }

        List<Integer> nums = new ArrayList<>();
        for (int i = 1; i <= n; i++) {
            nums.add(i);
        }

        k--;

        StringBuilder res = new StringBuilder();

        for (int i = n; i >= 1; i--) {
            int blockSize = factorial[i - 1];
            int index = k / blockSize;

            res.append(nums.remove(index));
            k %= blockSize;
        }

        return res.toString();
    }
}
```

- 空间复杂度：`O(n)`
