# 3625.统计梯形的数目 II

题目链接：[3625.统计梯形的数目 II](https://leetcode.cn/problems/count-number-of-trapezoids-ii/)

## 题目大意

给你一个二维整数数组 points，其中 points[i] = [xi, yi] 表示第 i 个点在笛卡尔平面上的坐标

返回可以从 points 中任意选择四个不同点组成的梯形的数量

梯形 是一种凸四边形，具有 至少一对 平行边。两条直线平行当且仅当它们的斜率相同

- –1000 <= xi, yi <= 1000
- All points are pairwise distinct.

## 解题

题目思路：枚举所有线段，按“平行关系”统计梯形

这题要数由点集组成的梯形数量。梯形的核心特征是：至少有一组对边平行

所以可以把问题转成：**从所有点对组成的线段中，找两条平行且不在同一条直线上的线段，它们就可以作为梯形的一组平行边**

**枚举所有点对，形成线段**

使用双重循环：

for i in range(n):
    for j in range(i):

枚举所有两点组合

每两个点可以形成一条线段

如果有 n 个点，一共有：`n * (n - 1) / 2` 条线段

**用斜率表示平行关系**

两条线段平行，当且仅当它们斜率相同

但是不能直接用浮点数斜率，比如：`dy / dx`，因为会有精度问题

所以代码用 `gcd` 把斜率化成整数 `pair`：`slope = (dy, dx)`

比如：`dy = 2, dx = 4`

可以约分成：`slope = (1, 2)`

这样斜率相同的线段会被分到同一组

特殊情况：

if dx == 0:
    slope = (1, 0)

表示竖直线

elif dy == 0:
    slope = (0, 1)

表示水平线

**用 line_key 区分是否在同一条直线上**

只有斜率相同还不够

因为同一条直线上的两条线段不能组成梯形的两条平行边

所以还要区分：`这条线段属于哪一条直线`

代码用：`line_key = sy * x1 - sx * y1` 作为直线的标识

对于特殊情况：

竖直线：line_key = x1
水平线：line_key = y1

所以：`line_groups[slope][line_key] += 1`

含义是：在某个斜率下，某条具体直线上，有多少条线段

*8统计平行但不共线的线段对**

对于同一个斜率 slope，如果有多条不同直线：

line A 上有 a 条线段
line B 上有 b 条线段
line C 上有 c 条线段

那么可以组成梯形的线段对数量是：`a*b + a*c + b*c`

代码用 running 滚动统计：

for lines in line_groups.values():
    running = 0
    for count in lines.values():
        res += running * count
        running += count

这个写法等价于统计所有不同直线之间的线段对

**为什么要减去平行四边形？**

一个普通梯形只有一组平行边，会被统计一次

但是平行四边形有两组平行边：AB ∥ CD, AD ∥ BC

所以同一个平行四边形会被统计两次

但题目通常只希望把它算作一个梯形，所以要减掉重复的一次

**如何识别平行四边形？**

平行四边形有一个经典性质：`两条对角线的中点相同`

所以代码对每条线段计算中点：`mid = (x1 + x2, y1 + y2)`

这里没有除以 2，因为用“双倍中点”可以避免小数

然后：`midpoint_groups[mid][slope] += 1` 表示：同一个中点下，不同斜率的线段数量

如果两条线段有相同中点，并且斜率不同，它们可以作为一个平行四边形的两条对角线

所以这部分重复统计要减掉：

for slopes in midpoint_groups.values():
    running = 0
    for count in slopes.values():
        res -= running * count
        running += count
      
**整体逻辑总结**

- 第一步：先多算
  - 按斜率分组，统计所有可以作为一组平行边的线段对：平行线段对数量
  - 这些线段对可以组成梯形
- 第二步：再减去重复
  - 平行四边形会因为有两组平行边被统计两次
  - 用“对角线中点相同”找出平行四边形，并减去重复的一次

```js
/**
 * @param {number[][]} points
 * @return {number}
 */
var countTrapezoids = function(points) {
    const lineGroups = new Map();      // slopeKey -> Map(lineKey -> count)
    const midpointGroups = new Map();  // midKey -> Map(slopeKey -> count)

    const gcd = (a, b) => {
        a = Math.abs(a);
        b = Math.abs(b);
        while (b !== 0) {
            const t = a % b;
            a = b;
            b = t;
        }
        return a;
    };

    const addToNestedMap = (map, key1, key2) => {
        if (!map.has(key1)) {
            map.set(key1, new Map());
        }

        const inner = map.get(key1);
        inner.set(key2, (inner.get(key2) || 0) + 1);
    };

    const n = points.length;

    for (let i = 0; i < n; i++) {
        const [x1, y1] = points[i];

        for (let j = 0; j < i; j++) {
            const [x2, y2] = points[j];

            let dx = x1 - x2;
            let dy = y1 - y2;

            let sy, sx;

            if (dx === 0) {
                sy = 1;
                sx = 0;
            } else if (dy === 0) {
                sy = 0;
                sx = 1;
            } else {
                const g = gcd(dx, dy);
                dx /= g;
                dy /= g;

                if (dx < 0) {
                    dx = -dx;
                    dy = -dy;
                }

                sy = dy;
                sx = dx;
            }

            const slopeKey = `${sy},${sx}`;

            let lineKey;
            if (sy === 1 && sx === 0) {
                lineKey = `${x1}`;
            } else if (sy === 0 && sx === 1) {
                lineKey = `${y1}`;
            } else {
                lineKey = `${sy * x1 - sx * y1}`;
            }

            const midKey = `${x1 + x2},${y1 + y2}`;

            addToNestedMap(lineGroups, slopeKey, lineKey);
            addToNestedMap(midpointGroups, midKey, slopeKey);
        }
    }

    let res = 0;

    for (const lines of lineGroups.values()) {
        let running = 0;
        for (const count of lines.values()) {
            res += running * count;
            running += count;
        }
    }

    for (const slopes of midpointGroups.values()) {
        let running = 0;
        for (const count of slopes.values()) {
            res -= running * count;
            running += count;
        }
    }

    return res;
};
```

```python
from collections import defaultdict
from math import gcd
from typing import List

class Solution:
    def countTrapezoids(self, points: List[List[int]]) -> int:
        line_groups = defaultdict(lambda: defaultdict(int))
        midpoint_groups = defaultdict(lambda: defaultdict(int))

        n = len(points)

        for i in range(n):
            x1, y1 = points[i]

            for j in range(i):
                x2, y2 = points[j]

                dx = x1 - x2
                dy = y1 - y2

                # Normalize slope.
                if dx == 0:
                    slope = (1, 0)
                elif dy == 0:
                    slope = (0, 1)
                else:
                    g = gcd(abs(dx), abs(dy))
                    dx //= g
                    dy //= g

                    # Normalize sign.
                    if dx < 0:
                        dx = -dx
                        dy = -dy

                    slope = (dy, dx)

                # Use a line key to distinguish parallel lines.
                # For normalized slope (dy, dx), one valid line invariant is:
                # dy * x - dx * y
                if slope == (1, 0):
                    line_key = x1
                elif slope == (0, 1):
                    line_key = y1
                else:
                    sy, sx = slope
                    line_key = sy * x1 - sx * y1

                # Use doubled midpoint to avoid fractions.
                mid = (x1 + x2, y1 + y2)

                line_groups[slope][line_key] += 1
                midpoint_groups[mid][slope] += 1

        res = 0

        # Count pairs of parallel segments on different lines.
        for lines in line_groups.values():
            running = 0
            for count in lines.values():
                res += running * count
                running += count

        # Subtract duplicated parallelograms.
        for slopes in midpoint_groups.values():
            running = 0
            for count in slopes.values():
                res -= running * count
                running += count

        return res
```

```java
import java.util.*;

class Solution {
    static class Pair {
        int a;
        int b;

        Pair(int a, int b) {
            this.a = a;
            this.b = b;
        }

        @Override
        public boolean equals(Object o) {
            if (!(o instanceof Pair)) return false;
            Pair other = (Pair) o;
            return this.a == other.a && this.b == other.b;
        }

        @Override
        public int hashCode() {
            return Objects.hash(a, b);
        }
    }

    public int countTrapezoids(int[][] points) {
        Map<Pair, Map<Integer, Integer>> lineGroups = new HashMap<>();
        Map<Pair, Map<Pair, Integer>> midpointGroups = new HashMap<>();

        int n = points.length;

        for (int i = 0; i < n; i++) {
            int x1 = points[i][0], y1 = points[i][1];

            for (int j = 0; j < i; j++) {
                int x2 = points[j][0], y2 = points[j][1];

                int dx = x1 - x2;
                int dy = y1 - y2;

                Pair slope;

                if (dx == 0) {
                    slope = new Pair(1, 0);
                } else if (dy == 0) {
                    slope = new Pair(0, 1);
                } else {
                    int g = gcd(Math.abs(dx), Math.abs(dy));
                    dx /= g;
                    dy /= g;

                    if (dx < 0) {
                        dx = -dx;
                        dy = -dy;
                    }

                    slope = new Pair(dy, dx);
                }

                int lineKey;
                if (slope.a == 1 && slope.b == 0) {
                    lineKey = x1;
                } else if (slope.a == 0 && slope.b == 1) {
                    lineKey = y1;
                } else {
                    int sy = slope.a, sx = slope.b;
                    lineKey = sy * x1 - sx * y1;
                }

                Pair mid = new Pair(x1 + x2, y1 + y2);

                lineGroups
                    .computeIfAbsent(slope, key -> new HashMap<>())
                    .put(lineKey, lineGroups.getOrDefault(slope, new HashMap<>()).getOrDefault(lineKey, 0) + 1);

                midpointGroups
                    .computeIfAbsent(mid, key -> new HashMap<>())
                    .put(slope, midpointGroups.getOrDefault(mid, new HashMap<>()).getOrDefault(slope, 0) + 1);
            }
        }

        int res = 0;

        for (Map<Integer, Integer> lines : lineGroups.values()) {
            int running = 0;
            for (int count : lines.values()) {
                res += running * count;
                running += count;
            }
        }

        for (Map<Pair, Integer> slopes : midpointGroups.values()) {
            int running = 0;
            for (int count : slopes.values()) {
                res -= running * count;
                running += count;
            }
        }

        return res;
    }

    private int gcd(int a, int b) {
        while (b != 0) {
            int temp = a % b;
            a = b;
            b = temp;
        }
        return a;
    }
}
```

- 时间复杂度: `O(n^2)`
  - 设点的数量是 `n`
  - 枚举所有点对：`O(n^2)`
  - 每个点对会做一次 gcd，所以更严格地说是：`O(n^2 * log C)`，其中 C 是坐标差值的最大范围
  - 通常面试里可以说：`O(n^2)`，因为 gcd 很快，坐标范围一般有限
  - 后面遍历哈希表的总元素数量也是所有线段数量级：`O(n^2)`
  - 所以总时间复杂度：`O(n^2)`
  - 更严格：`O(n^2 * log C)`
- 空间复杂度: `O(n^2)`
  - 所有点对形成的线段信息都会被统计进哈希表
  - 线段数量是：`O(n^2)`
  - 所以空间复杂度：`O(n^2)`
