# 939.最小面积矩形

题目链接：[939.最小面积矩形](https://leetcode.cn/problems/minimum-area-rectangle/)

## 题目大意

给你一个 X-Y 平面上的点数组 points，其中 points[i] = [xi, yi]

返回由这些点形成的矩形的最小面积，矩形的边与 X 轴和 Y 轴平行。如果不存在这样的矩形，则返回 0
  
请你在最优数对划分的方案下，返回最小的 最大数对和

限制:
- 1 <= points.length <= 500
- points[i].length == 2
- 0 <= xi, yi <= 4 * 10^4
- All the given points are unique.

## 解题

```js
/**
 * @param {number[][]} points
 * @return {number}
 */
var minAreaRect = function(points) {
    const pointSet = new Set();

    for (const [x, y] of points) {
        pointSet.add(`${x},${y}`);
    }

    let res = Infinity;
    const n = points.length;

    for (let i = 0; i < n; i++) {
        const [x1, y1] = points[i];

        for (let j = i + 1; j < n; j++) {
            const [x2, y2] = points[j];

            // Two points must be diagonal corners,
            // so they cannot share the same x or same y.
            if (x1 === x2 || y1 === y2) {
                continue;
            }

            // Check whether the other two corners exist.
            const p3 = `${x1},${y2}`;
            const p4 = `${x2},${y1}`;

            if (pointSet.has(p3) && pointSet.has(p4)) {
                const area = Math.abs(x1 - x2) * Math.abs(y1 - y2);
                res = Math.min(res, area);
            }
        }
    }

    return res === Infinity ? 0 : res;
};
```

```java
import java.util.*;

class Solution {
    public int minAreaRect(int[][] points) {
        Set<String> pointSet = new HashSet<>();

        for (int[] point : points) {
            int x = point[0];
            int y = point[1];
            pointSet.add(x + "," + y);
        }

        int res = Integer.MAX_VALUE;
        int n = points.length;

        for (int i = 0; i < n; i++) {
            int x1 = points[i][0];
            int y1 = points[i][1];

            for (int j = i + 1; j < n; j++) {
                int x2 = points[j][0];
                int y2 = points[j][1];

                // Two points must be diagonal corners,
                // so they cannot share the same x or same y.
                if (x1 == x2 || y1 == y2) {
                    continue;
                }

                // Check whether the other two corners exist.
                String p3 = x1 + "," + y2;
                String p4 = x2 + "," + y1;

                if (pointSet.contains(p3) && pointSet.contains(p4)) {
                    int area = Math.abs(x1 - x2) * Math.abs(y1 - y2);
                    res = Math.min(res, area);
                }
            }
        }

        return res == Integer.MAX_VALUE ? 0 : res;
    }
}
```

- 时间复杂度：`O(n^2)`
  - 点的数量是 `n`，点对数量大约是：`n * (n - 1) / 2` -> 时间复杂度是：`O(n^2)`
  - `HashSet` 平均是 `O(1)` 查询
- 空间复杂度：`O(n)`
  - 额外用了一个 `HashSet` 存所有点，最多存 `n` 个点，所以空间复杂度是：`O(n)`
