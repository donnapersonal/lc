# 3315.构造最小位运算数组 II

题目链接：[3315.构造最小位运算数组 II](https://leetcode.cn/problems/construct-the-minimum-bitwise-array-ii/)

## 题目大意

给你一个长度为 n 的 质数 数组 nums 。你的任务是返回一个长度为 n 的数组 ans ，对于每个下标 i ，以下 条件 均成立：

- ans[i] OR (ans[i] + 1) == nums[i]
  
除此以外，你需要 最小化 结果数组里每一个 ans[i] 

如果没法找到符合 条件 的 ans[i] ，那么 ans[i] = -1 

质数 指的是一个大于 1 的自然数，且它只有 1 和自己两个因数

```js
Example 1:
Input: nums = [2,3,5,7]
Output: [-1,1,4,3]
Explanation:
For i = 0, as there is no value for ans[0] that satisfies ans[0] OR (ans[0] + 1) = 2, so ans[0] = -1.
For i = 1, the smallest ans[1] that satisfies ans[1] OR (ans[1] + 1) = 3 is 1, because 1 OR (1 + 1) = 3.
For i = 2, the smallest ans[2] that satisfies ans[2] OR (ans[2] + 1) = 5 is 4, because 4 OR (4 + 1) = 5.
For i = 3, the smallest ans[3] that satisfies ans[3] OR (ans[3] + 1) = 7 is 3, because 3 OR (3 + 1) = 7.

Example 2:
Input: nums = [11,13,31]
Output: [9,12,15]
Explanation:
For i = 0, the smallest ans[0] that satisfies ans[0] OR (ans[0] + 1) = 11 is 9, because 9 OR (9 + 1) = 11.
For i = 1, the smallest ans[1] that satisfies ans[1] OR (ans[1] + 1) = 13 is 12, because 12 OR (12 + 1) = 13.
For i = 2, the smallest ans[2] that satisfies ans[2] OR (ans[2] + 1) = 31 is 15, because 15 OR (15 + 1) = 31.
```

限制：
- 1 <= nums.length <= 100
- 2 <= nums[i] <= 10^9
- nums[i] is a prime number.
 
## 解题

例如 x=100111，那么 x ∣ (x+1)=100111 ∣ 101000=101111

可以发现，x ∣ (x+1) 的本质是把二进制最右边的 0 置为 1

反过来，如果已知 x ∣ (x+1)=101111，那么倒推 x，需要把 101111 中的某个 1 变成 0。满足要求的 x 有：

100111
101011
101101
101110
 
其中最小的是 100111，也就是把 101111 最右边的 0 的右边的 1 置为 0

无解的情况：由于 x ∣ (x+1) 最低位一定是 1（因为 x 和 x+1 中必有一奇数），所以如果 nums[i] 是偶数（质数中只有 2），那么无解

把 101111 加一，得到 110000，再 AND 101111 取反后的值 010000，可以得到方法一中的 lowbit=10000

```js
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var minBitwiseArray = function(nums) {
    for (let i = 0; i < nums.length; i++) {
        const x = nums[i];

        if (x === 2) {
            nums[i] = -1;
        } else {
            nums[i] ^= (((x + 1) & ~x) >> 1);
        }
    }

    return nums;
};
```
```python
class Solution:
    def minBitwiseArray(self, nums: List[int]) -> List[int]:
        for i, x in enumerate(nums):
            if x == 2:
                nums[i] = -1
            else:
                nums[i] ^= ((x + 1) & ~x) >> 1
        return nums
```
```java
import java.util.*;

class Solution {
    public int[] minBitwiseArray(List<Integer> nums) {
        int n = nums.size();
        int[] res = new int[n];

        for (int i = 0; i < n; i++) {
            int x = nums.get(i);

            if (x == 2) {
                res[i] = -1;
            } else {
                res[i] = x ^ (((x + 1) & ~x) >> 1);
            }
        }

        return res;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 是 `nums` 的长度
- 空间复杂度：`O(1)`
