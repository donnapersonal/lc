# 3623.统计梯形的数目 I

题目链接：[3623.统计梯形的数目 I](https://leetcode.cn/problems/count-number-of-trapezoids-i/)

## 题目大意

给你一个二维整数数组 points，其中 points[i] = [xi, yi] 表示第 i 个点在笛卡尔平面上的坐标

水平梯形 是一种凸四边形，具有 至少一对 水平边（即平行于 x 轴的边）。两条直线平行当且仅当它们的斜率相同

返回可以从 points 中任意选择四个不同点组成的 水平梯形 数量

由于答案可能非常大，请返回结果对 109 + 7 取余数后的值

限制：
- 4 <= points.length <= 10^5
- –10^8 <= xi, yi <= 10^8
- All points are pairwise distinct.

## 解题

首先，统计每一行的点的个数，如果这一行有 c 个点，那么从 c 个点中选 2 个点，有 c(c−1)/2 种选法，可以组成一条水平边，即梯形的顶边或底边

枚举每一行，设这一行有 k = c(c−1)/2 条水平边，那么另外一条边就是之前遍历过的行的边数 s。根据乘法原理，之前遍历过的行与这一行，一共可以组成 s⋅k 个水平梯形，加入答案

⚠注意：另外一条边不能是其余所有行，这会导致重复计算

在最坏情况下，有两行，每行 n/2 个点，组成约 n^2/8 条线段，答案约为 n^4 / 64 = 1.5625×10^18
 ，这不超过 64 位整数最大值，所以无需在循环中取模

```js
/**
 * @param {number[][]} points
 * @return {number}
 */
var countTrapezoids = function(points) {
    const MOD = 1000000007n;
    const yCount = new Map();

    for (const [x, y] of points) {
        yCount.set(y, (yCount.get(y) || 0) + 1);
    }

    let res = 0n;
    let prevEdges = 0n;

    for (const count of yCount.values()) {
        const c = BigInt(count);
        const edges = c * (c - 1n) / 2n;

        res = (res + prevEdges * edges) % MOD;
        prevEdges = (prevEdges + edges) % MOD;
    }

    return Number(res);
};
```
```python
from collections import Counter
from typing import List

class Solution:
    def countTrapezoids(self, points: List[List[int]]) -> int:
        MOD = 1_000_000_007

        y_count = Counter(y for x, y in points)

        res = 0
        prev_edges = 0

        for count in y_count.values():
            edges = count * (count - 1) // 2
            res = (res + prev_edges * edges) % MOD
            prev_edges = (prev_edges + edges) % MOD

        return res
```
```java
import java.util.*;

class Solution {
    public int countTrapezoids(int[][] points) {
        final long MOD = 1_000_000_007L;

        Map<Integer, Integer> yCount = new HashMap<>();

        for (int[] p : points) {
            int y = p[1];
            yCount.put(y, yCount.getOrDefault(y, 0) + 1);
        }

        long res = 0;
        long prevEdges = 0;

        for (int count : yCount.values()) {
            long edges = (long) count * (count - 1) / 2;

            res = (res + prevEdges * edges) % MOD;
            prevEdges = (prevEdges + edges) % MOD;
        }

        return (int) res;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 是 `points` 的长度
- 空间复杂度：`O(n)`
