# 1922.统计好数字的数目

题目链接：[1922.统计好数字的数目](https://leetcode.cn/problems/count-good-numbers/)

## 题目大意

我们称一个数字字符串是 好数字 当它满足（下标从 0 开始）偶数 下标处的数字为 偶数 且 奇数 下标处的数字为 质数 （2，3，5 或 7）

比方说，"2582" 是好数字，因为偶数下标处的数字（2 和 8）是偶数且奇数下标处的数字（5 和 2）为质数。但 "3245" 不是 好数字，因为 3 在偶数下标处但不是偶数

给你一个整数 n ，请你返回长度为 n 且为好数字的数字字符串 总数 。由于答案可能会很大，请你将它对 109 + 7 取余后返回 

一个 数字字符串 是每一位都由 0 到 9 组成的字符串，且可能包含前导 0 

```js
Example 1:
Input: n = 1
Output: 5
Explanation: The good numbers of length 1 are "0", "2", "4", "6", "8".

Example 2:
Input: n = 4
Output: 400

Example 3:
Input: n = 50
Output: 564908303
```

限制:
- 1 <= n <= 10151 <= n <= 10^15

## 解题

对于偶数下标处的数字，它可以为 0,2,4,6,8 共计 5 种，而长度为 n 的数字字符串有 ⌊n+1/⌋ 个偶数下标，其中 ⌊x⌋ 表示对 x 向下取整

对于奇数下标处的数字，它可以为 2,3,5,7 共计 4 种，而长度为 n 的数字字符串有 ⌊n/2⌋ 个奇数下标

因此长度为 n 的数字字符串中，好数字的总数即为：5^⌊n+1/2⌋⋅4^⌊n/2⌋
 
在本题中，由于 n 的取值最大可以到 10^15，如果通过普通的乘法运算直接求出上式中的幂，会超出时间限制，因此需要使用快速幂算法对幂的求值进行优化

```js
/**
 * @param {number} n
 * @return {number}
 */
var countGoodNumbers = function(n) {
    const MOD = 1000000007n;

    function quickmul(x, y) {
        let ret = 1n;
        let mul = BigInt(x);

        while (y > 0n) {
            if (y % 2n === 1n) {
                ret = ret * mul % MOD;
            }

            mul = mul * mul % MOD;
            y = y / 2n;
        }

        return ret;
    }

    n = BigInt(n);

    const evenCount = (n + 1n) / 2n;
    const oddCount = n / 2n;

    return Number(quickmul(5, evenCount) * quickmul(4, oddCount) % MOD);
};
```
```python
class Solution:
    def countGoodNumbers(self, n: int) -> int:
        mod = 10**9 + 7
        
        def quickmul(x: int, y: int) -> int:
            ret, mul = 1, x
            while y > 0:
                if y % 2 == 1:
                    ret = ret * mul % mod
                mul = mul * mul % mod
                y //= 2
            return ret
            
        return quickmul(5, (n + 1) // 2) * quickmul(4, n // 2) % mod
```
```java
class Solution {
    private static final long MOD = 1_000_000_007L;

    public int countGoodNumbers(long n) {
        long evenCount = (n + 1) / 2;
        long oddCount = n / 2;

        long evenWays = quickMul(5, evenCount);
        long oddWays = quickMul(4, oddCount);

        return (int) (evenWays * oddWays % MOD);
    }

    private long quickMul(long x, long y) {
        long ret = 1;
        long mul = x;

        while (y > 0) {
            if (y % 2 == 1) {
                ret = ret * mul % MOD;
            }

            mul = mul * mul % MOD;
            y /= 2;
        }

        return ret;
    }
}
```

- 时间复杂度：`O(logn)`
- 空间复杂度：`O(1)`
