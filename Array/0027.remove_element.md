# 27.Remove Element

难度：Easy  
标签: Array、Two Pointers

## 题目

![27](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.45ibxhry2s.webp)

Constraints:
- 0 <= nums.length <= 100
- 0 <= nums[i] <= 50
- 0 <= val <= 100

## 解题

### Brute Force
> 每当 `nums[i] == val` 时：
> - 将 `nums[i]` 之后的所有元素向左移动
> - 将 `nums` 的长度减 1
> - 需重新检查当前位置，因为新的元素已移至该位置
>
> 时间复杂度：`O(n^2)`，空间复杂度：`O(1)`，最坏情况下，需移动被删除元素之后的大量元素
> 
> 瓶颈：
> - 每当找到 `val` 时，都会重复进行元素左移操作

```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        n = len(nums)
        i = 0
        while i < n:
            if nums[i] == val:
                for j in range(i, n-1):
                    nums[j] = nums[j+1]
                n -= 1
            else:
                i += 1
        
        return n
```

### 最优解
**Optimization**
> 只需要维护下一个应写入有效元素的位置
>
> 核心思路：
> 使用双指针：
>   - read（读指针） -> 遍历 nums 以寻找有效元素
>   - write（写指针） -> 指向下一个应写入有效元素的位置
> - 如果 nums[read] != val -> 令 nums[write] = nums[read]，并将 write 向前移动 1 位
> - 最后，write 的值即为有效元素的数量 k
>
> Invariant｜循环不变量
> - 在每次迭代之前，nums[0...write-1] 范围内的所有元素均为不等于 val 的有效元素

**Solution**
> - 初始化 write = 0
> - 使用 read 遍历 nums
> - 如果 nums[read] != val，将其放入 nums[write]
> - write += 1
> - 返回 write

```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        # Get the total number of elements in the list nums. n is used to define the range for our loop.
        n = len(nums)
        # Initialize a write pointer at index 0.
        # This pointer represents the next position where we should write a value that is not equal to val.
        write = 0
        # Start a loop using a read pointer, which goes from index 0 to n-1.
        # This pointer scans every element in the array.
        for read in range(n):
            # For each element nums[read], check if it's not equal to the target val. 
            # If it is not equal, we want to keep this element.
            if nums[read] != val:
                # Copy the value at fast to the position at write.
                # This effectively overwrites the positions of unwanted val values with valid ones.
                nums[write] = nums[read]
                # Move the slow pointer forward, indicating the next write position for a valid value.
                write += 1
        
        # After the loop ends, slow will be the count of values that are not equal to val.
        return write

# 如果不要求保留原数组顺序，可以使用对撞双指针，将目标值从前面替换为非目标值的尾部元素，从而减少写入操作，达到更优的性能
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        left, right = 0, len(nums)
        while left < right:
            if nums[left] == val:
                nums[left] = nums[right - 1]
                right -= 1
            else:
                left += 1
            
        return left
```
```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int n = nums.length;
        int write = 0;
        for (int read = 0; read < n; read++) {
            if (nums[read] != val) {
                nums[write] = nums[read];
                write++;
            }
        }
        return write;
    }
}
```

**Complexity**
> - Time: `O(n)`，每个元素处理最多 1 次
> - Space: `O(1)`，没有额外空间
