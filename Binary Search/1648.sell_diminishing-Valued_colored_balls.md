# 1648.销售价值减少的颜色球

题目链接：[1648.销售价值减少的颜色球](https://leetcode.cn/problems/sell-diminishing-valued-colored-balls/)

## 题目大意

你有一些球的库存 inventory ，里面包含着不同颜色的球。一个顾客想要 任意颜色 总数为 orders 的球

这位顾客有一种特殊的方式衡量球的价值：每个球的价值是目前剩下的 同色球 的数目。比方说还剩下 6 个黄球，那么顾客买第一个黄球的时候该黄球的价值为 6 。这笔交易以后，只剩下 5 个黄球了，所以下一个黄球的价值为 5 （也就是球的价值随着顾客购买同色球是递减的）

给你整数数组 inventory ，其中 inventory[i] 表示第 i 种颜色球一开始的数目。同时给你整数 orders ，表示顾客总共想买的球数目。你可以按照 任意顺序 卖球

请你返回卖了 orders 个球以后 最大 总价值之和。由于答案可能会很大，请你返回答案对 109 + 7 取余数 的结果

![example](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.3d5ekwslem.webp)

限制：
- 1 <= inventory.length <= 10^5
- 1 <= inventory[i] <= 10^9
- 1 <= orders <= min(sum(inventory[i]), 10^9)

## 解题

**二分查找**

可以二分查找 最后一次卖出时，球的价格 `x`。最后的答案由两部分组成：
- 对于所有数量 `> x` 的颜色，其肯定会减小到 `x`，因此用等差数列求和公式求和即可
- 如果执行完第 `1` 步，仍有剩余的 `orders`，则这些 `orders` 一定会以价格 `x` 卖出

```js
/**
 * @param {number[]} inventory
 * @param {number} orders
 * @return {number}
 */
var maxProfit = function(inventory, orders) {
    const MOD = 1000000007n;

    function countAbove(price) {
        let total = 0;

        for (const ai of inventory) {
            if (ai > price) {
                total += ai - price;
            }
        }

        return total;
    }

    let left = 0;
    let right = Math.max(...inventory);

    // Binary search for the smallest price T such that
    // number of balls with value > T is <= orders.
    while (left < right) {
        const mid = Math.floor((left + right) / 2);

        if (countAbove(mid) > orders) {
            left = mid + 1;
        } else {
            right = mid;
        }
    }

    const T = left;
    let res = 0n;

    // Sell all balls with value strictly greater than T.
    for (const ai of inventory) {
        if (ai > T) {
            const count = ai - T;

            // Sum: T + 1 + T + 2 + ... + ai
            const high = BigInt(ai);
            const low = BigInt(T + 1);
            const cnt = BigInt(count);

            res += (high + low) * cnt / 2n;
            res %= MOD;

            orders -= count;
        }
    }

    // Sell remaining balls at value T.
    res += BigInt(orders) * BigInt(T);
    res %= MOD;

    return Number(res);
```

```python
class Solution:
    def maxProfit(self, inventory: List[int], orders: int) -> int:
        MOD = 10 ** 9 + 7

        def count_above(price: int) -> int:
            """
            Count how many balls have value strictly greater than price.
            For a pile with ai balls, values are ai, ai-1, ..., 1.
            If ai > price, then we can sell ai - price balls above this price.
            """
            total = 0
            for ai in inventory:
                if ai > price:
                    total += ai - price
            return total

        # Binary search for the smallest price T such that
        # the number of balls with value > T is <= orders.
        left, right = 0, max(inventory)

        while left < right:
            mid = (left + right) // 2

            if count_above(mid) > orders:
                # If selling all balls above mid is still too many,
                # the threshold price must be higher.
                left = mid + 1
            else:
                # mid is a valid threshold, try to find a smaller one.
                right = mid

        T = left

        res = 0

        # First sell all balls with value strictly greater than T.
        for ai in inventory:
            if ai > T:
                count = ai - T

                # Sum of values: T+1 + T+2 + ... + ai
                res += (ai + T + 1) * count // 2
                res %= MOD

                orders -= count

        # Sell the remaining orders at value T.
        res += orders * T
        res %= MOD

        return res
```

```java
class Solution {
    private static final long MOD = 1_000_000_007L;

    public int maxProfit(int[] inventory, int orders) {
        int left = 0;
        int right = 0;

        for (int ai : inventory) {
            right = Math.max(right, ai);
        }

        // Binary search for the smallest price T such that
        // number of balls with value > T is <= orders.
        while (left < right) {
            int mid = left + (right - left) / 2;

            if (countAbove(inventory, mid, orders) > orders) {
                left = mid + 1;
            } else {
                right = mid;
            }
        }

        int T = left;
        long res = 0;

        // Sell all balls with value strictly greater than T.
        for (int ai : inventory) {
            if (ai > T) {
                long count = ai - T;

                // Sum: T + 1 + T + 2 + ... + ai
                long high = ai;
                long low = T + 1L;

                res += (high + low) * count / 2;
                res %= MOD;

                orders -= count;
            }
        }

        // Sell remaining balls at value T.
        res += (long) orders * T;
        res %= MOD;

        return (int) res;
    }

    private long countAbove(int[] inventory, int price, int orders) {
        long total = 0;

        for (int ai : inventory) {
            if (ai > price) {
                total += ai - price;
            }

            // Early stop to avoid unnecessary work and overflow risk.
            if (total > orders) {
                return total;
            }
        }

        return total;
    }
}
```

- 空间复杂度：`O(1)`
