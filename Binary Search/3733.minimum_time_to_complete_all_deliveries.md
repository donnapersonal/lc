# 3733.完成所有送货任务的最少时间

题目链接：[3733.完成所有送货任务的最少时间](https://leetcode.cn/problems/minimum-time-to-complete-all-deliveries/)

## 题目大意

给你两个大小为 2 的整数数组：d = [d1, d2] 和 r = [r1, r2]

两架送货无人机负责完成特定数量的送货任务。无人机 i 必须完成 di 次送货

每次送货花费 正好 一小时，并且在任何给定小时内 只有一架 无人机可以送货

此外，两架无人机都需要在特定时间间隔进行充电，在此期间它们不能送货。无人机 i 必须每 ri 小时充电一次（即在 ri 的倍数小时进行充电）

返回完成所有送货所需的 最小 总时间（以小时为单位）的整数

```js
Example 1:
Input: d = [3,1], r = [2,3]
Output: 5
Explanation:
The first drone delivers at hours 1, 3, 5 (recharges at hours 2, 4).
The second drone delivers at hour 2 (recharges at hour 3).

Example 2:
Input: d = [1,3], r = [2,2]
Output: 7
Explanation:
The first drone delivers at hour 3 (recharges at hours 2, 4, 6).
The second drone delivers at hours 1, 5, 7 (recharges at hours 2, 4, 6).

Example 3:
Input: d = [2,1], r = [3,4]
Output: 3
Explanation:
The first drone delivers at hours 1, 2 (recharges at hour 3).
The second drone delivers at hour 3.
```

限制：
- d = [d1, d2]
- 1 <= di <= 10^9
- r = [r1, r2]
- 2 <= ri <= 3 * 10^4
 
## 解题

题目核心：安排两类任务的最短时间

对应的题意可以理解为：

有两类任务：

d = [d1, d2]
r = [r1, r2]

第一类任务需要做 d1 次
第二类任务需要做 d2 次

在第 x 天：
- 如果 x 能被 r1 整除，这一天不能做第一类任务
- 如果 x 能被 r2 整除，这一天不能做第二类任务
- 每天最多做一个任务
  
问最少需要多少天，才能完成所有任务

**先理解一个基础问题**

先不看两类任务，只看一个问题：如果我需要完成 d 个任务，但是每逢 r 的倍数那天不能做，最少需要多少天？

比如：

d = 5
r = 3

也就是：需要做 5 个任务。第 3、6、9、12... 天不能做。

可用天数是：1, 2, 4, 5, 7, 8, ...

第 5 个可用天是：7 -》所以答案是 7

**公式 f(d, r) 是什么意思？**

代码里：

def f(d: int, r: int) -> int:
    return d + (d - 1) // (r - 1)

这个函数的意思是：找到完成 d 个“不被 r 整除的天”所需要的最少天数

也就是在 1, 2, 3, ..., t 里面，至少要有 d 个数不能被 r 整除

**为什么公式是 d + (d - 1) // (r - 1)？**

每连续 r 天里面：1, 2, 3, ..., r，其中只有一个数是 r 的倍数，也就是第 r 天不能用

所以每 r 天里，有：r - 1 天可以用

例如 r = 3：1, 2, 3, 4, 5, 6, 7, 8, 9...

可用天是：1, 2, 4, 5, 7, 8...

每 3 天里有 2 天可用

如果我要拿到 d 个可用天，那么本来至少需要 d 天

但是每经过一组 r - 1 个可用天，就会遇到 1 个不可用天，需要额外跳过一天

所以额外需要跳过的不可用天数量是：`(d - 1) // (r - 1)`

因此最少时间是：`d + (d - 1) // (r - 1)`

**为什么最后取三个值的最大值？**

代码最后：`return max(f(d1, r1), f(d2, r2), f(d1 + d2, l))`

这里有三个限制条件
- 条件 1：第一类任务必须有足够可用天
  - 第一类任务不能安排在 r1 的倍数天
  - 所以至少需要：f(d1, r1) 天，才能保证有 d1 天可以做第一类任务
- 条件 2：第二类任务必须有足够可用天
  - 第二类任务不能安排在 r2 的倍数天
  - 所以至少需要：f(d2, r2) 天，才能保证有 d2 天可以做第二类任务。
- 条件 3：总任务数必须有足够可用天
- 总共需要做：d1 + d2 个任务

每天最多做一个任务

有些天既不能做第一类，也不能做第二类

**什么天同时不能做两类任务？**

就是同时是 r1 和 r2 的倍数的天。也就是：lcm(r1, r2) 的倍数天

代码里：l = lcm(r1, r2)

所以，想完成总共 d1 + d2 个任务，至少需要有这么多天不是 l 的倍数

这个限制是： `f(d1 + d2, l)`

**为什么答案是三者最大值？**

因为最终时间必须同时满足三个条件：

- 1. 有足够天数做第一类任务
- 2. 有足够天数做第二类任务
- 3. 有足够总天数做所有任务

所以最少时间必须至少是：f(d1, r1)

也必须至少是：f(d2, r2)

也必须至少是：f(d1 + d2, lcm(r1, r2))

因此答案就是三者最大值：max(f(d1, r1), f(d2, r2), f(d1 + d2, l))

```js
/**
 * @param {number[]} d
 * @param {number[]} r
 * @return {number}
 */
var minimumTime = function(d, r) {
    const d1 = d[0];
    const d2 = d[1];
    const r1 = r[0];
    const r2 = r[1];

    const l = lcm(r1, r2);

    let left = d1 + d2 - 1;
    let right = (d1 + d2) * 2 - 1;

    while (left + 1 < right) {
        const mid = Math.floor((left + right) / 2);

        if (check(mid, d1, d2, r1, r2, l)) {
            right = mid;
        } else {
            left = mid;
        }
    }

    return right;
};

function check(t, d1, d2, r1, r2, l) {
    return d1 <= t - Math.floor(t / r1)
        && d2 <= t - Math.floor(t / r2)
        && d1 + d2 <= t - Math.floor(t / l);
}

function gcd(a, b) {
    while (b !== 0) {
        const temp = a % b;
        a = b;
        b = temp;
    }
    return a;
}

function lcm(a, b) {
    return Math.floor(a / gcd(a, b)) * b;
}
```
```python
class Solution:
    def minimumTime(self, d: List[int], r: List[int]) -> int:
        d1, d2 = d
        r1, r2 = r
        l = lcm(r1, r2)

        def check(t: int) -> bool:
            return d1 <= t - t // r1 and d2 <= t - t // r2 and d1 + d2 <= t - t // l

        left = d1 + d2 - 1
        right = (d1 + d2) * 2 - 1
        while left + 1 < right:
            mid = (left + right) // 2
            if check(mid):
                right = mid
            else:
                left = mid
        return right
```
```java
import java.util.*;

class Solution {
    public long minimumTime(int[] d, int[] r) {
        long d1 = d[0];
        long d2 = d[1];
        long r1 = r[0];
        long r2 = r[1];

        long l = lcm(r1, r2);

        long left = d1 + d2 - 1;
        long right = (d1 + d2) * 2 - 1;

        while (left + 1 < right) {
            long mid = left + (right - left) / 2;

            if (check(mid, d1, d2, r1, r2, l)) {
                right = mid;
            } else {
                left = mid;
            }
        }

        return right;
    }

    private boolean check(long t, long d1, long d2, long r1, long r2, long l) {
        return d1 <= t - t / r1
            && d2 <= t - t / r2
            && d1 + d2 <= t - t / l;
    }

    private long gcd(long a, long b) {
        while (b != 0) {
            long temp = a % b;
            a = b;
            b = temp;
        }
        return a;
    }

    private long lcm(long a, long b) {
        return a / gcd(a, b) * b;
    }
}
```

- 时间复杂度：`O(logmax(r1, r2)+log(d1 + d2))`。计算 `lcm` 需要 `O(logmax(r1,r2))` 时间，二分需要 `O(log(d1+d2))` 时间
- 空间复杂度：`O(1)`
