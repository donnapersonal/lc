# 792.匹配子序列的单词数

题目链接：[792.匹配子序列的单词数](https://leetcode.cn/problems/number-of-matching-subsequences/)

## 题目大意

给定字符串 s 和字符串数组 words, 返回  words[i] 中是s的子序列的单词个数 

字符串的 子序列 是从原始字符串中生成的新字符串，可以从中删去一些字符(可以是none)，而不改变其余字符的相对顺序

例如， “ace” 是 “abcde” 的子序列

```js
Example 1:
Input: s = "abcde", words = ["a","bb","acd","ace"]
Output: 3
Explanation: There are three strings in words that are a subsequence of s: "a", "acd", "ace".

Example 2:
Input: s = "dsahjpjauf", words = ["ahjpjau","ja","ahbwzgqnuk","tnmlanowax"]
Output: 2
```

限制：
- 1 <= s.length <= 5 * 10^4
- 1 <= words.length <= 5000
- 1 <= words[i].length <= 50
- s and words[i] consist of only lowercase English letters.

## 解题

**二分查找**

可以先用数组或哈希表 d 存放字符串 s 每个字符的下标，即 d[c] 为 s 中所有字符 c 的下标组成的数组

然后遍历 words 中的每个单词 w，通过二分查找的方法，判断 w 是否为 s 的子序列，是则答案加 1。判断逻辑如下：
- 定义指针 i 表示当前指向字符串 s 的第 i 个字符，初始化为 −1
- 遍历字符串 w 中的每个字符 c，在 d[c] 中二分查找第一个大于 i 的位置 j，如果不存在，则说明 w 不是 s 的子序列，直接跳出循环；否则，将 i 更新为 d[c][j]，继续遍历下一个字符
- 如果遍历完 w 中的所有字符，说明 w 是 s 的子序列

```js
/**
 * @param {string} s
 * @param {string[]} words
 * @return {number}
 */
var numMatchingSubseq = function(s, words) {
    const pos = new Map();

    for (let i = 0; i < s.length; i++) {
        const c = s[i];

        if (!pos.has(c)) {
            pos.set(c, []);
        }

        pos.get(c).push(i);
    }

    function upperBound(arr, target) {
        let left = 0;
        let right = arr.length;

        while (left < right) {
            const mid = Math.floor(left + (right - left) / 2);

            if (arr[mid] <= target) {
                left = mid + 1;
            } else {
                right = mid;
            }
        }

        return left;
    }

    function check(word) {
        let prev = -1;

        for (const c of word) {
            if (!pos.has(c)) {
                return false;
            }

            const list = pos.get(c);
            const idx = upperBound(list, prev);

            if (idx === list.length) {
                return false;
            }

            prev = list[idx];
        }

        return true;
    }

    let ans = 0;

    for (const word of words) {
        if (check(word)) {
            ans++;
        }
    }

    return ans;
};
```

```java
import java.util.*;

class Solution {
    public int numMatchingSubseq(String s, String[] words) {
        Map<Character, List<Integer>> pos = new HashMap<>();

        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);
            pos.computeIfAbsent(c, key -> new ArrayList<>()).add(i);
        }

        int ans = 0;

        for (String word : words) {
            if (check(word, pos)) {
                ans++;
            }
        }

        return ans;
    }

    private boolean check(String word, Map<Character, List<Integer>> pos) {
        int prev = -1;

        for (int i = 0; i < word.length(); i++) {
            char c = word.charAt(i);

            if (!pos.containsKey(c)) {
                return false;
            }

            List<Integer> list = pos.get(c);
            int idx = upperBound(list, prev);

            if (idx == list.size()) {
                return false;
            }

            prev = list.get(idx);
        }

        return true;
    }

    private int upperBound(List<Integer> list, int target) {
        int left = 0;
        int right = list.size();

        while (left < right) {
            int mid = left + (right - left) / 2;

            if (list.get(mid) <= target) {
                left = mid + 1;
            } else {
                right = mid;
            }
        }

        return left;
    }
}
```

- 空间复杂度: `O(m)`
