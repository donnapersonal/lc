# 3026.最大好子数组和

题目链接：[3026.最大好子数组和](https://leetcode.cn/problems/maximum-good-subarray-sum/)

## 题目大意

给你一个长度为 n 的数组 nums 和一个 正 整数 k 

如果 nums 的一个子数组中，第一个元素和最后一个元素 差的绝对值恰好 为 k ，我们称这个子数组为 好 的。换句话说，如果子数组 nums[i..j] 满足 |nums[i] - nums[j]| == k ，那么它是一个好子数组

请你返回 nums 中 好 子数组的 最大 和，如果没有好子数组，返回 0 

```js
Example 1:
Input: nums = [1,2,3,4,5,6], k = 1
Output: 11
Explanation: The absolute difference between the first and last element must be 1 for a good subarray. All the good subarrays are: [1,2], [2,3], [3,4], [4,5], and [5,6]. The maximum subarray sum is 11 for the subarray [5,6].

Example 2:
Input: nums = [-1,3,2,4,5], k = 3
Output: 11
Explanation: The absolute difference between the first and last element must be 3 for a good subarray. All the good subarrays are: [-1,3,2], and [2,4,5]. The maximum subarray sum is 11 for the subarray [2,4,5].

Example 3:
Input: nums = [-1,-2,-3,-4], k = 2
Output: -6
Explanation: The absolute difference between the first and last element must be 2 for a good subarray. All the good subarrays are: [-1,-2,-3], and [-2,-3,-4]. The maximum subarray sum is -6 for the subarray [-1,-2,-3].
```

限制：
- 2 <= nums.length <= 10^5
- -10^9 <= nums[i] <= 10^9
- 1 <= k <= 10^9
 
## 解题

```js
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
var maximumSubarraySum = function(nums, k) {
    const minPrefix = new Map();

    let prefix = 0;
    let res = -Infinity;

    for (const x of nums) {
        const left1 = x - k;
        const left2 = x + k;

        const p1 = minPrefix.has(left1) ? minPrefix.get(left1) : Infinity;
        const p2 = minPrefix.has(left2) ? minPrefix.get(left2) : Infinity;

        const bestStartPrefix = Math.min(p1, p2);

        if (bestStartPrefix !== Infinity) {
            res = Math.max(res, prefix + x - bestStartPrefix);
        }

        const oldPrefix = minPrefix.has(x) ? minPrefix.get(x) : Infinity;
        minPrefix.set(x, Math.min(oldPrefix, prefix));

        prefix += x;
    }

    return res === -Infinity ? 0 : res;
};
```
```python
class Solution:
    def maximumSubarraySum(self, nums: List[int], k: int) -> int:
        min_s = defaultdict(lambda: inf)
        s = 0
        res = -inf
        for x in nums:
            res = max(res, s + x - min(min_s[x - k], min_s[x + k]))
            min_s[x] = min(min_s[x], s)
            s += x
            
        return res if res > -inf else 0
```
```java
import java.util.*;

class Solution {
    public long maximumSubarraySum(int[] nums, int k) {
        Map<Integer, Long> minPrefix = new HashMap<>();

        long prefix = 0L;
        long res = Long.MIN_VALUE;

        for (int x : nums) {
            long p1 = minPrefix.getOrDefault(x - k, Long.MAX_VALUE);
            long p2 = minPrefix.getOrDefault(x + k, Long.MAX_VALUE);

            long bestStartPrefix = Math.min(p1, p2);

            if (bestStartPrefix != Long.MAX_VALUE) {
                res = Math.max(res, prefix + x - bestStartPrefix);
            }

            long oldPrefix = minPrefix.getOrDefault(x, Long.MAX_VALUE);
            minPrefix.put(x, Math.min(oldPrefix, prefix));

            prefix += x;
        }

        return res == Long.MIN_VALUE ? 0 : res;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 是 `nums` 的长度
- 空间复杂度：`O(n)`
