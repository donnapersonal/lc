# 1292.元素和小于等于阈值的正方形的最大边长

题目链接：[1292.元素和小于等于阈值的正方形的最大边长](https://leetcode.cn/problems/maximum-side-length-of-a-square-with-sum-less-than-or-equal-to-threshold/)

## 题目大意

给你一个大小为 m x n 的矩阵 mat 和一个整数阈值 threshold

请你返回元素总和小于或等于阈值的正方形区域的最大边长；如果没有这样的正方形区域，则返回 0 

![example](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.6wrckau6gd.webp)

限制：
- m == mat.length, n == mat[i].length
- 1 <= m, n <= 300
- 0 <= mat[i][j] <= 10^4
- 0 <= threshold <= 10^5

## 解题

预处理二维前缀和后，暴力的做法是写一个三重循环：
- 外面两重循环，枚举正方形的左上角 `(i,j)`
- 最内层循环，枚举正方形的边长为 `1,2,3,…` 直到出界或者正方形元素和超过 `threshold` 为止，在此过程中更新答案 `res` 的最大值
  
这样做的时间复杂度是 `O(mnmin(m,n))`

只需改一个地方，就能让算法的时间复杂度变小：
- 最内层循环，从 `res+1` 开始枚举
  
比如现在 `res=3`，那么枚举正方形的边长为 `1,2,3` 是毫无意义的，不会让答案变得更大。所以直接从 `res+1=4` 开始枚举更好


```js
/**
 * @param {number[][]} mat
 * @param {number} threshold
 * @return {number}
 */
var maxSideLength = function(mat, threshold) {
    const m = mat.length;
    const n = mat[0].length;

    // prefix sum matrix: size (m + 1) x (n + 1)
    const prefix = Array.from({ length: m + 1 }, () => Array(n + 1).fill(0));

    for (let i = 0; i < m; i++) {
        for (let j = 0; j < n; j++) {
            prefix[i + 1][j + 1] =
                prefix[i + 1][j] +
                prefix[i][j + 1] -
                prefix[i][j] +
                mat[i][j];
        }
    }

    // Return sum of submatrix:
    // top-left: (r1, c1)
    // bottom-right: (r2, c2)
    function query(r1, c1, r2, c2) {
        return (
            prefix[r2 + 1][c2 + 1] -
            prefix[r2 + 1][c1] -
            prefix[r1][c2 + 1] +
            prefix[r1][c1]
        );
    }

    let res = 0;

    for (let i = 0; i < m; i++) {
        for (let j = 0; j < n; j++) {
            while (
                i + res < m &&
                j + res < n &&
                query(i, j, i + res, j + res) <= threshold
            ) {
                res++;
            }
        }
    }

    return res;
};
```
```python
class Solution:
    def maxSideLength(self, mat: List[List[int]], threshold: int) -> int:
        m, n = len(mat), len(mat[0])
        s = [[0] * (n + 1) for _ in range(m + 1)]
        for i, row in enumerate(mat):
            for j, x in enumerate(row):
                s[i + 1][j + 1] = s[i + 1][j] + s[i][j + 1] - s[i][j] + x

        # 返回左上角在 (r1, c1)，右下角在 (r2, c2) 的子矩阵元素和
        def query(r1: int, c1: int, r2: int, c2: int) -> int:
            return s[r2 + 1][c2 + 1] - s[r2 + 1][c1] - s[r1][c2 + 1] + s[r1][c1]

        res = 0
        for i in range(m):
            for j in range(n):
                # 边长为 ans+1 的正方形，左上角在 (i, j)，右下角在 (i+res, j+res)
                while i + res < m and j + res < n and query(i, j, i + res, j + res) <= threshold:
                    res += 1

        return res
```
```java
class Solution {
    public int maxSideLength(int[][] mat, int threshold) {
        int m = mat.length;
        int n = mat[0].length;

        // prefix sum matrix: size (m + 1) x (n + 1)
        int[][] prefix = new int[m + 1][n + 1];

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                prefix[i + 1][j + 1] =
                    prefix[i + 1][j] +
                    prefix[i][j + 1] -
                    prefix[i][j] +
                    mat[i][j];
            }
        }

        int res = 0;

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                while (
                    i + res < m &&
                    j + res < n &&
                    query(prefix, i, j, i + res, j + res) <= threshold
                ) {
                    res++;
                }
            }
        }

        return res;
    }

    private int query(int[][] prefix, int r1, int c1, int r2, int c2) {
        return prefix[r2 + 1][c2 + 1]
             - prefix[r2 + 1][c1]
             - prefix[r1][c2 + 1]
             + prefix[r1][c1];
    }
}
```

- 时间复杂度：`O(mn)`，其中 `m` 和 `n` 分别是 `grid` 的行数和列数。虽然我们写了个三重循环，但由于答案最大是 `min(m,n)`，所以最内层的 `res++` 最多执行 `min(m,n)` 次，三重循环的时间复杂度为 `O(mn+min(m,n))=O(mn)`
- 空间复杂度：`O(mn)`
