# 1685.有序数组中差绝对值之和

题目链接：[1685.有序数组中差绝对值之和](https://leetcode.cn/problems/sum-of-absolute-differences-in-a-sorted-array/)

## 题目大意

给你一个 非递减 有序整数数组 nums 

请你建立并返回一个整数数组 result，它跟 nums 长度相同，且result[i] 等于 nums[i] 与数组中所有其他元素差的绝对值之和

换句话说， result[i] 等于 sum(|nums[i]-nums[j]|) ，其中 0 <= j < nums.length 且 j != i （下标从 0 开始）

```js
Example 1:
Input: nums = [2,3,5]
Output: [4,3,5]
Explanation: Assuming the arrays are 0-indexed, then
result[0] = |2-2| + |2-3| + |2-5| = 0 + 1 + 3 = 4,
result[1] = |3-2| + |3-3| + |3-5| = 1 + 0 + 2 = 3,
result[2] = |5-2| + |5-3| + |5-5| = 3 + 2 + 0 = 5.

Example 2:
Input: nums = [1,4,6,8,10]
Output: [24,15,13,15,21]
```

限制：
- 2 <= nums.length <= 10^5
- 1 <= nums[i] <= nums[i + 1] <= 10^4

## 解题

因为 nums 是非递减排序数组，所以对于当前位置 i，设：`x = nums[i]`

那么：
- 左边元素 <= x
- 右边元素 >= x

所以绝对值可以被拆掉

**对左边的贡献**

- 左边有 i 个元素：`nums[0], nums[1], ..., nums[i - 1]`
  - 因为它们都 <= x，所以：`|x - nums[j]| = x - nums[j]`
  - 左边总贡献：`(x - nums[0]) + (x - nums[1]) + ... + (x - nums[i - 1])`
  - 可以整理成：`x * i - leftSum`
  - 其中：`leftSum = nums[0] + nums[1] + ... + nums[i - 1]`
- 对右边的贡献
  - 右边有：`n - i - 1` 个元素：`nums[i + 1], nums[i + 2], ..., nums[n - 1]`
  - 因为它们都 `>= x`，所以：`|x - nums[j]| = nums[j] - x`
  - 右边总贡献：`(nums[i + 1] - x) + (nums[i + 2] - x) + ... + (nums[n - 1] - x)`

可以整理成：`rightSum - x * (n - i - 1)`

其中：`rightSum = nums[i + 1] + nums[i + 2] + ... + nums[n - 1]`

**最终公式**

所以：`res[i] = 左边贡献 + 右边贡献`

也就是：`res[i] = x * i - leftSum + rightSum - x * (n - i - 1)`

**为什么可以用 rolling sum？**

我们从左到右遍历数组

一开始：
- leftSum = 0
- rightSum = 所有元素之和

当遍历到 nums[i] = x 时：
- 第一步，把当前元素从右边移除：`rightSum -= x`
- 这时：
  - `leftSum = nums[0] + ... + nums[i - 1]`
  - `rightSum = nums[i + 1] + ... + nums[n - 1]`
- 正好可以用来计算：`左边贡献 + 右边贡献`

算完以后，把当前元素加入左边：`leftSum += x`

供下一个位置使用

```js
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var getSumAbsoluteDifferences = function(nums) {
    const n = nums.length;
    let leftSum = 0;
    let rightSum = nums.reduce((sum, num) => sum + num, 0);
    const res = new Array(n).fill(0);

    for (let i = 0; i < n; i++) {
        const x = nums[i];

        rightSum -= x;

        res[i] = x * i - leftSum + rightSum - x * (n - i - 1);

        leftSum += x;
    }

    return res;
};
```
```python
from typing import List

class Solution:
    def getSumAbsoluteDifferences(self, nums: List[int]) -> List[int]:
        n = len(nums)
        left_sum = 0
        right_sum = sum(nums)
        res = [0] * n

        for i, x in enumerate(nums):
            right_sum -= x

            left = x * i - left_sum
            right = right_sum - x * (n - i - 1)

            res[i] = left + right

            left_sum += x

        return res
```
```java
class Solution {
    public int[] getSumAbsoluteDifferences(int[] nums) {
        int n = nums.length;
        int leftSum = 0;
        int rightSum = 0;

        for (int num : nums) {
            rightSum += num;
        }

        int[] res = new int[n];

        for (int i = 0; i < n; i++) {
            int x = nums[i];

            rightSum -= x;

            res[i] = x * i - leftSum + rightSum - x * (n - i - 1);

            leftSum += x;
        }

        return res;
    }
}
```

- 时间复杂度：`O(n)`
  - 先遍历一次数组，计算 `rightSum` -> `O(n)`
  - 再遍历一次数组，计算每个位置的答案 -> `O(n)`
  - 所以总时间复杂度是：`O(n)`
- 空间复杂度：`O(1)` 额外空间
  - 除了返回结果数组 `res` 之外，只用了：`leftSum`、`rightSum`、`n`、`x`，这些常数变量
  - 所以额外空间复杂度是：`O(1)`
  - 如果把输出数组也算入空间，那么是：`O(n)`
