# 3355.零数组变换 I

题目链接：[3355.零数组变换 I](https://leetcode.cn/problems/zero-array-transformation-i/)

## 题目大意

给定一个长度为 n 的整数数组 nums 和一个二维数组 queries，其中 queries[i] = [li, ri]

对于每个查询 queries[i]：
- 在 nums 的下标范围 [li, ri] 内选择一个下标 子集
- 将选中的每个下标对应的元素值减 1
  
零数组 是指所有元素都等于 0 的数组

如果在按顺序处理所有查询后，可以将 nums 转换为 零数组 ，则返回 true，否则返回 false

```js
Example 1:
Input: nums = [1,0,1], queries = [[0,2]]
Output: true
Explanation:
For i = 0:
Select the subset of indices as [0, 2] and decrement the values at these indices by 1.
The array will become [0, 0, 0], which is a Zero Array.

Example 2:
Input: nums = [4,3,2,1], queries = [[1,3],[0,2]]
Output: false
Explanation:
For i = 0:
Select the subset of indices as [1, 2, 3] and decrement the values at these indices by 1.
The array will become [4, 2, 1, 0].
For i = 1:
Select the subset of indices as [0, 1, 2] and decrement the values at these indices by 1.
The array will become [3, 1, 0, 0], which is not a Zero Array.
```

限制：
- 1 <= nums.length <= 10^5
- 0 <= nums[i] <= 10^5
- 1 <= queries.length <= 10^5
- queries[i].length == 2
- 0 <= li <= ri < nums.length

## 解题

这题核心思路: **每个 query [l, r] 表示区间 l 到 r 的每个 nums[i] 最多可以减 1 次**

所以对每个位置 i 来说，我们只关心：**有多少个 queries 覆盖了 i**

如果位置 i 被覆盖了 coverage 次，那么它最多可以被减：coverage 次

要让 nums[i] 变成 0，必须满足：`coverage >= nums[i]`，否则这个位置不可能减到 0。

这可以用`差分数组`计算

> 差分数组可以把一次区间加 1 变成：
> - diff[l] += 1
> - diff[r + 1] -= 1
> 最后做 prefix sum，就能得到每个位置实际被覆盖多少次

![alt text](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.491x2incth.webp)

代码实现时，计算的是「区间减一」执行的次数。累加差分数组，可以得到在 nums[i] 处减一的次数 sumD

如果 nums[i]>sumD，意味着 nums[i] 减少后大于 0，不符合要求

```js
/**
 * @param {number[]} nums
 * @param {number[][]} queries
 * @return {boolean}
 */
var isZeroArray = function(nums, queries) {
    const n = nums.length;
    const diff = new Array(n + 1).fill(0);

    for (const [left, right] of queries) {
        diff[left] += 1;
        diff[right + 1] -= 1;
    }

    let coverage = 0;

    for (let i = 0; i < n; i++) {
        coverage += diff[i];

        if (coverage < nums[i]) {
            return false;
        }
    }

    return true;
};
```
```python
from typing import List

class Solution:
    def isZeroArray(self, nums: List[int], queries: List[List[int]]) -> bool:
        n = len(nums)
        diff = [0] * (n + 1)

        for left, right in queries:
            diff[left] += 1
            diff[right + 1] -= 1

        coverage = 0

        for i in range(n):
            coverage += diff[i]

            if coverage < nums[i]:
                return False

        return True
```
```java
class Solution {
    public boolean isZeroArray(int[] nums, int[][] queries) {
        int n = nums.length;
        int[] diff = new int[n + 1];

        for (int[] query : queries) {
            int left = query[0];
            int right = query[1];

            diff[left] += 1;
            diff[right + 1] -= 1;
        }

        int coverage = 0;

        for (int i = 0; i < n; i++) {
            coverage += diff[i];

            if (coverage < nums[i]) {
                return false;
            }
        }

        return true;
    }
}
```

- 时间复杂度：`O(n+q)`，其中 `n` 是 `nums` 的长度，`q` 是 `queries` 的长度
- 空间复杂度：`O(n)`
