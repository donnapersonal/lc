# 886.可能的二分法

题目链接：[886.可能的二分法](https://leetcode.cn/problems/possible-bipartition/)

## 题目大意

给定一组 n 人（编号为 1, 2, ..., n）， 我们想把每个人分进任意大小的两组。每个人都可能不喜欢其他人，那么他们不应该属于同一组

给定整数 n 和数组 dislikes ，其中 dislikes[i] = [ai, bi] ，表示不允许将编号为 ai 和  bi的人归入同一组。当可以用这种方法将所有人分进两组时，返回 true；否则返回 false

```js
Example 1:
Input: n = 4, dislikes = [[1,2],[1,3],[2,4]]
Output: true
Explanation: The first group has [1,4], and the second group has [2,3].

Example 2:
Input: n = 3, dislikes = [[1,2],[1,3],[2,3]]
Output: false
Explanation: We need at least 3 groups to divide them. We cannot put them in two groups.
```

限制:
- 1 <= n <= 2000
- 0 <= dislikes.length <= 10^4
- dislikes[i].length == 2
- 1 <= ai < bi <= n
- All the pairs of dislikes are unique.

## 解题

首先题目给定 n 个人为一组（编号为 1,2,…,n），其中 n 为偶数，并给出数组 dislike，其中 dislike[i]={ai,bi} 表示编号 ai 的用户不喜欢用户 bi，1 ≤ ai < bi ≤ n。现在判断是否能将 n 个人分成两组，并满足当一个人不喜欢某一个人时，该两人不在同一组中出现

可以尝试用「染色法」来解决问题：假设第一组中的人是红色，第二组中的人为蓝色。我们依次遍历每一个人，如果当前的人没有被分组，就将其分到第一组（即染为红色），那么这个人不喜欢的人必须分到第二组中（染为蓝色）。然后任何新被分到第二组中的人，其不喜欢的人必须被分到第一组，依此类推。如果在染色的过程中存在冲突，就表示这个任务是不可能完成的，否则说明染色的过程有效（即存在合法的分组方案）

可以通过「广度优先搜索」来实现「染色法」的过程

```js
/**
 * @param {number} n
 * @param {number[][]} dislikes
 * @return {boolean}
 */
var possibleBipartition = function(n, dislikes) {
    const graph = Array.from({ length: n }, () => []);

    for (const [x, y] of dislikes) {
        graph[x - 1].push(y - 1);
        graph[y - 1].push(x - 1);
    }

    const color = new Array(n).fill(0);

    for (let i = 0; i < n; i++) {
        if (color[i] === 0) {
            const queue = [i];
            let head = 0;
            color[i] = 1;

            while (head < queue.length) {
                const x = queue[head++];

                for (const y of graph[x]) {
                    if (color[y] === color[x]) {
                        return false;
                    }

                    if (color[y] === 0) {
                        color[y] = -color[x];
                        queue.push(y);
                    }
                }
            }
        }
    }

    return true;
};
```

```java
import java.util.*;

class Solution {
    public boolean possibleBipartition(int n, int[][] dislikes) {
        List<Integer>[] graph = new ArrayList[n];

        for (int i = 0; i < n; i++) {
            graph[i] = new ArrayList<>();
        }

        for (int[] dislike : dislikes) {
            int x = dislike[0] - 1;
            int y = dislike[1] - 1;

            graph[x].add(y);
            graph[y].add(x);
        }

        int[] color = new int[n];

        for (int i = 0; i < n; i++) {
            if (color[i] == 0) {
                Queue<Integer> queue = new LinkedList<>();
                queue.offer(i);
                color[i] = 1;

                while (!queue.isEmpty()) {
                    int x = queue.poll();

                    for (int y : graph[x]) {
                        if (color[y] == color[x]) {
                            return false;
                        }

                        if (color[y] == 0) {
                            color[y] = -color[x];
                            queue.offer(y);
                        }
                    }
                }
            }
        }

        return true;
    }
}
```

- 空间复杂度：`O(n+m)`，其中 `n` 题目给定的人数，`m` 为给定的 `dislike` 数组的大小
