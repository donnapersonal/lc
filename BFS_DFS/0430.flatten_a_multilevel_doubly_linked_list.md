# 430.展平多级双向链表

题目链接：[430.展平多级双向链表](https://leetcode.com/problems/flatten-a-multilevel-doubly-linked-list/)

## 题目大意

给定一个双向链表，其中包含的节点具有指向下一个节点的指针、指向前一个节点的指针以及一个额外的指向子节点的指针。这个子节点指针可能指向也可能不指向另一个单独的双向链表，该链表也包含这些特殊节点。这些子链表本身可能包含一个或多个子节点，依此类推，从而形成一个多层数据结构，如下例所示

给定head链表第一层的元素，将该链表扁平化，使所有节点都出现在一个单层双向链表中。设curr是一个带有子链表的节点。子链表中的节点在扁平化后的链表中应位于curr 之前和 curr.next 之后 

返回扁平化列表的最终结果head。列表中的所有节点都必须将其子指针设置为空null

## 解题

给你一个多级双向链表。每个节点有 4 个指针：

val
prev
next
child

普通双向链表是：1 <-> 2 <-> 3 <-> 4

但这道题中，某些节点可能还有 child 指针，指向另一条双向链表：

1 <-> 2 <-> 3 <-> 4
          |
          7 <-> 8 <-> 9
               |
               11 <-> 12

要把它展开成一个普通的双向链表：1 <-> 2 <-> 3 <-> 7 <-> 8 <-> 11 <-> 12 <-> 9 <-> 4

展开规则是：当前节点 -> child 链表 -> 原来的 next 链表

也就是 `DFS` 前序遍历

**核心思路：用栈模拟 DFS**

这题本质上是：**按照 DFS preorder 顺序，把所有节点重新串成一条双向链表**

对于每个节点 cur：
- 先处理 cur
- 再处理 cur.child
- 最后处理 cur.next

但是因为用的是`栈`，栈是：`后进先出 LIFO`，所以要注意入栈顺序

为了让 `child` 先被处理，应该：
- 先 push cur.next
- 再 push cur.child
- 因为 `child` 后入栈，所以它会先出栈


**为什么要用 pseudoHead？**

```python
pseudoHead = Node(0, None, head, None)
prev = pseudoHead
```

这是一个虚拟头节点

它的作用是让代码更统一

如果不用 pseudoHead，处理第一个节点时需要特殊判断：如果这是第一个节点，就不能设置 prev.next

用了 pseudoHead 以后，每个节点都可以统一这样连接：

```python
prev.next = cur
cur.prev = prev
```

最后再把虚拟头节点断开：

```python
pseudoHead.next.prev = None
return pseudoHead.next
```

```js
/**
 * // Definition for a Node.
 * function Node(val, prev, next, child) {
 *    this.val = val;
 *    this.prev = prev;
 *    this.next = next;
 *    this.child = child;
 * };
 */

/**
 * @param {Node} head
 * @return {Node}
 */
var flatten = function(head) {
    if (!head) {
        return head;
    }

    const pseudoHead = new Node(0, null, head, null);
    let prev = pseudoHead;
    const stack = [head];

    while (stack.length > 0) {
        const cur = stack.pop();

        prev.next = cur;
        cur.prev = prev;

        if (cur.next) {
            stack.push(cur.next);
        }

        if (cur.child) {
            stack.push(cur.child);
            cur.child = null;
        }

        prev = cur;
    }

    pseudoHead.next.prev = null;
    return pseudoHead.next;
};
```

```java
/*
// Definition for a Node.
class Node {
    public int val;
    public Node prev;
    public Node next;
    public Node child;
};
*/

class Solution {
    public Node flatten(Node head) {
        if (head == null) {
            return head;
        }

        Node pseudoHead = new Node(0, null, head, null);
        Node prev = pseudoHead;

        Deque<Node> stack = new ArrayDeque<>();
        stack.push(head);

        while (!stack.isEmpty()) {
            Node cur = stack.pop();

            prev.next = cur;
            cur.prev = prev;

            if (cur.next != null) {
                stack.push(cur.next);
            }

            if (cur.child != null) {
                stack.push(cur.child);
                cur.child = null;
            }

            prev = cur;
        }

        pseudoHead.next.prev = null;
        return pseudoHead.next;
    }
}
```

- 时间复杂度：`O(n)`，其中 n 是链表中所有节点的数量。每个节点最多入栈一次，出栈一次，处理一次。
- 空间复杂度：`O(n)`，最坏情况下，栈里可能存放很多节点
  - 如果链表结构非常宽，或者很多节点都有 next 和 child，栈空间可能达到 `O(n)`
