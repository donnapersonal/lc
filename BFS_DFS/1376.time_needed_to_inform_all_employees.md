# 1376.通知所有员工所需的时间

题目链接：[1376.通知所有员工所需的时间](https://leetcode.cn/problems/time-needed-to-inform-all-employees/)

## 题目大意

公司里有 n 名员工，每个员工的 ID 都是独一无二的，编号从 0 到 n - 1。公司的总负责人通过 headID 进行标识

在 manager 数组中，每个员工都有一个直属负责人，其中 manager[i] 是第 i 名员工的直属负责人。对于总负责人，manager[headID] = -1。题目保证从属关系可以用树结构显示

公司总负责人想要向公司所有员工通告一条紧急消息。他将会首先通知他的直属下属们，然后由这些下属通知他们的下属，直到所有的员工都得知这条紧急消息

第 i 名员工需要 informTime[i] 分钟来通知它的所有直属下属（也就是说在 informTime[i] 分钟后，他的所有直属下属都可以开始传播这一消息）

返回通知所有员工这一紧急消息所需要的 分钟数 

![example](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.mp5teit8.webp)

限制：
- 1 <= n <= 10^5
- 0 <= headID < n
- manager.length == n
- 0 <= manager[i] < n
- manager[headID] == -1
- informTime.length == n
- 0 <= informTime[i] <= 1000
- informTime[i] == 0 if employee i has no subordinates.
- It is guaranteed that all the employees can be informed.

## 解题

最贴合题目本质：公司上下级是一棵树

这题本质是：`manager -> direct reports` 也就是一棵从 `headID` 出发的树

每个经理通知所有直属下属需要 `informTime[manager]` 分钟

所以最终总时间就是：`从 headID 到最慢叶子员工路径上的 informTime 之和`，也可以理解成一棵树上的最大路径和

`return max_path_sum + informTime[x]`

意思是：`当前节点通知完所有下属后，总时间等于当前节点自己的通知时间，加上下属子树中最慢的那条路径`

```js
/**
 * @param {number} n
 * @param {number} headID
 * @param {number[]} manager
 * @param {number[]} informTime
 * @return {number}
 */
var numOfMinutes = function(n, headID, manager, informTime) {
    const graph = Array.from({ length: n }, () => []);

    for (let i = 0; i < n; i++) {
        const m = manager[i];

        if (m >= 0) {
            graph[m].push(i);
        }
    }

    function dfs(x) {
        let maxPathSum = 0;

        for (const y of graph[x]) {
            maxPathSum = Math.max(maxPathSum, dfs(y));
        }

        return maxPathSum + informTime[x];
    }

    return dfs(headID);
};
```

```java
import java.util.*;

class Solution {
    public int numOfMinutes(int n, int headID, int[] manager, int[] informTime) {
        List<List<Integer>> graph = new ArrayList<>();

        for (int i = 0; i < n; i++) {
            graph.add(new ArrayList<>());
        }

        for (int i = 0; i < n; i++) {
            int m = manager[i];

            if (m >= 0) {
                graph.get(m).add(i);
            }
        }

        return dfs(headID, graph, informTime);
    }

    private int dfs(int x, List<List<Integer>> graph, int[] informTime) {
        int maxPathSum = 0;

        for (int y : graph.get(x)) {
            maxPathSum = Math.max(maxPathSum, dfs(y, graph, informTime));
        }

        return maxPathSum + informTime[x];
    }
}
```

- 时间复杂度：`O(n)`，每个节点都恰好访问一次
- 空间复杂度：`O(n)`，最坏情况下，树退化成一条链，递归需要 `O(n)` 的栈空间
