# 2013.检测正方形

题目链接：[2013.检测正方形](https://leetcode.cn/problems/detect-squares/)

## 题目大意

给你一个在 X-Y 平面上的点构成的数据流。设计一个满足下述要求的算法：
- 添加 一个在数据流中的新点到某个数据结构中。可以添加 重复 的点，并会视作不同的点进行处理
- 给你一个查询点，请你从数据结构中选出三个点，使这三个点和查询点一同构成一个 面积为正 的 轴对齐正方形 ，统计 满足该要求的方案数目
  
轴对齐正方形 是一个正方形，除四条边长度相同外，还满足每条边都与 x-轴 或 y-轴 平行或垂直

实现 DetectSquares 类：
- DetectSquares() 使用空数据结构初始化对象
- void add(int[] point) 向数据结构添加一个新的点 point = [x, y]
- int count(int[] point) 统计按上述方式与点 point = [x, y] 共同构造 轴对齐正方形 的方案数

![example](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.8z763srqti.webp)

限制:
- point.length == 2
- 0 <= x, y <= 1000
- At most 3000 calls in total will be made to add and count.

## 解题

对于 add 操作，可以使用「哈希表 套 哈希表」的方式，以 {x, {y : 点 (x,y) 数量}} 的形式对传入点进行存储

对于 count 查询而言，假定传入的点为 (x,y)，我们可以先查询 x 行都有哪些列，枚举这些列（ 即枚举点 (x,ny) ），由 y 和 ny 可得正方形边长 len，此时再检查唯一确定的两点 (x±len,y) 和 (x±len,ny) 的出现次数，应用乘法原理，即可知道该正方形的方案数，统计所有合法方案数即是该询问的答案

利用题目范围给定的 x 和 y 具有明确的范围 0 <= x, y <= 1000，我们可以使用数组充当哈希表，但是为了拓展性和减少边界判断，即支持将平面拓展到任意大小，最好还是直接使用哈希表

```js
class DetectSquares {
    constructor() {
        this.points = new Map(); // x -> Map(y -> count)
    }

    /**
     * @param {number[]} point
     * @return {void}
     */
    add(point) {
        const [x, y] = point;

        if (!this.points.has(x)) {
            this.points.set(x, new Map());
        }

        const counter = this.points.get(x);
        counter.set(y, (counter.get(y) || 0) + 1);
    }

    /**
     * @param {number[]} point
     * @return {number}
     */
    count(point) {
        const [x, y] = point;

        if (!this.points.has(x)) {
            return 0;
        }

        let res = 0;
        const sameX = this.points.get(x);

        for (const [ny, cntVertical] of sameX.entries()) {
            if (ny === y) {
                continue;
            }

            const side = ny - y;

            for (const nx of [x + side, x - side]) {
                const otherX = this.points.get(nx);

                if (!otherX) {
                    continue;
                }

                const cnt1 = otherX.get(y) || 0;
                const cnt2 = otherX.get(ny) || 0;

                res += cntVertical * cnt1 * cnt2;
            }
        }

        return res;
    }
}
```
```python
from collections import defaultdict, Counter
from typing import List

class DetectSquares:

    def __init__(self):
        self.points = defaultdict(Counter)  # x -> Counter of y

    def add(self, point: List[int]) -> None:
        x, y = point
        self.points[x][y] += 1

    def count(self, point: List[int]) -> int:
        x, y = point
        res = 0

        # If no point has this x, no vertical side can be formed.
        if x not in self.points:
            return 0

        # Pick another point (x, ny) on the same vertical line.
        for ny, cnt_vertical in self.points[x].items():
            if ny == y:
                continue

            side = ny - y

            # Two possible squares:
            # one to the right, one to the left.
            for nx in (x + side, x - side):
                cnt1 = self.points[nx][y]
                cnt2 = self.points[nx][ny]

                res += cnt_vertical * cnt1 * cnt2

        return res
```

```java
import java.util.*;

class DetectSquares {
    private Map<Integer, Map<Integer, Integer>> points;

    public DetectSquares() {
        points = new HashMap<>();
    }

    public void add(int[] point) {
        int x = point[0];
        int y = point[1];

        points.putIfAbsent(x, new HashMap<>());
        Map<Integer, Integer> counter = points.get(x);
        counter.put(y, counter.getOrDefault(y, 0) + 1);
    }

    public int count(int[] point) {
        int x = point[0];
        int y = point[1];

        if (!points.containsKey(x)) {
            return 0;
        }

        int res = 0;
        Map<Integer, Integer> sameX = points.get(x);

        for (Map.Entry<Integer, Integer> entry : sameX.entrySet()) {
            int ny = entry.getKey();
            int cntVertical = entry.getValue();

            if (ny == y) {
                continue;
            }

            int side = ny - y;

            int[] candidates = new int[]{x + side, x - side};

            for (int nx : candidates) {
                Map<Integer, Integer> otherX = points.getOrDefault(nx, new HashMap<>());

                int cnt1 = otherX.getOrDefault(y, 0);
                int cnt2 = otherX.getOrDefault(ny, 0);

                res += cntVertical * cnt1 * cnt2;
            }
        }

        return res;
    }
}

 * DetectSquares obj = new DetectSquares();
 * obj.add(point);
 * int param_2 = obj.count(point);
 */
```

- 空间复杂度：`O(n)`
