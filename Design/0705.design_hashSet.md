# 705.设计哈希集合

题目链接：[705.设计哈希集合](https://leetcode.cn/problems/design-hashset/)

## 题目大意

不使用任何内建的哈希表库设计一个哈希集合（HashSet）

实现 MyHashSet 类：
- void add(key) 向哈希集合中插入值 key 
- bool contains(key) 返回哈希集合中是否存在这个值 key 
- void remove(key) 将给定值 key 从哈希集合中删除。如果哈希集合中没有这个值，什么也不做

```js
Example 1:
Input
["MyHashSet", "add", "add", "contains", "contains", "add", "contains", "remove", "contains"]
[[], [1], [2], [1], [3], [2], [2], [2], [2]]
Output
[null, null, null, true, false, null, true, null, false]

Explanation
MyHashSet myHashSet = new MyHashSet();
myHashSet.add(1);      // set = [1]
myHashSet.add(2);      // set = [1, 2]
myHashSet.contains(1); // return True
myHashSet.contains(3); // return False, (not found)
myHashSet.add(2);      // set = [1, 2]
myHashSet.contains(2); // return True
myHashSet.remove(2);   // set = [1]
myHashSet.contains(2); // return False, (already removed)
```

限制：
- 0 <= key <= 10^6
- At most 104 calls will be made to add, remove, and contains.

## 解题

### 思路 1: 哈希桶 + 拉链法 separate chaining

**核心思路**

准备一个固定大小的数组：self.buckets = [[] for _ in range(self.size)]

数组里的每个位置叫一个 bucket，每个 bucket 是一个小 list，用来存放 hash 到同一个位置的 key

**如何定位 key 应该放在哪里？**

用 hash 函数：

def _hash(self, key: int) -> int:
    return key % self.size

比如：

key = 1000
size = 769
bucket index = 1000 % 769 = 231

所以 1000 会被放进：self.buckets[231]

**如何处理哈希冲突？**

不同的 key 可能 hash 到同一个 bucket

比如：

5 % 769 = 5
774 % 769 = 5

所以 5 和 774 都会进入同一个 bucket：buckets[5] = [5, 774]

这种把冲突元素放在同一个链/列表里的方法叫：拉链法 separate chaining

**add(key)**
- 先通过 hash 找到对应 bucket
- 遍历 bucket，判断 key 是否已经存在
- 如果存在，什么都不做
- 如果不存在，把 key 加入 bucket
- 因为 HashSet 不能有重复元素，所以添加前要先检查

**remove(key)**
- 先找到 key 对应的 bucket
- 在 bucket 里查找 key
- 如果找到，就删除
- 如果没找到，什么都不做
  
**contains(key)**
- 通过 hash 找到对应 bucket
- 遍历 bucket
- 如果找到 key，返回 True
- 否则返回 False
  
**为什么 size = 769？**
- 769 是一个质数
- 用质数作为 bucket 数量，配合 % 取模，通常可以让 key 分布更均匀，减少冲突

```js
var MyHashSet = function() {
    this.size = 769;
    this.buckets = Array.from({ length: this.size }, () => []);
};

/** 
 * @param {number} key
 * @return {number}
 */
MyHashSet.prototype._hash = function(key) {
    return key % this.size;
};

/** 
 * @param {number} key
 * @return {void}
 */
MyHashSet.prototype.add = function(key) {
    const bucket = this.buckets[this._hash(key)];

    for (const num of bucket) {
        if (num === key) {
            return;
        }
    }

    bucket.push(key);
};

/** 
 * @param {number} key
 * @return {void}
 */
MyHashSet.prototype.remove = function(key) {
    const bucket = this.buckets[this._hash(key)];

    for (let i = 0; i < bucket.length; i++) {
        if (bucket[i] === key) {
            bucket.splice(i, 1);
            return;
        }
    }
};

/** 
 * @param {number} key
 * @return {boolean}
 */
MyHashSet.prototype.contains = function(key) {
    const bucket = this.buckets[this._hash(key)];

    for (const num of bucket) {
        if (num === key) {
            return true;
        }
    }

    return false;
};

/** 
 * Your MyHashSet object will be instantiated and called as such:
 * var obj = new MyHashSet()
 * obj.add(key)
 * obj.remove(key)
 * var param_3 = obj.contains(key)
 */
```

```python
class MyHashSet:

    def __init__(self):
        self.size = 769
        self.buckets = [[] for _ in range(self.size)]

    def _hash(self, key: int) -> int:
        return key % self.size

    def add(self, key: int) -> None:
        bucket = self.buckets[self._hash(key)]

        for num in bucket:
            if num == key:
                return

        bucket.append(key)

    def remove(self, key: int) -> None:
        bucket = self.buckets[self._hash(key)]

        for i, num in enumerate(bucket):
            if num == key:
                bucket.pop(i)
                return

    def contains(self, key: int) -> bool:
        bucket = self.buckets[self._hash(key)]

        for num in bucket:
            if num == key:
                return True

        return False
```

```java
import java.util.*;

class MyHashSet {
    private static final int SIZE = 769;
    private List<Integer>[] buckets;

    public MyHashSet() {
        buckets = new LinkedList[SIZE];

        for (int i = 0; i < SIZE; i++) {
            buckets[i] = new LinkedList<>();
        }
    }

    private int hash(int key) {
        return key % SIZE;
    }

    public void add(int key) {
        int idx = hash(key);
        List<Integer> bucket = buckets[idx];

        for (int num : bucket) {
            if (num == key) {
                return;
            }
        }

        bucket.add(key);
    }

    public void remove(int key) {
        int idx = hash(key);
        List<Integer> bucket = buckets[idx];

        Iterator<Integer> iterator = bucket.iterator();

        while (iterator.hasNext()) {
            int num = iterator.next();

            if (num == key) {
                iterator.remove();
                return;
            }
        }
    }

    public boolean contains(int key) {
        int idx = hash(key);
        List<Integer> bucket = buckets[idx];

        for (int num : bucket) {
            if (num == key) {
                return true;
            }
        }

        return false;
    }
}
```

**时间复杂度**

设：
- n = 总 key 数量
- m = bucket 数量，这里是 769
- k = 某个 bucket 里的元素数量

平均时间复杂度：O(1)

如果 hash 分布比较均匀，每个 bucket 里的元素数量很少

所以：

| 操作         | 平均时间复杂度 |
| ---------- | ------: |
| `add`      |  `O(1)` |
| `remove`   |  `O(1)` |
| `contains` |  `O(1)` |

最坏时间复杂度：O(n)

如果很多 key 都 hash 到同一个 bucket，比如全部进入同一个 list：buckets[0] = [key1, key2, key3, ..., keyn]

那么查找、删除、添加前检查都需要遍历整个 bucket

所以最坏情况是：

| 操作         | 最坏时间复杂度 |
| ---------- | ------: |
| `add`      |  `O(n)` |
| `remove`   |  `O(n)` |
| `contains` |  `O(n)` |

**空间复杂度：O(n + m)**

空间包括：
- bucket 数组本身：O(m)
- 存储所有 key：O(n)

所以总空间是：O(n + m)

因为这里 m = 769 是固定常数，也可以简化为：O(n)

**手写链表**

```python
class MyHashSet:

    class Node:
        def __init__(self, key):
            self.key = key
            self.next = None

    def __init__(self):
        self.size = 769
        self.buckets = [None] * self.size

    def _hash(self, key: int) -> int:
        return key % self.size

    def add(self, key: int) -> None:
        idx = self._hash(key)

        if self.buckets[idx] is None:
            self.buckets[idx] = self.Node(key)
            return

        cur = self.buckets[idx]
        while cur:
            if cur.key == key:
                return
            if cur.next is None:
                break
            cur = cur.next

        cur.next = self.Node(key)

    def remove(self, key: int) -> None:
        idx = self._hash(key)
        cur = self.buckets[idx]
        prev = None

        while cur:
            if cur.key == key:
                if prev:
                    prev.next = cur.next
                else:
                    self.buckets[idx] = cur.next
                return

            prev = cur
            cur = cur.next

    def contains(self, key: int) -> bool:
        idx = self._hash(key)
        cur = self.buckets[idx]

        while cur:
            if cur.key == key:
                return True
            cur = cur.next

        return False
```
- 时间复杂度：由于没有扩容的逻辑，最坏情况下复杂度为 O(n)，一般情况下复杂度为 O(1)
- 空间复杂度：O(1)

### 思路 2: 简单数组

由于题目给出了 0 <= key <= 10^6 数据范围，同时限定了 key 只能是 int

可以直接使用一个 boolean 数组记录某个 key 是否存在，key 直接对应 boolean 的下标

```python
class MyHashSet:

    def __init__(self):
        self.nodes = [False] * 1000001

    def add(self, key: int) -> None:
        self.nodes[key] = True

    def remove(self, key: int) -> None:
        self.nodes[key] = False

    def contains(self, key: int) -> bool:
        return self.nodes[key]
```

- 时间复杂度：O(1)
- 空间复杂度：O(1)

### 思路 3: bitmap

事实上还可以实现一个类似「bitmap」数据结构

使用 int 中的每一位代表一个位置

由于数据范围为 0 <= key <= 10^6，我们最多需要的 int 数量不会超过 40000

因此我们可以建立一个 buckets 数组，数组装载的 int 类型数值
- 先对 key 进行 key / 32，确定当前 key 所在桶的位置（大概位置）
- 再对 key 进行 key % 32，确定当前 key 所在桶中的哪一位（精确位置）
  
根据位运算对「精确位置」进行修改

```python
class MyHashSet:

    def __init__(self):
        # key 范围是 0 <= key <= 10^6
        # 一个 int 存 32 个 bit
        # 需要大约 1_000_001 / 32 个 bucket
        self.bits = [0] * 31251

    def add(self, key: int) -> None:
        bucket_idx = key // 32
        bit_idx = key % 32

        self.bits[bucket_idx] |= 1 << bit_idx

    def remove(self, key: int) -> None:
        bucket_idx = key // 32
        bit_idx = key % 32

        self.bits[bucket_idx] &= ~(1 << bit_idx)

    def contains(self, key: int) -> bool:
        bucket_idx = key // 32
        bit_idx = key % 32

        return (self.bits[bucket_idx] >> bit_idx) & 1 == 1
```

- 时间复杂度：`O(1)`
- 空间复杂度：`O(1)`
