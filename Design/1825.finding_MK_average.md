# 1825.求出 MK 平均值

题目链接：[1825.求出 MK 平均值](https://leetcode.cn/problems/finding-mk-average/)

## 题目大意

给你两个整数 m 和 k ，以及数据流形式的若干整数。你需要实现一个数据结构，计算这个数据流的 MK 平均值 

MK 平均值 按照如下步骤计算：
- 如果数据流中的整数少于 m 个，MK 平均值 为 -1 ，否则将数据流中最后 m 个元素拷贝到一个独立的容器中
- 从这个容器中删除最小的 k 个数和最大的 k 个数
- 计算剩余元素的平均值，并 向下取整到最近的整数 
  
请你实现 MKAverage 类：
- MKAverage(int m, int k) 用一个空的数据流和两个整数 m 和 k 初始化 MKAverage 对象
- void addElement(int num) 往数据流中插入一个新的元素 num 
- int calculateMKAverage() 对当前的数据流计算并返回 MK 平均数 ，结果需 向下取整到最近的整数 

```js
Example 1:
Input
["MKAverage", "addElement", "addElement", "calculateMKAverage", "addElement", "calculateMKAverage", "addElement", "addElement", "addElement", "calculateMKAverage"]
[[3, 1], [3], [1], [], [10], [], [5], [5], [5], []]
Output
[null, null, null, -1, null, 3, null, null, null, 5]

Explanation
MKAverage obj = new MKAverage(3, 1); 
obj.addElement(3);        // current elements are [3]
obj.addElement(1);        // current elements are [3,1]
obj.calculateMKAverage(); // return -1, because m = 3 and only 2 elements exist.
obj.addElement(10);       // current elements are [3,1,10]
obj.calculateMKAverage(); // The last 3 elements are [3,1,10].
                          // After removing smallest and largest 1 element the container will be [3].
                          // The average of [3] equals 3/1 = 3, return 3
obj.addElement(5);        // current elements are [3,1,10,5]
obj.addElement(5);        // current elements are [3,1,10,5,5]
obj.addElement(5);        // current elements are [3,1,10,5,5,5]
obj.calculateMKAverage(); // The last 3 elements are [5,5,5].
                          // After removing smallest and largest 1 element the container will be [5].
                          // The average of [5] equals 5/1 = 5, return 5
```

限制：
- 3 <= m <= 10^5
- 1 < k*2 < m
- 1 <= num <= 10^5
- At most 10^5 calls will be made to addElement and calculateMKAverage.

## 解题

这题要设计一个数据结构，支持两个操作：
- addElement(num)
- calculateMKAverage()

MK Average 的定义是：只看最近 m 个数，去掉最小的 k 个和最大的 k 个，然后对剩下的数求平均值，向下取整

**核心思路：维护三个有序集合**

把最近 m 个数分成三部分：
- lo   = 最小的 k 个数
- mid  = 中间的 m - 2k 个数
- hi   = 最大的 k 个数

真正要求平均的是 `mid`

所以只要维护：`mid_sum = mid 中所有数的和`

那么：`MKAverage = mid_sum // (m - 2 * k)`

**为什么需要队列？**

因为题目只关心最近 `m` 个数

每次加入新数后，如果数量超过 m，就要删除最早加入的那个数

所以用队列：`q = deque()` 保存加入顺序

**addElement 的过程**

每次加入一个新数 num，做三件事
- 把新数放进合适的集合
  - 如果 num 很小，放入 lo
  - 如果 num 很大，放入 hi
  - 否则放入 mid，并更新 mid_sum
- 如果窗口超过 m，删除最旧的数
  - 然后判断 old 在哪个集合里：
    - 如果在 lo，从 lo 删除
    - 如果在 hi，从 hi 删除
    - 如果在 mid，从 mid 删除，并且 mid_sum -= old
- 重新平衡三个集合
  - 目标始终是：len(lo) == k、len(hi) == k、mid 存中间剩下的数
  - 如果 lo 太多，就把 lo 里最大的数移到 mid
  - 如果 hi 太多，就把 hi 里最小的数移到 mid
  - 如果 lo 不够，就从 mid 里拿最小的数补给 lo
  - 如果 hi 不够，就从 mid 里拿最大的数补给 hi
  - 每次移动进出 mid，都要同步更新 mid_sum

**calculateMKAverage**

如果当前元素数量还不到 m：`return -1`

否则：`return mid_sum // (m - 2 * k)`

```python
from collections import deque
from sortedcontainers import SortedList

class MKAverage:

    def __init__(self, m: int, k: int):
        self.m = m
        self.k = k
        self.q = deque()

        self.lo = SortedList()   # smallest k elements
        self.mid = SortedList()  # middle elements
        self.hi = SortedList()   # largest k elements

        self.mid_sum = 0

    def addElement(self, num: int) -> None:
        # Step 1: insert num into one of the three groups
        if not self.lo or num <= self.lo[-1]:
            self.lo.add(num)
        elif not self.hi or num >= self.hi[0]:
            self.hi.add(num)
        else:
            self.mid.add(num)
            self.mid_sum += num

        self.q.append(num)

        # Step 2: remove oldest element if window size exceeds m
        if len(self.q) > self.m:
            old = self.q.popleft()

            if old in self.lo:
                self.lo.remove(old)
            elif old in self.hi:
                self.hi.remove(old)
            else:
                self.mid.remove(old)
                self.mid_sum -= old

        # Step 3: rebalance groups
        while len(self.lo) > self.k:
            x = self.lo.pop()
            self.mid.add(x)
            self.mid_sum += x

        while len(self.hi) > self.k:
            x = self.hi.pop(0)
            self.mid.add(x)
            self.mid_sum += x

        while len(self.lo) < self.k and self.mid:
            x = self.mid.pop(0)
            self.lo.add(x)
            self.mid_sum -= x

        while len(self.hi) < self.k and self.mid:
            x = self.mid.pop()
            self.hi.add(x)
            self.mid_sum -= x

    def calculateMKAverage(self) -> int:
        if len(self.q) < self.m:
            return -1

        return self.mid_sum // (self.m - 2 * self.k)
```

```java
import java.util.*;

class MKAverage {
    private int m, k;
    private Queue<Integer> queue;
    private TreeMap<Integer, Integer> lo;
    private TreeMap<Integer, Integer> mid;
    private TreeMap<Integer, Integer> hi;
    private int loSize, midSize, hiSize;
    private long midSum;

    public MKAverage(int m, int k) {
        this.m = m;
        this.k = k;
        this.queue = new ArrayDeque<>();
        this.lo = new TreeMap<>();
        this.mid = new TreeMap<>();
        this.hi = new TreeMap<>();
        this.loSize = 0;
        this.midSize = 0;
        this.hiSize = 0;
        this.midSum = 0;
    }

    public void addElement(int num) {
        if (loSize == 0 || num <= lo.lastKey()) {
            add(lo, num);
            loSize++;
        } else if (hiSize == 0 || num >= hi.firstKey()) {
            add(hi, num);
            hiSize++;
        } else {
            add(mid, num);
            midSize++;
            midSum += num;
        }

        queue.offer(num);

        if (queue.size() > m) {
            int old = queue.poll();

            if (contains(lo, old)) {
                remove(lo, old);
                loSize--;
            } else if (contains(hi, old)) {
                remove(hi, old);
                hiSize--;
            } else {
                remove(mid, old);
                midSize--;
                midSum -= old;
            }
        }

        rebalance();
    }

    public int calculateMKAverage() {
        if (queue.size() < m) {
            return -1;
        }

        return (int) (midSum / (m - 2 * k));
    }

    private void rebalance() {
        while (loSize > k) {
            int x = lo.lastKey();
            remove(lo, x);
            loSize--;

            add(mid, x);
            midSize++;
            midSum += x;
        }

        while (hiSize > k) {
            int x = hi.firstKey();
            remove(hi, x);
            hiSize--;

            add(mid, x);
            midSize++;
            midSum += x;
        }

        while (loSize < k && midSize > 0) {
            int x = mid.firstKey();
            remove(mid, x);
            midSize--;
            midSum -= x;

            add(lo, x);
            loSize++;
        }

        while (hiSize < k && midSize > 0) {
            int x = mid.lastKey();
            remove(mid, x);
            midSize--;
            midSum -= x;

            add(hi, x);
            hiSize++;
        }
    }

    private void add(TreeMap<Integer, Integer> map, int x) {
        map.put(x, map.getOrDefault(x, 0) + 1);
    }

    private void remove(TreeMap<Integer, Integer> map, int x) {
        int count = map.get(x);
        if (count == 1) {
            map.remove(x);
        } else {
            map.put(x, count - 1);
        }
    }

    private boolean contains(TreeMap<Integer, Integer> map, int x) {
        return map.containsKey(x);
    }
}
```

- 空间复杂度: `O(m)`
