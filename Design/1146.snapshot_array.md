# 1146.快照数组

题目链接：[1146.快照数组](https://leetcode.cn/problems/snapshot-array/)

## 题目大意

实现支持下列接口的「快照数组」- SnapshotArray：
- SnapshotArray(int length) - 初始化一个与指定长度相等的 类数组 的数据结构。初始时，每个元素都等于 0
- void set(index, val) - 会将指定索引 index 处的元素设置为 val
- int snap() - 获取该数组的快照，并返回快照的编号 snap_id（快照号是调用 snap() 的总次数减去 1）
- int get(index, snap_id) - 根据指定的 snap_id 选择快照，并返回该快照指定索引 index 的值

```js
Example 1:
Input: ["SnapshotArray","set","snap","set","get"]
[[3],[0,5],[],[0,6],[0,0]]
Output: [null,null,0,null,5]
Explanation: 
SnapshotArray snapshotArr = new SnapshotArray(3); // set the length to be 3
snapshotArr.set(0,5);  // Set array[0] = 5
snapshotArr.snap();  // Take a snapshot, return snap_id = 0
snapshotArr.set(0,6);
snapshotArr.get(0,0);  // Get the value of array[0] with snap_id = 0, return 5
```

限制:
- 1 <= length <= 5 * 10^4
- 0 <= index < length
- 0 <= val <= 10^9
- 0 <= snap_id < (the total number of times we call snap())
- At most 5 * 104 calls will be made to set, snap, and get.

## 解题

对数组中的每个位置 i 维护一个数组 data[i]，其中存储了若干二元组，按照时间顺序记录了每一次对位置 i 的修改操作

对于操作 set(i, val)，记当前的快照编号（这里指下一次调用 snap() 操作会返回的编号）为 snap_cnt，那么在 data[i] 的末尾放入二元组 (snap_cnt, val) 即可

这样一来，对于 get(i, snap_id) 操作，我们就可以通过在 data[i] 上进行二分查找，得到第 snap_id 个快照对应的值。具体地，对于 data[i] 中的二元组 (snap_cnt, val)，我们需要找到最后一个出现的，并且满足 snap_cnt ≤ snap_id 的二元组，此时它的 val 即为答案。如果不存在这样的二元组，说明在第 snap_id 个快照时，它的值没有被修改过，此时答案为 0

```js
class SnapshotArray {
    /**
     * @param {number} length
     */
    constructor(length) {
        this.snapId = 0;
        this.data = Array.from({ length }, () => []);
    }

    /**
     * @param {number} index
     * @param {number} val
     * @return {void}
     */
    set(index, val) {
        const records = this.data[index];

        if (records.length > 0 && records[records.length - 1][0] === this.snapId) {
            records[records.length - 1] = [this.snapId, val];
        } else {
            records.push([this.snapId, val]);
        }
    }

    /**
     * @return {number}
     */
    snap() {
        const cur = this.snapId;
        this.snapId++;
        return cur;
    }

    /**
     * @param {number} index
     * @param {number} snap_id
     * @return {number}
     */
    get(index, snap_id) {
        const records = this.data[index];

        let left = 0;
        let right = records.length - 1;
        let ans = -1;

        while (left <= right) {
            const mid = Math.floor((left + right) / 2);

            if (records[mid][0] <= snap_id) {
                ans = mid;
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        return ans === -1 ? 0 : records[ans][1];
    }
}
```

```python
from bisect import bisect_right
from typing import List

class SnapshotArray:

    def __init__(self, length: int):
        self.snap_id = 0
        self.data = [[] for _ in range(length)]

    def set(self, index: int, val: int) -> None:
        if self.data[index] and self.data[index][-1][0] == self.snap_id:
            self.data[index][-1] = (self.snap_id, val)
        else:
            self.data[index].append((self.snap_id, val))

    def snap(self) -> int:
        cur = self.snap_id
        self.snap_id += 1
        return cur

    def get(self, index: int, snap_id: int) -> int:
        records = self.data[index]
        pos = bisect_right(records, (snap_id, float("inf")))

        if pos == 0:
            return 0

        return records[pos - 1][1]
```

```java
import java.util.*;

class SnapshotArray {
    private int snapId;
    private List<int[]>[] data;

    public SnapshotArray(int length) {
        this.snapId = 0;
        this.data = new ArrayList[length];

        for (int i = 0; i < length; i++) {
            data[i] = new ArrayList<>();
        }
    }

    public void set(int index, int val) {
        List<int[]> records = data[index];

        if (!records.isEmpty() && records.get(records.size() - 1)[0] == snapId) {
            records.get(records.size() - 1)[1] = val;
        } else {
            records.add(new int[]{snapId, val});
        }
    }

    public int snap() {
        int cur = snapId;
        snapId++;
        return cur;
    }

    public int get(int index, int snap_id) {
        List<int[]> records = data[index];

        int left = 0;
        int right = records.size() - 1;
        int ans = -1;

        while (left <= right) {
            int mid = left + (right - left) / 2;

            if (records.get(mid)[0] <= snap_id) {
                ans = mid;
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        if (ans == -1) {
            return 0;
        }

        return records.get(ans)[1];
    }
}
```

- 时间复杂度：
  - SnapshotArray(length) 的时间复杂度为 O(length)。我们需要 O(length) 初始化数组 data
  - set(index, val) 的时间复杂度为 O(1)
  - snap() 的时间复杂度为 O(1)
  - get(index, snap_id) 的时间复杂度为 O(logS)，其中 S 是调用 set(index, val) 的次数。在最坏情况下，所有的 set(index, val) 操作作用在同一个位置，那么对该位置进行二分查找需要的时间为 O(logS)
- 空间复杂度：O(S)，即为数组 data 需要使用的空间。此外，所有非初始化的操作均只需要 O(1) 额外空间
