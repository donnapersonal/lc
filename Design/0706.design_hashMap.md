# 706.设计哈希映射

题目链接：[706.设计哈希映射](https://leetcode.cn/problems/design-hashmap/)

## 题目大意

不使用任何内建的哈希表库设计一个哈希映射（HashMap）

实现 MyHashMap 类：
- MyHashMap() 用空映射初始化对象
- void put(int key, int value) 向 HashMap 插入一个键值对 (key, value) 。如果 key 已经存在于映射中，则更新其对应的值 value 
- int get(int key) 返回特定的 key 所映射的 value ；如果映射中不包含 key 的映射，返回 -1 
- void remove(key) 如果映射中存在 key 的映射，则移除 key 和它所对应的 value 

```js
Example 1:
Input
["MyHashMap", "put", "put", "get", "get", "put", "get", "remove", "get"]
[[], [1, 1], [2, 2], [1], [3], [2, 1], [2], [2], [2]]
Output
[null, null, null, 1, -1, null, 1, null, -1]

Explanation
MyHashMap myHashMap = new MyHashMap();
myHashMap.put(1, 1); // The map is now [[1,1]]
myHashMap.put(2, 2); // The map is now [[1,1], [2,2]]
myHashMap.get(1);    // return 1, The map is now [[1,1], [2,2]]
myHashMap.get(3);    // return -1 (i.e., not found), The map is now [[1,1], [2,2]]
myHashMap.put(2, 1); // The map is now [[1,1], [2,1]] (i.e., update the existing value)
myHashMap.get(2);    // return 1, The map is now [[1,1], [2,1]]
myHashMap.remove(2); // remove the mapping for 2, The map is now [[1,1]]
myHashMap.get(2);    // return -1 (i.e., not found), The map is now [[1,1]]
```

限制：
- 0 <= key, value <= 10^6
- At most 104 calls will be made to put, get, and remove.

## 解题

### 思路 1：哈希桶 + 拉链法 separate chaining

**核心思路**

创建一个固定大小的 bucket 数组：

self.size = 1000
self.buckets = [[] for _ in range(self.size)]

每个 bucket 是一个小 list，用来存放多个 (key, value)

通过 hash 函数决定某个 key 应该放在哪个 bucket：

def hash(self, key):
    return key % self.size

比如：

key = 1001
1001 % 1000 = 1

所以 (1001, value) 会被放到：

self.buckets[1]


**为什么 bucket 里要存 (key, value)？**

因为 HashMap 不是只判断 key 是否存在，而是要通过 key 找到对应的 value

所以每个 bucket 里存的是： (key, value)

例如：

buckets[1] = [(1, 10), (1001, 20)]

这里 1 和 1001 发生了哈希冲突，因为：

1 % 1000 = 1
1001 % 1000 = 1

它们会被放在同一个 bucket 里。

这种处理冲突的方法叫：拉链法 separate chaining

**put(key, value)**
- 通过 hash 找到 key 对应的 bucket
- 遍历这个 bucket
- 如果 key 已经存在，就更新它的 value
- 如果 key 不存在，就添加新的 (key, value)

例如：

put(1, 10)
put(1001, 20)

因为它们都 hash 到 bucket 1：buckets[1] = [(1, 10), (1001, 20)]

如果再执行：put(1, 99)

就更新 key 1 的 value：buckets[1] = [(1, 99), (1001, 20)]

**get(key)**
- 通过 hash 找到对应 bucket
- 遍历 bucket
- 如果找到 key，返回它的 value
- 如果没找到，返回 -1
  
**remove(key)**
- 通过 hash 找到对应 bucket
- 保留所有 k != key 的元素
- 等价于把目标 key 从 bucket 中删除

例如：buckets[1] = [(1, 99), (1001, 20)]

执行：remove(1)

之后：buckets[1] = [(1001, 20)]

```js
var MyHashMap = function() {
    this.size = 1000;
    this.buckets = Array.from({ length: this.size }, () => []);
};

/**
 * @param {number} key
 * @return {number}
 */
MyHashMap.prototype.hash = function(key) {
    return key % this.size;
};

/**
 * @param {number} key
 * @param {number} value
 * @return {void}
 */
MyHashMap.prototype.put = function(key, value) {
    const index = this.hash(key);
    const bucket = this.buckets[index];

    for (let i = 0; i < bucket.length; i++) {
        const [k, v] = bucket[i];

        if (k === key) {
            bucket[i] = [key, value];
            return;
        }
    }

    bucket.push([key, value]);
};

/**
 * @param {number} key
 * @return {number}
 */
MyHashMap.prototype.get = function(key) {
    const index = this.hash(key);
    const bucket = this.buckets[index];

    for (const [k, v] of bucket) {
        if (k === key) {
            return v;
        }
    }

    return -1;
};

/**
 * @param {number} key
 * @return {void}
 */
MyHashMap.prototype.remove = function(key) {
    const index = this.hash(key);
    const bucket = this.buckets[index];

    for (let i = 0; i < bucket.length; i++) {
        const [k, v] = bucket[i];

        if (k === key) {
            bucket.splice(i, 1);
            return;
        }
    }
```

```python
class MyHashMap:

    def __init__(self):
        self.size = 1000
        self.buckets = [[] for _ in range(self.size)]

    def hash(self, key):
        return key % self.size

    def put(self, key: int, value: int) -> None:
        index = self.hash(key)
        for i, (k, v) in enumerate(self.buckets[index]):
            if k == key:
                self.buckets[index][i] = (key, value)
                return

        self.buckets[index].append((key, value))

    def get(self, key: int) -> int:
        index = self.hash(key)
        for k, v in self.buckets[index]:
            if k == key:
                return v

        return -1

    def remove(self, key: int) -> None:
        index = self.hash(key)
        self.buckets[index] = [(k, v) for (k, v) in self.buckets[index] if k != key]
```
```java
import java.util.*;

class MyHashMap {
    private static final int SIZE = 1000;
    private List<int[]>[] buckets;

    public MyHashMap() {
        buckets = new ArrayList[SIZE];

        for (int i = 0; i < SIZE; i++) {
            buckets[i] = new ArrayList<>();
        }
    }

    private int hash(int key) {
        return key % SIZE;
    }

    public void put(int key, int value) {
        int index = hash(key);

        for (int i = 0; i < buckets[index].size(); i++) {
            int[] pair = buckets[index].get(i);

            if (pair[0] == key) {
                pair[1] = value;
                return;
            }
        }

        buckets[index].add(new int[]{key, value});
    }

    public int get(int key) {
        int index = hash(key);

        for (int[] pair : buckets[index]) {
            if (pair[0] == key) {
                return pair[1];
            }
        }

        return -1;
    }

    public void remove(int key) {
        int index = hash(key);

        for (int i = 0; i < buckets[index].size(); i++) {
            if (buckets[index].get(i)[0] == key) {
                buckets[index].remove(i);
                return;
            }
        }
    }
}
```

设：
- n = HashMap 中存储的 key-value 数量
- m = bucket 数量，这里是 1000
- k = 某个 bucket 中的元素数量

平均时间复杂度：O(1)

如果 hash 分布比较均匀，每个 bucket 中的元素数量很少。

所以：

| 操作       | 平均时间复杂度 |
| -------- | ------: |
| `put`    |  `O(1)` |
| `get`    |  `O(1)` |
| `remove` |  `O(1)` |

如果很多 key 都 hash 到同一个 bucket，比如：0, 1000, 2000, 3000, ...

它们都会进入：bucket[0]

这时一个 bucket 可能包含所有元素，遍历 bucket 就需要 O(n)。

所以：

| 操作       | 最坏时间复杂度 |
| -------- | ------: |
| `put`    |  `O(n)` |
| `get`    |  `O(n)` |
| `remove` |  `O(n)` |

空间包括：
- bucket 数组：O(m)
- 所有 (key, value)：O(n)

所以总空间是：`O(n + m)`

因为这里 m = 1000 是固定大小，也可以简化成：`O(n)`

**手写链表node**

```python
class MyHashMap:

    class Node:
        def __init__(self, key: int, value: int):
            self.key = key
            self.value = value
            self.next = None

    def __init__(self):
        self.size = 10009
        self.buckets = [None] * self.size

    def _hash(self, key: int) -> int:
        return key % self.size

    def put(self, key: int, value: int) -> None:
        idx = self._hash(key)

        # 如果当前 bucket 为空，直接放入新节点
        if self.buckets[idx] is None:
            self.buckets[idx] = self.Node(key, value)
            return

        cur = self.buckets[idx]

        # 遍历链表，检查 key 是否已经存在
        while cur:
            if cur.key == key:
                cur.value = value
                return

            # 如果走到链表尾部，准备尾插
            if cur.next is None:
                break

            cur = cur.next

        # key 不存在，尾插新节点
        cur.next = self.Node(key, value)

    def get(self, key: int) -> int:
        idx = self._hash(key)
        cur = self.buckets[idx]

        while cur:
            if cur.key == key:
                return cur.value
            cur = cur.next

        return -1

    def remove(self, key: int) -> None:
        idx = self._hash(key)
        cur = self.buckets[idx]
        prev = None

        while cur:
            if cur.key == key:
                # 删除头节点
                if prev is None:
                    self.buckets[idx] = cur.next
                # 删除中间或尾部节点
                else:
                    prev.next = cur.next
                return

            prev = cur
            cur = cur.next
```

- 时间复杂度：由于没有扩容的逻辑，最坏情况下复杂度为 O(n)，一般情况下复杂度为 O(1)
- 空间复杂度：O(1)

### 思路 2: 简单数组解法 - direct addressing

```python
class MyHashMap:

    def __init__(self):
        self.N = 1000009
        self.map = [-1] * self.N

    def put(self, key: int, value: int) -> None:
        self.map[key] = value

    def get(self, key: int) -> int:
        return self.map[key]

    def remove(self, key: int) -> None:
        self.map[key] = -1
```

- 时间复杂度：`O(1)`
- 空间复杂度：`O(1)`


### 思路 3：开放寻址法 - 哈希数组 + 线性探测 linear probing + 懒删除 lazy deletion

所有 (key, value) 都直接存在同一个数组里，不额外使用链表

**如果发生冲突怎么办？**

比如：key1 % size == key2 % size

说明两个 key 想放到同一个位置

这时不用链表，而是继续向后找下一个空位

例如：

size = 10

put(1, 100)
1 % 10 = 1
nodes[1] = (1, 100)

put(11, 200)
11 % 10 = 1
nodes[1] 已经被 1 占用
继续找 nodes[2]
nodes[2] = (11, 200)

这种方法叫：`linear probing`

```python
class MyHashMap:

    class Node:
        def __init__(self, key: int, value: int):
            self.key = key
            self.value = value
            self.is_deleted = False

    def __init__(self):
        self.size = 10009
        self.offset = 1
        self.nodes = [None] * self.size

    def _hash(self, key: int) -> int:
        # Python 中 key % size 已经可以得到非负下标
        return key % self.size

    def _get_index(self, key: int) -> int:
        idx = self._hash(key)

        # 如果当前位置不为空，并且不是当前 key，就继续往后找
        while self.nodes[idx] is not None and self.nodes[idx].key != key:
            idx = (idx + self.offset) % self.size

        return idx

    def put(self, key: int, value: int) -> None:
        idx = self._get_index(key)
        node = self.nodes[idx]

        if node is not None:
            node.value = value
            node.is_deleted = False
        else:
            self.nodes[idx] = self.Node(key, value)

    def get(self, key: int) -> int:
        idx = self._get_index(key)
        node = self.nodes[idx]

        if node is None or node.is_deleted:
            return -1

        return node.value

    def remove(self, key: int) -> None:
        idx = self._get_index(key)
        node = self.nodes[idx]

        if node is not None:
            node.is_deleted = True
```

- 时间复杂度：一般情况下复杂度为 O(1)，极端情况下为 O(n)
- 空间复杂度：O(1)
