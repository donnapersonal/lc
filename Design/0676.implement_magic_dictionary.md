# 676.实现一个魔法字典

题目链接：[676.实现一个魔法字典](https://leetcode.cn/problems/implement-magic-dictionary/)

## 题目大意

设计一个使用单词列表进行初始化的数据结构，单词列表中的单词 互不相同。如果给出一个单词，请判定能否只将这个单词中一个字母换成另一个字母，使得所形成的新单词存在于你构建的字典中

实现 MagicDictionary 类：

- MagicDictionary() 初始化对象
- void buildDict(String[] dictionary) 使用字符串数组 dictionary 设定该数据结构，dictionary 中的字符串互不相同
- bool search(String searchWord) 给定一个字符串 searchWord ，判定能否只将字符串中 一个 字母换成另一个字母，使得所形成的新字符串能够与字典中的任一字符串匹配。如果可以，返回 true；否则，返回 false

```js
Example 1:
Input
["MagicDictionary", "buildDict", "search", "search", "search", "search"]
[[], [["hello", "leetcode"]], ["hello"], ["hhllo"], ["hell"], ["leetcoded"]]
Output
[null, null, false, true, false, false]

Explanation:
MagicDictionary magicDictionary = new MagicDictionary();
magicDictionary.buildDict(["hello", "leetcode"]);
magicDictionary.search("hello"); // return False
magicDictionary.search("hhllo"); // We can change the second 'h' to 'e' to match "hello" so we return True
magicDictionary.search("hell"); // return False
magicDictionary.search("leetcoded"); // return False
```

限制：
- 1 <= dictionary.length <= 100
- 1 <= dictionary[i].length <= 100
- dictionary[i] consists of only lower-case English letters.
- All the strings in dictionary are distinct.
- 1 <= searchWord.length <= 100
- searchWord consists of only lower-case English letters.
- buildDict will be called only once before search.
- At most 100 calls will be made to search.

## 解题

令 `dictionary` 为 `dict`，令 `searchWord` 为 `s`，整体题意：给定字符串 `s`，问能否存在替换掉 `s` 中的某个字符，使得新字符串出现在 `dict` 数组中

使用`字典树/Trie`求解该问题：

- `buildDict` 操作：可以将所有的 `dict[i]` 存入字典树中，方便后续检索
- `search` 操作：
  - 设计递归函数 `query(s, idx, p, limit)`，其中 `s` 为待检索的字符串，`idx` 为当前处理到字符串 `s` 的哪一位，`p` 为当前搜索到字典树的索引编号（起始有 `p=0`），`limit` 为当前剩余的替换字符次数，根据题意，`limit` 固定为 `1`，含义为必须替换掉 `s` 的一个字符
  - 对于 `s[idx]` 而言，可以枚举新字符串在当前位置是何种字符（`C=26` 个选择），若当前枚举到的字符与 `s[idx]` 一致，则不消耗替换次数
  - 爆搜过程中替换次数为负数直接剪枝，当爆搜到结尾位置，再检查当前的字典树索引 `p` 是否为单词结尾节点（对应查询数组 `dict` 中是否存在该字符串），以及剩余的替换次数 `limit` 是否为 `0`

```js
class TrieNode {
  constructor() {
    this.children = new Map();
    this.isEnd = false;
  }
}

var MagicDictionary = function() {
    this.root = new TrieNode();
};

/** 
 * @param {string[]} dictionary
 * @return {void}
 */
MagicDictionary.prototype.buildDict = function(dictionary) {
    for (const word of dictionary) {
      this.add(word);
    }
};

MagicDictionary.prototype.add = function(word) {
    let node = this.root;

    for (const ch of word) {
      if (!node.children.has(ch)) {
        node.children.set(ch, new TrieNode());
      }
      node = node.children.get(ch);
    }

    node.isEnd = true;
  }

/** 
 * @param {string} searchWord
 * @return {boolean}
 */
MagicDictionary.prototype.search = function(searchWord) {
    return this.query(searchWord, 0, this.root, 1);
};

MagicDictionary.prototype.query = function(word, idx, node, limit) {
    if (limit < 0) return false;

    if (idx === word.length) {
      return node.isEnd && limit === 0;
    }

    const cur = word[idx];

    for (const [ch, child] of node.children.entries()) {
      const nextLimit = ch === cur ? limit : limit - 1;

      if (this.query(word, idx + 1, child, nextLimit)) {
        return true;
      }
    }

    return false;
}
```

```python
class MagicDictionary:

    def __init__(self):
        self.children = {}
        self.is_end = False

    def buildDict(self, dictionary: List[str]) -> None:
        for word in dictionary:
            self.add(word)
    
    def add(self, word):
        node = self
        for ch in word:
            if ch not in node.children:
                node.children[ch] = MagicDictionary()
            node = node.children[ch]
        node.is_end = True

    def search(self, searchWord: str) -> bool:
        return self.query(searchWord, 0, 1)
    
    def query(self, word, idx, limit):
        if limit < 0:
            return False

        if idx == len(word):
            return self.is_end and limit == 0

        cur = word[idx]

        for ch, child in self.children.items():
            if ch == cur:
                if child.query(word, idx + 1, limit):
                    return True
            else:
                if child.query(word, idx + 1, limit - 1):
                    return True

        return False
```

```java
class MagicDictionary {
    int N = 100 * 100, M = 26, idx = 0;
    int[][] tr = new int[N][M];
    boolean[] isEnd = new boolean[N * M];
    void add(String s) {
        int p = 0;
        for (int i = 0; i < s.length(); i++) {
            int u = s.charAt(i) - 'a';
            if (tr[p][u] == 0) tr[p][u] = ++idx;
            p = tr[p][u];
        }
        isEnd[p] = true;
    }
    boolean query(String s, int idx, int p, int limit) {
        if (limit < 0) return false;
        if (idx == s.length()) return isEnd[p] && limit == 0;
        int u = s.charAt(idx) - 'a';
        for (int i = 0; i < 26; i++) {
            if (tr[p][i] == 0) continue;
            if (query(s, idx + 1, tr[p][i], i == u ? limit : limit - 1)) return true;
        }
        return false;
    }
    public void buildDict(String[] ss) {
        for (String s : ss) add(s);
    }
    public boolean search(String s) {
        return query(s, 0, 0, 1);
    }
}

class MagicDictionary {
    static class TrieNode {
        TrieNode[] children = new TrieNode[26];
        boolean isEnd = false;
    }

    private final TrieNode root = new TrieNode();

    public MagicDictionary() {
    }

    public void buildDict(String[] dictionary) {
        for (String word : dictionary) {
            add(word);
        }
    }

    private void add(String word) {
        TrieNode node = root;
        for (char ch : word.toCharArray()) {
            int idx = ch - 'a';
            if (node.children[idx] == null) {
                node.children[idx] = new TrieNode();
            }
            node = node.children[idx];
        }
        node.isEnd = true;
    }

    public boolean search(String searchWord) {
        return dfs(root, searchWord, 0, 1);
    }

    private boolean dfs(TrieNode node, String word, int idx, int limit) {
        if (node == null) return false;
        if (limit < 0) return false;

        if (idx == word.length()) {
            return node.isEnd && limit == 0;
        }

        int target = word.charAt(idx) - 'a';

        for (int c = 0; c < 26; c++) {
            if (node.children[c] == null) continue;

            int nextLimit = (c == target) ? limit : limit - 1;

            if (dfs(node.children[c], word, idx + 1, nextLimit)) {
                return true;
            }
        }

        return false;
    }
}

Why this Java version is better for interviews than fixed arrays of ints？

Your earlier Java version used:

int N = 100 * 100, M = 26;
int[][] tr = new int[N][M];
boolean[] isEnd = new boolean[N * M];

That works if constraints are small and fixed, but in an interview I would usually prefer the TrieNode version because:

no hard-coded size assumptions
easier to explain
cleaner and safer
more idiomatic for general-purpose design questions

A fixed-array trie is fine when:

constraints are known
you want raw speed
memory layout matters

But for interview explanation, pointer-style trie is usually cleaner.
```
