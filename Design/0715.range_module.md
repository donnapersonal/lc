# 715.Range 模块

题目链接：[715.Range 模块](https://leetcode.cn/problems/range-module/)

## 题目大意

Range模块是跟踪数字范围的模块。设计一个数据结构来跟踪表示为 半开区间 的范围并查询它们

半开区间 [left, right) 表示所有 left <= x < right 的实数 x 

实现 RangeModule 类:
- RangeModule() 初始化数据结构的对象
- void addRange(int left, int right) 添加 半开区间 [left, right)，跟踪该区间中的每个实数。添加与当前跟踪的数字部分重叠的区间时，应当添加在区间 [left, right) 中尚未跟踪的任何数字到该区间中
- boolean queryRange(int left, int right) 只有在当前正在跟踪区间 [left, right) 中的每一个实数时，才返回 true ，否则返回 false 
- void removeRange(int left, int right) 停止跟踪 半开区间 [left, right) 中当前正在跟踪的每个实数

```js
Example 1:
Input
["RangeModule", "addRange", "removeRange", "queryRange", "queryRange", "queryRange"]
[[], [10, 20], [14, 16], [10, 14], [13, 15], [16, 17]]
Output
[null, null, null, true, false, true]

Explanation
RangeModule rangeModule = new RangeModule();
rangeModule.addRange(10, 20);
rangeModule.removeRange(14, 16);
rangeModule.queryRange(10, 14); // return True,(Every number in [10, 14) is being tracked)
rangeModule.queryRange(13, 15); // return False,(Numbers like 14, 14.03, 14.17 in [13, 15) are not being tracked)
rangeModule.queryRange(16, 17); // return True, (The number 16 in [16, 17) is still being tracked, despite the remove operation)

```

限制：
- 1 <= left < right <= 10^9
- At most 104 calls will be made to addRange, queryRange, and removeRange.

## 解题

这题要维护一堆被覆盖的区间，支持三个操作：
- `addRange(left, right)`  覆盖 `[left, right)`
- `removeRange(left, right)`   取消覆盖 `[left, right)`
- `queryRange(left, right)`   查询 `[left, right)` 是否全部被覆盖

注意区间是 左闭右开：`[left, right)`

所以在线段树内部可以转成闭区间：`[left, right - 1]`

**为什么用动态线段树？**

因为坐标范围很大，比如：1 <= left < right <= 10^9

如果直接建数组，需要 `10^9` 空间，不现实

所以用 `动态线段树：只有访问到某个区间时，才创建对应节点`

**每个节点表示什么？**

每个节点代表一个区间 `[start, end]`

节点里维护两个信息：`cover/lazy`

`cover`
- 表示当前节点对应的整个区间是否都被覆盖
- `cover = True`   表示整个区间都被覆盖
- `cover = False`  表示整个区间没有被完全覆盖
  
`lazy`

表示懒标记：
- `lazy = 0`   没有待下传操作
- `lazy = 1`   整个区间被覆盖
- `lazy = -1`  整个区间被取消覆盖
  
**addRange 怎么做？**

addRange(left, right) 就是把 [left, right - 1] 标记为覆盖

在线段树里做区间更新：

- 如果当前节点区间完全在更新范围内：
  - cover = True
  - lazy = 1
  
**removeRange 怎么做？**

removeRange(left, right) 就是把 [left, right - 1] 标记为不覆盖

在线段树里做区间更新：
- 如果当前节点区间完全在删除范围内：
  - cover = False
  - lazy = -1
  
**queryRange 怎么做？**

queryRange(left, right) 就是查询 [left, right - 1] 是否完全被覆盖

如果查询区间完全包含当前节点：`return node.cover`，否则递归查左右子树

只有左右需要查询的部分都被覆盖，才返回 True

**为什么需要 lazy？**

假设要覆盖一个很大的区间：[1, 1e9]

如果递归到底，会非常慢

所以当一个节点区间被整体覆盖时，我们不继续往下递归，而是直接打标记：
- node.cover = True
- node.lazy = 1

等以后真的需要访问它的子区间时，再把这个标记下传给左右孩子

这就是 `lazy propagation`

```js
var RangeModule = function() {
    this.root = new Node();
    this.MIN = 1;
    this.MAX = 1e9;
};

function Node() {
    this.left = null;
    this.right = null;
    this.cover = false;
    this.lazy = 0; // 0 = no tag, 1 = cover, -1 = uncover
}

RangeModule.prototype.addRange = function(left, right) {
    this.update(this.root, this.MIN, this.MAX, left, right - 1, 1);
};

RangeModule.prototype.removeRange = function(left, right) {
    this.update(this.root, this.MIN, this.MAX, left, right - 1, -1);
};

RangeModule.prototype.queryRange = function(left, right) {
    return this.query(this.root, this.MIN, this.MAX, left, right - 1);
};

RangeModule.prototype.update = function(node, start, end, l, r, val) {
    if (r < start || end < l) {
        return;
    }

    if (l <= start && end <= r) {
        node.cover = val === 1;
        node.lazy = val;
        return;
    }

    this.pushDown(node);

    const mid = Math.floor(start + (end - start) / 2);

    if (l <= mid) {
        this.update(node.left, start, mid, l, r, val);
    }

    if (r > mid) {
        this.update(node.right, mid + 1, end, l, r, val);
    }

    node.cover = node.left.cover && node.right.cover;
};

RangeModule.prototype.query = function(node, start, end, l, r) {
    if (r < start || end < l) {
        return true;
    }

    if (l <= start && end <= r) {
        return node.cover;
    }

    this.pushDown(node);

    const mid = Math.floor(start + (end - start) / 2);
    let res = true;

    if (l <= mid) {
        res = res && this.query(node.left, start, mid, l, r);
    }

    if (r > mid) {
        res = res && this.query(node.right, mid + 1, end, l, r);
    }

    return res;
};

RangeModule.prototype.pushDown = function(node) {
    if (node.left === null) {
        node.left = new Node();
    }

    if (node.right === null) {
        node.right = new Node();
    }

    if (node.lazy === 0) {
        return;
    }

    node.left.cover = node.lazy === 1;
    node.right.cover = node.lazy === 1;

    node.left.lazy = node.lazy;
    node.right.lazy = node.lazy;

    node.lazy = 0;
};

 * Your RangeModule object will be instantiated and called as such:
 * var obj = new RangeModule()
 * obj.addRange(left,right)
 * var param_2 = obj.queryRange(left,right)
 * obj.removeRange(left,right)
 */
```
```python
class RangeModule:

    class Node:
        def __init__(self):
            self.left = None
            self.right = None
            self.cover = False   # whether this whole interval is fully covered
            self.lazy = 0        # 0: no lazy tag, 1: cover, -1: uncover

    def __init__(self):
        self.root = self.Node()
        self.MIN = 1
        self.MAX = 10 ** 9

    def addRange(self, left: int, right: int) -> None:
        self._update(self.root, self.MIN, self.MAX, left, right - 1, 1)

    def removeRange(self, left: int, right: int) -> None:
        self._update(self.root, self.MIN, self.MAX, left, right - 1, -1)

    def queryRange(self, left: int, right: int) -> bool:
        return self._query(self.root, self.MIN, self.MAX, left, right - 1)

    def _update(self, node: Node, start: int, end: int, l: int, r: int, val: int) -> None:
        # No overlap
        if r < start or end < l:
            return

        # Fully covered by update range
        if l <= start and end <= r:
            node.cover = (val == 1)
            node.lazy = val
            return

        self._push_down(node)

        mid = start + (end - start) // 2

        if l <= mid:
            self._update(node.left, start, mid, l, r, val)

        if r > mid:
            self._update(node.right, mid + 1, end, l, r, val)

        node.cover = node.left.cover and node.right.cover

    def _query(self, node: Node, start: int, end: int, l: int, r: int) -> bool:
        # No overlap
        if r < start or end < l:
            return True

        # Fully inside query range
        if l <= start and end <= r:
            return node.cover

        self._push_down(node)

        mid = start + (end - start) // 2
        res = True

        if l <= mid:
            res = res and self._query(node.left, start, mid, l, r)

        if r > mid:
            res = res and self._query(node.right, mid + 1, end, l, r)

        return res

    def _push_down(self, node: Node) -> None:
        if node.left is None:
            node.left = self.Node()

        if node.right is None:
            node.right = self.Node()

        if node.lazy == 0:
            return

        node.left.cover = (node.lazy == 1)
        node.right.cover = (node.lazy == 1)

        node.left.lazy = node.lazy
        node.right.lazy = node.lazy

        node.lazy = 0
```
```java
class RangeModule {
    class Node {
        Node left;
        Node right;
        boolean cover;
        int lazy; // 0 = no tag, 1 = cover, -1 = uncover
    }

    private final int MIN = 1;
    private final int MAX = 1_000_000_000;
    private final Node root;

    public RangeModule() {
        root = new Node();
    }

    public void addRange(int left, int right) {
        update(root, MIN, MAX, left, right - 1, 1);
    }

    public void removeRange(int left, int right) {
        update(root, MIN, MAX, left, right - 1, -1);
    }

    public boolean queryRange(int left, int right) {
        return query(root, MIN, MAX, left, right - 1);
    }

    private void update(Node node, int start, int end, int l, int r, int val) {
        if (r < start || end < l) {
            return;
        }

        if (l <= start && end <= r) {
            node.cover = val == 1;
            node.lazy = val;
            return;
        }

        pushDown(node);

        int mid = start + (end - start) / 2;

        if (l <= mid) {
            update(node.left, start, mid, l, r, val);
        }

        if (r > mid) {
            update(node.right, mid + 1, end, l, r, val);
        }

        node.cover = node.left.cover && node.right.cover;
    }

    private boolean query(Node node, int start, int end, int l, int r) {
        if (r < start || end < l) {
            return true;
        }

        if (l <= start && end <= r) {
            return node.cover;
        }

        pushDown(node);

        int mid = start + (end - start) / 2;
        boolean res = true;

        if (l <= mid) {
            res = res && query(node.left, start, mid, l, r);
        }

        if (r > mid) {
            res = res && query(node.right, mid + 1, end, l, r);
        }

        return res;
    }

    private void pushDown(Node node) {
        if (node.left == null) {
            node.left = new Node();
        }

        if (node.right == null) {
            node.right = new Node();
        }

        if (node.lazy == 0) {
            return;
        }

        node.left.cover = node.lazy == 1;
        node.right.cover = node.lazy == 1;

        node.left.lazy = node.lazy;
        node.right.lazy = node.lazy;

        node.lazy = 0;
    }
}
```

- 时间复杂度
  - 设坐标范围是：U = 10^9
  - 每次操作只走线段树高度：`O(log U)`

  所以：

  | Operation     |       Time |
  | ------------- | ---------: |
  | `addRange`    | `O(log U)` |
  | `removeRange` | `O(log U)` |
  | `queryRange`  | `O(log U)` |

- 空间复杂度：`O(number of created nodes)`

因为是动态开点，不会真的创建 `10^9` 个节点
