# 271.Encode and Decode Strings

难度: Medium
标签: Design、Array、String

![271](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.7zr4q8y5z3.webp)

限制：
- 1 <= strs.length <= 200
- 0 <= strs[i].length <= 200
- strs[i] contains any possible characters out of 256 valid ASCII characters.

## 解题

```python
class Codec:
    def encode(self, strs: List[str]) -> str:
        """Encodes a list of strings to a single string.
        """
        # Store the encoded components in a list so that
        # we can join them efficiently at the end.
        encoded_parts = []

        for value in strs:
            # Encode each string as:
            # length + "#" + original string
            encoded_parts.append(str(len(value)))
            encoded_parts.append("#")
            encoded_parts.append(value)

        return "".join(encoded_parts)
        
    def decode(self, s: str) -> List[str]:
        """Decodes a single string to a list of strings.
        """
        decoded_strings = []

        # i points to the beginning of the next length header.
        i = 0

        while i < len(s):
            # Find the '#' that terminates the length header.
            j = i

            while s[j] != "#":
                j += 1

            # Parse the number before '#'.
            length = int(s[i:j])

            # The actual string begins immediately after '#'.
            start = j + 1
            end = start + length

            # Read exactly 'length' characters.
            decoded_strings.append(s[start:end])

            # Continue with the next encoded string.
            i = end

        return decoded_strings
```

```java
public class Codec {

    // Encodes a list of strings to a single string.
    public String encode(List<String> strs) {
        StringBuilder encoded = new StringBuilder();
        for (String value : strs) {
            // Format: <length>#<string>
            encoded
                .append(value.length())
                .append('#')
                .append(value);
        }

        return encoded.toString();
    }

    // Decodes a single string to a list of strings.
    public List<String> decode(String s) {
        List<String> result = new ArrayList<>();
        int i = 0;

        while (i < s.length()) {
            // Find the end of the length header.
            int j = i;

            while (s.charAt(j) != '#') {
                j++;
            }

            int length = Integer.parseInt(s.substring(i, j));

            // Read exactly 'length' characters after '#'.
            int start = j + 1;
            int end = start + length;

            result.add(s.substring(start, end));
            i = end;
        }

        return result;
    }
}
```

- 时间复杂度: `O(m+n)` for each `encode()` and `decode()` function calls.
- 空间复杂度: `O(m+n)` for each `encode()` and `decode()` function calls.

其中 `m` 是所有字符串长度之和，`n` 是字符串的数量
