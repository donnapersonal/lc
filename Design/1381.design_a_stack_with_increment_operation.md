# 1381.设计一个带有递增操作的栈

题目链接：[1381.设计一个带有递增操作的栈](https://leetcode.com/problems/design-a-stack-with-increment-operation/)

## 题目大意

设计一个支持对其元素进行递增操作的栈

实现该类CustomStack：
- CustomStack(int maxSize)初始化对象，使其包含maxSize栈中元素的最大数量
- void push(int x)如果栈尚未到达栈顶，则将元素添加x到栈顶maxSize
- int pop()弹出并返回栈顶元素，-1如果栈为空则返回 false
- void inc(int k, int val)k将栈底元素的值加1。val如果栈中的元素少于 1 k，则将栈中的所有元素的值加 1

```js
Example 1:
Input
["CustomStack","push","push","pop","push","push","push","increment","increment","pop","pop","pop","pop"]
[[3],[1],[2],[],[2],[3],[4],[5,100],[2,100],[],[],[],[]]
Output
[null,null,null,2,null,null,null,null,null,103,202,201,-1]
Explanation
CustomStack stk = new CustomStack(3); // Stack is Empty []
stk.push(1);                          // stack becomes [1]
stk.push(2);                          // stack becomes [1, 2]
stk.pop();                            // return 2 --> Return top of the stack 2, stack becomes [1]
stk.push(2);                          // stack becomes [1, 2]
stk.push(3);                          // stack becomes [1, 2, 3]
stk.push(4);                          // stack still [1, 2, 3], Do not add another elements as size is 4
stk.increment(5, 100);                // stack becomes [101, 102, 103]
stk.increment(2, 100);                // stack becomes [201, 202, 103]
stk.pop();                            // return 103 --> Return top of the stack 103, stack becomes [201, 202]
stk.pop();                            // return 202 --> Return top of the stack 202, stack becomes [201]
stk.pop();                            // return 201 --> Return top of the stack 201, stack becomes []
stk.pop();                            // return -1 --> Stack is empty return -1.
```

限制:
- 1 <= maxSize, x, k <= 1000
- 0 <= val <= 100
- At most 1000 calls will be made to each method of increment, push and pop each separately.

## 解题

题目难点在哪里？
- 如果我们直接写：
  ```python
  for i in range(min(k, size)):
    stack[i] += val
  ```
- 那么 increment(k, val) 的时间复杂度是：O(k)
- 如果 increment 被频繁调用，效率会比较差

所以最优解的目标是：
- push: O(1)
- pop: O(1)
- increment: O(1)

核心方法是：`Lazy Propagation / 延迟增量`

核心思路：`不要马上加，先记账`

- 额外维护一个数组：**inc，其中：inc[i] 表示：从栈底到下标 i 的这些元素，都应该额外加上 inc[i]**
- 也就是说，increment(k, val) 不真的去修改前 k 个元素，而是只在某个位置做一个标记
- 比如当前栈是：stack = [1, 2, 3, 4]
  - 调用：increment(3, 100) 意思是让底部 3 个元素变成：[101, 102, 103, 4]
  - 但我们不直接改 stack，我们只做：inc[2] += 100
  - 因为下标 2 对应的是底部 3 个元素的最高位置
  - 此时：stack = [1, 2, 3, 4], inc   = [0, 0, 100, 0]
  - 真正弹出元素时，才把增量算进去

为什么 pop 时要把增量往下传？

假设：

stack = [1, 2, 3]
inc   = [0, 0, 100]

这表示：底部 3 个元素都应该 +100

如果现在 pop()，弹出的是 3

结果应该是：3 + 100 = 103

弹出 3 后，剩下：stack = [1, 2]

但是 1 和 2 仍然应该加 100

所以需要把：inc[2] 传给下面一层：inc[1] += inc[2]，然后清空 inc[2]

这就是代码里的关键：

```python
if self._top > 0:
    self._inc[self._top - 1] += self._inc[self._top]
```

```js
/**
 * @param {number} maxSize
 */
var CustomStack = function(maxSize) {
    this.stack = new Array(maxSize).fill(0);
    this.inc = new Array(maxSize).fill(0);
    this.top = -1;
};

/** 
 * @param {number} x
 * @return {void}
 */
CustomStack.prototype.push = function(x) {
    if (this.top < this.stack.length - 1) {
        this.top++;
        this.stack[this.top] = x;
    }
};

/**
 * @return {number}
 */
CustomStack.prototype.pop = function() {
    if (this.top < 0) {
        return -1;
    }

    const result = this.stack[this.top] + this.inc[this.top];

    if (this.top > 0) {
        this.inc[this.top - 1] += this.inc[this.top];
    }

    this.inc[this.top] = 0;
    this.top--;

    return result;
};

/** 
 * @param {number} k 
 * @param {number} val
 * @return {void}
 */
CustomStack.prototype.increment = function(k, val) {
    if (this.top >= 0) {
        const index = Math.min(this.top, k - 1);
        this.inc[index] += val;
    }
};
```
```python
class CustomStack:
    def __init__(self, max_size: int):
        self._stack = [0] * max_size
        self._inc = [0] * max_size
        self._top = -1

    def push(self, x: int) -> None:
        if self._top < len(self._stack) - 1:
            self._top += 1
            self._stack[self._top] = x

    def pop(self) -> int:
        if self._top < 0:
            return -1

        result = self._stack[self._top] + self._inc[self._top]

        if self._top > 0:
            self._inc[self._top - 1] += self._inc[self._top]

        self._inc[self._top] = 0
        self._top -= 1

        return result

    def increment(self, k: int, val: int) -> None:
        if self._top >= 0:
            index = min(self._top, k - 1)
            self._inc[index] += val
```
```java
class CustomStack {
    private int[] stack;
    private int[] inc;
    private int top;

    public CustomStack(int maxSize) {
        stack = new int[maxSize];
        inc = new int[maxSize];
        top = -1;
    }

    public void push(int x) {
        if (top < stack.length - 1) {
            top++;
            stack[top] = x;
        }
    }

    public int pop() {
        if (top < 0) {
            return -1;
        }

        int result = stack[top] + inc[top];

        if (top > 0) {
            inc[top - 1] += inc[top];
        }

        inc[top] = 0;
        top--;

        return result;
    }

    public void increment(int k, int val) {
        if (top >= 0) {
            int index = Math.min(top, k - 1);
            inc[index] += val;
        }
    }
}
```

- push(x)，只移动一次 top，并写入一个值
  - Time: O(1)
  - Space: O(1)

- pop()，只访问栈顶元素，并最多把增量传给下一层
  - Time: O(1)
  - Space: O(1)

- increment(k, val)，没有循环修改底部 k 个元素，只在 inc 数组某个位置打一个标记
  - Time: O(1)
  - Space: O(1)
