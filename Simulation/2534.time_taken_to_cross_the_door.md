# 2534.通过门的时间

题目链接：[2534.通过门的时间](https://leetcode.cn/problems/time-taken-to-cross-the-door/)

## 题目大意

n 个人，按从 0 到 n - 1 编号。现在有一扇门，每个人只能通过门进入或离开一次，耗时一秒

给你一个 非递减顺序 排列的整数数组 arrival ，数组长度为 n ，其中 arrival[i] 是第 i 个人到达门前的时间。另给你一个长度为 n 的数组 state ，其中 state[i] 是 0 则表示第 i 个人希望进入这扇门，是 1 则表示 TA 想要离开这扇门

如果 同时 有两个或更多人想要使用这扇门，则必须遵循以下规则：
- 如果前一秒 没有 使用门，那么想要 离开 的人会先离开
- 如果前一秒使用门 进入 ，那么想要 进入 的人会先进入
- 如果前一秒使用门 离开 ，那么想要 离开 的人会先离开
- 如果多个人都想朝同一方向走（都进入或都离开），编号最小的人会先通过门
  
返回一个长度为 n 的数组 answer ，其中 answer[i] 是第 i 个人通过门的时刻（秒）

注意：
- 每秒只有一个人可以通过门
- 为遵循上述规则，一个人可以在到达门附近后等待，而不通过门进入或离开

```js
Example 1:
Input: arrival = [0,1,1,2,4], state = [0,1,0,0,1]
Output: [0,3,1,2,4]
Explanation: At each second we have the following:
- At t = 0: Person 0 is the only one who wants to enter, so they just enter through the door.
- At t = 1: Person 1 wants to exit, and person 2 wants to enter. Since the door was used the previous second for entering, person 2 enters.
- At t = 2: Person 1 still wants to exit, and person 3 wants to enter. Since the door was used the previous second for entering, person 3 enters.
- At t = 3: Person 1 is the only one who wants to exit, so they just exit through the door.
- At t = 4: Person 4 is the only one who wants to exit, so they just exit through the door.

Example 2:
Input: arrival = [0,0,0], state = [1,0,1]
Output: [0,2,1]
Explanation: At each second we have the following:
- At t = 0: Person 1 wants to enter while persons 0 and 2 want to exit. Since the door was not used in the previous second, the persons who want to exit get to go first. Since person 0 has a smaller index, they exit first.
- At t = 1: Person 1 wants to enter, and person 2 wants to exit. Since the door was used in the previous second for exiting, person 2 exits.
- At t = 2: Person 1 is the only one who wants to enter, so they just enter through the door.
```

限制：
- n == arrival.length == state.length
- 1 <= n <= 10^5
- 0 <= arrival[i] <= n
- arrival is sorted in non-decreasing order.
- state[i] is either 0 or 1.

## 解题

由于数组 arrival 为非严格单调递增，因此同一个方向的所有人到达门的顺序为编号递增顺序。可以使用两个队列分别存储进入门和离开门的所有人的编号，模拟每个人通过门的过程。两个队列分别为进入门队列和离开门队列

用 prev 表示上一秒的门的状态，可能的取值有三种，分别是 −1 表示未使用、0 表示进入、1 表示离开，初始时 prev=−1。用 time 表示当前时间，初始时 time=0。用 index 表示数组 arrival 和 index 的下标，初始时 index=0

当 index<n 或两个队列中至少有一个不为空时，还有人未通过门，需要执行如下循环操作

- 当 index<n 且 arrival[index]≤time 时，编号 index 的人已经到达门，根据 state[index] 确定编号 index 的人的方向，当 state[index]=0 时将编号 index 添加到进入门队列，当 state[index]=1 时将编号 index 添加到离开门队列，然后将 index 的值增加 1。重复该操作直到 index=n 或 arrival[index]>time

- 根据两个队列是否为空以及上一秒的门的状态 prev，决定当前时间通过门的人的编号，执行相应的操作

  - 如果两个队列都不为空且 prev 的值等于 −1 或 1，则当前时间有一个人离开，从离开门队列取出队首元素记为 person，将 time 填入 answer[person]，将 prev 的值更新为 1
  - 如果两个队列都不为空且 prev 的值等于 0，则当前时间有一个人进入，从进入门队列取出队首元素记为 person，将 time 填入 answer[person]，将 prev 的值更新为 0
  - 如果只有进入门队列不为空，则当前时间有一个人进入，从进入门队列取出队首元素记为 person，将 time 填入 answer[person]，将 prev 的值更新为 0
  - 如果只有离开门队列不为空，则当前时间有一个人离开，从离开门队列取出队首元素记为 person，将 time 填入 answer[person]，将 prev 的值更新为 1
  - 如果两个队列都为空，则当前时间没有人通过门，将 prev 的值更新为 −1

- 将 time 的值增加 1

当 index=n 且两个队列都为空时，所有人都通过门，返回答案数组 answer

```js
/**
 * @param {number[]} arrival
 * @param {number[]} state
 * @return {number[]}
 */
/**
 * @param {number[]} arrival
 * @param {number[]} state
 * @return {number[]}
 */
var timeTaken = function(arrival, state) {
    const ENTER = 0;
    const EXIT = 1;
    const UNUSED = -1;

    const n = arrival.length;
    const res = new Array(n).fill(0);

    const queues = [[], []];
    let enterHead = 0;
    let exitHead = 0;

    let time = 0;
    let i = 0;
    let prev = UNUSED;

    while (i < n || enterHead < queues[ENTER].length || exitHead < queues[EXIT].length) {
        while (i < n && arrival[i] <= time) {
            queues[state[i]].push(i);
            i++;
        }

        const hasEnter = enterHead < queues[ENTER].length;
        const hasExit = exitHead < queues[EXIT].length;

        if (hasEnter && hasExit) {
            if (prev === UNUSED || prev === EXIT) {
                const person = queues[EXIT][exitHead++];
                res[person] = time;
                prev = EXIT;
            } else {
                const person = queues[ENTER][enterHead++];
                res[person] = time;
                prev = ENTER;
            }
        } else if (hasEnter) {
            const person = queues[ENTER][enterHead++];
            res[person] = time;
            prev = ENTER;
        } else if (hasExit) {
            const person = queues[EXIT][exitHead++];
            res[person] = time;
            prev = EXIT;
        } else {
            prev = UNUSED;
        }

        time++;
    }

    return res;
};
```
```python
from collections import deque
from typing import List

class Solution:
    def timeTaken(self, arrival: List[int], state: List[int]) -> List[int]:
        ENTER = 0
        EXIT = 1
        UNUSED = -1

        n = len(arrival)
        res = [0] * n

        queues = [deque(), deque()]  # queues[0]: enter, queues[1]: exit

        time = 0
        i = 0
        prev = UNUSED

        while i < n or queues[ENTER] or queues[EXIT]:
            # Add all people who have arrived by current time.
            while i < n and arrival[i] <= time:
                queues[state[i]].append(i)
                i += 1

            # If both queues have people, decide by previous door state.
            if queues[ENTER] and queues[EXIT]:
                if prev == UNUSED or prev == EXIT:
                    person = queues[EXIT].popleft()
                    res[person] = time
                    prev = EXIT
                else:
                    person = queues[ENTER].popleft()
                    res[person] = time
                    prev = ENTER

            # Only enter queue has people.
            elif queues[ENTER]:
                person = queues[ENTER].popleft()
                res[person] = time
                prev = ENTER

            # Only exit queue has people.
            elif queues[EXIT]:
                person = queues[EXIT].popleft()
                res[person] = time
                prev = EXIT

            # No one is waiting at this time.
            else:
                prev = UNUSED

            time += 1

        return res
```
```java
import java.util.*;

class Solution {
    static final int ENTER = 0;
    static final int EXIT = 1;
    static final int UNUSED = -1;

    public int[] timeTaken(int[] arrival, int[] state) {
        int n = arrival.length;
        int[] res = new int[n];

        Queue<Integer>[] queues = new Queue[2];
        queues[ENTER] = new ArrayDeque<>();
        queues[EXIT] = new ArrayDeque<>();

        int time = 0;
        int i = 0;
        int prev = UNUSED;

        while (i < n || !queues[ENTER].isEmpty() || !queues[EXIT].isEmpty()) {
            while (i < n && arrival[i] <= time) {
                queues[state[i]].offer(i);
                i++;
            }

            if (!queues[ENTER].isEmpty() && !queues[EXIT].isEmpty()) {
                if (prev == UNUSED || prev == EXIT) {
                    int person = queues[EXIT].poll();
                    res[person] = time;
                    prev = EXIT;
                } else {
                    int person = queues[ENTER].poll();
                    res[person] = time;
                    prev = ENTER;
                }
            } else if (!queues[ENTER].isEmpty()) {
                int person = queues[ENTER].poll();
                res[person] = time;
                prev = ENTER;
            } else if (!queues[EXIT].isEmpty()) {
                int person = queues[EXIT].poll();
                res[person] = time;
                prev = EXIT;
            } else {
                prev = UNUSED;
            }

            time++;
        }

        return res;
    }
}
```

- 时间复杂度：`O(n)`，其中 n 是数组 arrival 和 state 的长度。由于所有人的到达时间都不超过 n 秒，因此总时间不超过 2n 秒，每一秒的操作时间是 O(1)，时间复杂度是 `O(n)`
- 空间复杂度：`O(n)`，其中 n 是数组 arrival 和 state 的长度。两个队列中的元素个数之和不超过 n
