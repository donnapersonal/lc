# 581.最短无序连续子数组

题目链接：[581.最短无序连续子数组](https://leetcode.cn/problems/shortest-unsorted-continuous-subarray/)

## 题目大意

给你一个整数数组 nums，需要找出一个 连续子数组 ，如果对这个子数组进行升序排序，那么整个数组都会变为升序排序

请找出符合题意的 最短 子数组，并输出它的长度

```js
Example 1:
Input: nums = [2,6,4,8,10,9,15]
Output: 5
Explanation: You need to sort [6, 4, 8, 10, 9] in ascending order to make the whole array sorted in ascending order.

Example 2:
Input: nums = [1,2,3,4]
Output: 0

Example 3:
Input: nums = [1]
Output: 0
```

限制:
- 1 <= nums.length <= 10^4
- -10^5 <= nums[i] <= 10^5

## 解题

假设 numsB 在 nums 中对应区间为 [left,right]

注意到 numsB 和 numsC 中任意一个数都大于等于 numsA 中任意一个数。因此有 numsA 中每一个数 numsi 都满足：

可以从大到小枚举 i，用一个变量 minn 记录 min_{j=i+1}^{n−1} numsj。每次移动 i，都可以 O(1) 地更新 minn。这样最后一个使得不等式不成立的 i 即为 left。left 左侧即为 numsA 能取得的最大范围

同理，可以用类似的方法确定 right。在实际代码中，可以在一次循环中同时完成左右边界的计算

特别地，需要特判 nums 有序的情况，此时 numsB 的长度为 0。当我们计算完成左右边界，即可返回 numsB 的长度

```js
/**
 * @param {number[]} nums
 * @return {number}
 */
var findUnsortedSubarray = function(nums) {
    const n = nums.length;

    let maxSeen = Number.NEGATIVE_INFINITY;
    let minSeen = Number.POSITIVE_INFINITY;

    let right = -1;
    let left = -1;

    for (let i = 0; i < n; i++) {
        if (maxSeen > nums[i]) {
            right = i;
        } else {
            maxSeen = nums[i];
        }

        const j = n - i - 1;

        if (minSeen < nums[j]) {
            left = j;
        } else {
            minSeen = nums[j];
        }
    }

    return right === -1 ? 0 : right - left + 1;
};
```
```python
class Solution:
    def findUnsortedSubarray(self, nums: List[int]) -> int:
        n = len(nums)
        max_seen, right = float("-inf"), -1
        min_seen, left = float("inf"), -1

        for i in range(n):
            # Scan from left to right to find the right boundary.
            if max_seen > nums[i]:
                right = i
            else:
                max_seen = nums[i]

            # Scan from right to left to find the left boundary.
            j = n - i - 1
            if min_seen < nums[j]:
                left = j
            else:
                min_seen = nums[j]

        return 0 if right == -1 else right - left + 1
```

```java
class Solution {
    public int findUnsortedSubarray(int[] nums) {
        int n = nums.length;

        int maxSeen = Integer.MIN_VALUE;
        int minSeen = Integer.MAX_VALUE;

        int right = -1;
        int left = -1;

        for (int i = 0; i < n; i++) {
            if (maxSeen > nums[i]) {
                right = i;
            } else {
                maxSeen = nums[i];
            }

            int j = n - i - 1;

            if (minSeen < nums[j]) {
                left = j;
            } else {
                minSeen = nums[j];
            }
        }

        return right == -1 ? 0 : right - left + 1;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 是给定数组的长度，仅需要遍历该数组一次
- 时间复杂度：`O(1)`，只需要常数的空间保存若干变量
