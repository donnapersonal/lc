# 1028.从前序遍历中恢复一棵树

题目链接：[1028.从前序遍历中恢复一棵树](https://leetcode.com/problems/recover-a-tree-from-preorder-traversal/)

## 题目大意

我们对二叉树的根节点执行前序深度优先搜索（DFS）

遍历过程中，我们在每个节点输出 D 个短横线（其中 D 为该节点的深度），然后输出该节点的值。如果一个节点的深度为 D，则其直接子节点的深度为 D + 1。根节点的深度为 0

如果一个节点只有一个子节点，则该子节点一定是左子节点

给定此次遍历的输出结果，恢复该树并返回其根节点

- 1 <= Node.val <= 10^9

## 解题

这题最自然的方法是：`字符串解析 + 栈 + 构造二叉树`

栈里存什么？栈里维护从根节点到当前节点的路径

比如当前走到节点 3 时，栈可能是：[1, 2, 3] 表示当前路径是：1 -> 2 -> 3

> 为什么栈可以找到父节点？
> - 每个节点前面的 - 数量表示它的深度
> - 如果当前节点深度是 depth，那么它的父节点深度一定是：depth - 1
> - 而栈的长度如果调整到 depth，那么：stack[-1] 就是当前节点的父节点

```js
// Definition for a binary tree node.
// function TreeNode(val, left, right) {
//     this.val = val === undefined ? 0 : val;
//     this.left = left === undefined ? null : left;
//     this.right = right === undefined ? null : right;
// }

var recoverFromPreorder = function(traversal) {
    const stack = [];
    let i = 0;
    const n = traversal.length;

    while (i < n) {
        let depth = 0;

        // Count dashes to get the node depth
        while (i < n && traversal[i] === "-") {
            depth++;
            i++;
        }

        let value = 0;

        // Parse the node value
        while (i < n && traversal[i] >= "0" && traversal[i] <= "9") {
            value = value * 10 + Number(traversal[i]);
            i++;
        }

        const node = new TreeNode(value);

        // Keep only the path to this node's parent
        while (stack.length > depth) {
            stack.pop();
        }

        // Attach current node to its parent
        if (stack.length > 0) {
            const parent = stack[stack.length - 1];

            if (parent.left === null) {
                parent.left = node;
            } else {
                parent.right = node;
            }
        }

        stack.push(node);
    }

    return stack[0];
};
```

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):

class Solution:
    def recoverFromPreorder(self, traversal: str) -> Optional[TreeNode]:

        i = 0
        n = len(traversal)

        while i < n:
            depth = 0

            # Count dashes to get the node depth
            while i < n and traversal[i] == "-":
                depth += 1
                i += 1

            value = 0

            # Parse the node value
            while i < n and traversal[i].isdigit():
                value = value * 10 + int(traversal[i])
                i += 1

            node = TreeNode(value)

            # Keep only the path to this node's parent
            while len(stack) > depth:
                stack.pop()

            # Attach current node to its parent
            if stack:
                parent = stack[-1]

                if parent.left is None:
                    parent.left = node
                else:
                    parent.right = node

            stack.append(node)

        return stack[0]

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */

import java.util.*;

class Solution {
    public TreeNode recoverFromPreorder(String traversal) {
        Deque<TreeNode> stack = new ArrayDeque<>();
        int i = 0;
        int n = traversal.length();

        while (i < n) {
            int depth = 0;

            // Count dashes to get the node depth
            while (i < n && traversal.charAt(i) == '-') {
                depth++;
                i++;
            }

            int value = 0;

            // Parse the node value
            while (i < n && Character.isDigit(traversal.charAt(i))) {
                value = value * 10 + (traversal.charAt(i) - '0');
                i++;
            }

            TreeNode node = new TreeNode(value);

            // Keep only the path to this node's parent
            while (stack.size() > depth) {
                stack.pollLast();
            }

            // Attach current node to its parent
            if (!stack.isEmpty()) {
                TreeNode parent = stack.peekLast();

                if (parent.left == null) {
                    parent.left = node;
                } else {
                    parent.right = node;
                }
            }

            stack.offerLast(node);
        }

        return stack.peekFirst();
    }
}
```

- 时间复杂度：`O(n)`，每个字符最多被扫描一次；每个节点最多入栈一次；每个节点最多出栈一次。所以总时间是：`O(n)`
- 空间复杂度：`O(h)`，其中：`h = 树的高度`
  - 栈里保存的是当前从根节点到某个节点的路径
  - 如果树比较平衡，空间接近：`O(log m)`，其中 m 是节点数
  - 如果树退化成链表，空间最坏是：`O(m)`
  - 如果用字符串长度 `n` 来表示，也可以说最坏是：`O(n)`
