# 539.最小时间差

题目链接：[539.最小时间差](https://leetcode.cn/problems/minimum-time-difference/)

## 题目大意

给定一个 24 小时制（小时:分钟 "HH:MM"）的时间列表，找出列表中任意两个时间的最小时间差并以分钟数表示

```js
Example 1:
Input: timePoints = ["23:59","00:00"]
Output: 1

Example 2:
Input: timePoints = ["00:00","23:59","00:00"]
Output: 0
```

限制：
- 2 <= timePoints.length <= 2 * 10^4
- timePoints[i] is in the format "HH:MM".

## 解题

核心思路

一天只有：`24 * 60 = 1440` 种不同的分钟

所以可以用一个长度为 1440 的布尔数组 seen 来记录每个时间点是否出现过

例如：

"00:00" -> 0
"00:01" -> 1
"23:59" -> 1439

如果某个分钟已经出现过，说明有重复时间，最小时间差直接是：0

**为什么可以不用排序？**
- 因为 seen[minute] 的下标本身就是按时间从小到大排列的
- 所以只需要从 0 扫到 1439，就相当于按时间顺序遍历所有出现过的时间点

这题可以利用“一天只有 1440 分钟”的特点，用桶数组记录出现过的时间；然后按分钟顺序扫描桶，比较相邻时间差，并额外比较首尾跨天差值

```js
/**
 * @param {string[]} timePoints
 * @return {number}
 */
var findMinDifference = function(timePoints) {
    if (timePoints.length > 1440) {
        return 0;
    }

    const seen = Array(1440).fill(false);

    for (const t of timePoints) {
        const hours = Number(t.slice(0, 2));
        const mins = Number(t.slice(3, 5));
        const minutes = hours * 60 + mins;

        if (seen[minutes]) {
            return 0;
        }

        seen[minutes] = true;
    }

    let prev = -1;
    let first = -1;
    let last = -1;
    let res = 1440;

    for (let minute = 0; minute < 1440; minute++) {
        if (seen[minute]) {
            if (first === -1) {
                first = minute;
            }

            if (prev !== -1) {
                res = Math.min(res, minute - prev);
            }

            prev = minute;
            last = minute;
        }
    }

    res = Math.min(res, first + 1440 - last);

    return res;
};
```

class Solution:
    def findMinDifference(self, timePoints: List[str]) -> int:
        if len(timePoints) > 1440:
            return 0

        seen = [False] * 1440

        for t in timePoints:
            minutes = int(t[:2]) * 60 + int(t[3:])
            if seen[minutes]:
                return 0
            seen[minutes] = True

        prev = -1
        first = -1
        last = -1

                if first == -1:
                    first = minute
                if prev != -1:
                    res = min(res, minute - prev)
                prev = minute
                last = minute

        res = min(res, first + 1440 - last)
        return res

```java
import java.util.*;

class Solution {
    public int findMinDifference(List<String> timePoints) {
        if (timePoints.size() > 1440) {
            return 0;
        }

        boolean[] seen = new boolean[1440];

        for (String t : timePoints) {
            int hours = Integer.parseInt(t.substring(0, 2));
            int minutesPart = Integer.parseInt(t.substring(3, 5));
            int minutes = hours * 60 + minutesPart;

            if (seen[minutes]) {
                return 0;
            }

            seen[minutes] = true;
        }

        int prev = -1;
        int first = -1;
        int last = -1;
        int res = 1440;

        for (int minute = 0; minute < 1440; minute++) {
            if (seen[minute]) {
                if (first == -1) {
                    first = minute;
                }

                if (prev != -1) {
                    res = Math.min(res, minute - prev);
                }

                prev = minute;
                last = minute;
            }
        }

        res = Math.min(res, first + 1440 - last);

        return res;
    }
}
```

- 时间复杂度：`O(n + 1440)`，n = len(timePoints)
  - 遍历输入时间点一次：O(n)
  - 扫描 1440 个分钟桶：O(1440)
  - 由于 1440 是固定常数，也可以说：`O(n)`
- 空间复杂度：`O(1440)`
  - 因为用了一个长度为 1440 的布尔数组
  - 由于 1440 是固定常数，也可以说：`O(1)`
