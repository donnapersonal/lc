# 2054.两个最好的不重叠活动

题目链接：[2054.两个最好的不重叠活动](https://leetcode.cn/problems/two-best-non-overlapping-events/)

## 题目大意

给你一个下标从 0 开始的二维整数数组 events ，其中 events[i] = [startTimei, endTimei, valuei] 。第 i 个活动开始于 startTimei ，结束于 endTimei ，如果你参加这个活动，那么你可以得到价值 valuei 。你 最多 可以参加 两个时间不重叠 活动，使得它们的价值之和 最大 

请你返回价值之和的 最大值 

注意，活动的开始时间和结束时间是 包括 在活动时间内的，也就是说，你不能参加两个活动且它们之一的开始时间等于另一个活动的结束时间。更具体的，如果你参加一个活动，且结束时间为 t ，那么下一个活动必须在 t + 1 或之后的时间开始

![alt text](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.26m3j3e1zu.webp)

限制:
- 2 <= events.length <= 10^5
- events[i].length == 3
- 1 <= startTimei <= endTimei <= 10^9
- 1 <= valuei <= 10^6

```js
var maxTwoEvents = function(events) {
    const timeline = [];

    for (const [start, end, value] of events) {
        timeline.push([start, 0, value]); // start event
        timeline.push([end, 1, value]);   // end event
    }

    // Sort by time first, then event type.
    // Important: start event should come before end event at the same time.
    timeline.sort((a, b) => {
        if (a[0] !== b[0]) return a[0] - b[0];
        return a[1] - b[1];
    });

    let bestFirst = 0;
    let res = 0;

    for (const [time, eventType, value] of timeline) {
        if (eventType === 0) {
            // Current event starts.
            // Combine it with the best event that ended strictly before this time.
            res = Math.max(res, bestFirst + value);
        } else {
            // Current event ends.
            // It becomes available for future events.
            bestFirst = Math.max(bestFirst, value);
        }
    }

    return res;
};
```

```java
import java.util.*;

class Solution {
    public int maxTwoEvents(int[][] events) {
        List<int[]> timeline = new ArrayList<>();

        for (int[] event : events) {
            int start = event[0];
            int end = event[1];
            int value = event[2];

            timeline.add(new int[]{start, 0, value}); // start event
            timeline.add(new int[]{end, 1, value});   // end event
        }

        // Sort by time first, then event type.
        // Important: start event should come before end event at the same time.
        Collections.sort(timeline, (a, b) -> {
            if (a[0] != b[0]) {
                return a[0] - b[0];
            }
            return a[1] - b[1];
        });

        int bestFirst = 0;
        int res = 0;

        for (int[] event : timeline) {
            int eventType = event[1];
            int value = event[2];

            if (eventType == 0) {
                // Current event starts.
                // Combine it with the best event that ended strictly before this time.
                res = Math.max(res, bestFirst + value);
            } else {
                // Current event ends.
                // It becomes available for future events.
                bestFirst = Math.max(bestFirst, value);
            }
        }

        return res;
    }
}
```
