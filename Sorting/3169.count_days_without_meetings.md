# 3169.无需开会的工作日

题目链接：[3169.无需开会的工作日](https://leetcode.cn/problems/count-days-without-meetings/)

## 题目大意

给你一个正整数 days，表示员工可工作的总天数（从第 1 天开始）。另给你一个二维数组 meetings，长度为 n，其中 meetings[i] = [start_i, end_i] 表示第 i 次会议的开始和结束天数（包含首尾）

返回员工可工作且没有安排会议的天数

注意：会议时间可能会有重叠

```js
Example 1:
Input: days = 10, meetings = [[5,7],[1,3],[9,10]]
Output: 2
Explanation:
There is no meeting scheduled on the 4th and 8th days.

Example 2:
Input: days = 5, meetings = [[2,4],[1,3]]
Output: 1
Explanation:
There is no meeting scheduled on the 5th day.

Example 3:
Input: days = 6, meetings = [[1,6]]
Output: 0
Explanation:
Meetings are scheduled for all working days.
```

限制：
- 1 <= days <= 10^9
- 1 <= meetings.length <= 10^5
- meetings[i].length == 2
- 1 <= meetings[i][0] <= meetings[i][1] <= days

## 解题

正难则反，答案等于 days 减「有会议安排的天数」

要计算有会议安排的天数，做法同 56. 合并区间的题解。累加每个区间的长度，即为有会议安排的天数

由于本题只需计算合并区间的长度，所以只需记录当前合并区间的左右端点

```js
/**
 * @param {number} days
 * @param {number[][]} meetings
 * @return {number}
 */
var countDays = function(days, meetings) {
    meetings.sort((a, b) => a[0] - b[0]);

    let start = 1;
    let end = 0;

    for (const [s, e] of meetings) {
        if (s > end) {
            days -= end - start + 1;
            start = s;
        }

        end = Math.max(end, e);
    }

    days -= end - start + 1;

    return days;
};
```
```python
class Solution:
    def countDays(self, days: int, meetings: List[List[int]]) -> int:
        meetings.sort(key=lambda p: p[0])  # 按照左端点从小到大排序
        start, end = 1, 0  # 当前合并区间的左右端点
        for s, e in meetings:
            if s > end:  # 不相交
                days -= end - start + 1  # 当前合并区间的长度
                start = s  # 下一个合并区间的左端点
            end = max(end, e)
        days -= end - start + 1  # 最后一个合并区间的长度
        return days
```
```java
import java.util.*;

class Solution {
    public int countDays(int days, int[][] meetings) {
        Arrays.sort(meetings, (a, b) -> Integer.compare(a[0], b[0]));

        int start = 1;
        int end = 0;

        for (int[] meeting : meetings) {
            int s = meeting[0];
            int e = meeting[1];

            if (s > end) {
                days -= end - start + 1;
                start = s;
            }

            end = Math.max(end, e);
        }

        days -= end - start + 1;

        return days;
    }
}
```

- 时间复杂度：`O(nlogn)`，其中 `n` 是 `meetings` 的长度，瓶颈在排序上
- 空间复杂度：`O(1)`，忽略排序的栈开销
