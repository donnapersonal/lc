# 2548.填满背包的最大价格

题目链接：[2548.填满背包的最大价格](https://leetcode.cn/problems/maximum-price-to-fill-a-bag/)

## 题目大意

给定一个二维整数数组 items，其中 items[i] = [pricei, weighti] 表示第 i 个物品的价格和重量

还给定一个 正 整数容量 capacity 

每个物品可以分成两个部分，比率为 part1 和 part2，其中 part1 + part2 == 1 

- 第一个物品的重量是 weighti * part1，价格是 pricei * part1 
- 同样，第二个物品的重量是 weighti * part2，价格是 pricei * part2

使用给定的物品，返回填满容量为 capacity 的背包所需的 最大总价格。如果无法填满背包，则返回 -1。与实际答案的差距在 10-5 以内的 实际答案 将被视为接受

```js
Example 1:
Input: items = [[50,1],[10,8]], capacity = 5
Output: 55.00000
Explanation: 
We divide the 2nd item into two parts with part1 = 0.5 and part2 = 0.5.
The price and weight of the 1st item are 5, 4. And similarly, the price and the weight of the 2nd item are 5, 4.
The array items after operation becomes [[50,1],[5,4],[5,4]]. 
To fill a bag with capacity 5 we take the 1st element with a price of 50 and the 2nd element with a price of 5.
It can be proved that 55.0 is the maximum total price that we can achieve.

Example 2:
Input: items = [[100,30]], capacity = 50
Output: -1.00000
Explanation: It is impossible to fill a bag with the given item.
```

限制：
- 1 <= items.length <= 10^5
- items[i].length == 2
- 1 <= pricei, weighti <= 10^4
- 1 <= capacity <= 10^9

## 解题

这道题目是分数背包问题，每个物品可以选择一部分，根据选择的重量比例计算选择部分的价格

根据`贪心`策略，为了使总价格最大，应考虑每个物品的单位重量价格，按照单位重量价格`降序`选择物品

贪心策略的正确性说明：根据贪心策略选择单位重量价格最大的物品填满背包的容量 `capacity` 之后，将总价格记为 `totalPrice`。如果不遵循贪心策略，假设有重量 `w` 的物品在贪心策略下的单位重量价格是 `p`，将这部分物品替换成单位重量价格是 `p′` 的物品，`p′ < p`，则替换之后，总价格为 `totalPrice′ = totalPrice + (p′−p)×w`，由于 `p′ < p`，因此 `totalPrice′ < totalPrice`，即不遵循贪心策略的总价格一定不会优于遵循贪心策略的总价格。因此使用贪心策略可以得到最大总价格

使用贪心策略计算最大总价格的做法如下
- 将数组 items 按单位重量价格降序排序。为了避免浮点数运算产生误差，应使用乘法代替除法，即对于两个物品 `a` 和 `b`， 
`a[0]/a[1] > b[0]/b[01]` 等价于 `a[0]×b[1]>b[0]×a[1]`
- 从左到右遍历排序后的数组 `items`，遍历过程中维护背包的剩余容量 `remain`，初始时 `remain=capacity`。对于遍历到的每个物品，将价格和重量分别记为 `price` 和 `weight`，执行如下操作
  - 计算当前物品可以放进背包的最大重量 `curWeight = min(weight,remain)`
  - 如果 `curWeight=weight`，则当前物品全部放进背包，将总价格增加 `price`；如果 `curWeight<weight`，则当前物品只有重量为 `curWeight` 的部分放进背包，将总价格增加 `price/weight × curWeight`
  - 将 `remain` 的值减少 `curWeight`

重复上述操作，直到数组 `items` 遍历结束或 `remain` 变成 `0` 时结束操作。当结束操作时，如果 `remain=0`，则背包填满，返回总价格，否则背包无法填满，返回 `−1`

```js
/**
 * @param {number[][]} items
 * @param {number} capacity
 * @return {number}
 */
var maxPrice = function(items, capacity) {
    let totalPrice = 0;
        
    items.sort((a, b) => b[0] * a[1] - a[0] * b[1]);
    
    let remain = capacity;
    const length = items.length;
    
    for (let i = 0; i < length && remain > 0; i++) {
        const price = items[i][0];
        const weight = items[i][1];
        const currWeight = Math.min(weight, remain);
        
        if (currWeight === weight) {
            totalPrice += price;
        } else {
            totalPrice += (price / weight) * currWeight;
        }
        
        remain -= currWeight;
    }
    
    return remain === 0 ? totalPrice : -1;
};
```

```python
class Solution:
    def maxPrice(self, items: List[List[int]], capacity: int) -> float:
        total_price = 0.0

        items.sort(key=lambda x: x[0] / x[1], reverse=True)

        remain = capacity
        length = len(items)

        for i in range(length):
            if remain <= 0:
                break

            price = items[i][0]
            weight = items[i][1]
            cur_weight = min(weight, remain)

            if cur_weight == weight:
                total_price += price
            else:
                total_price += price / weight * cur_weight

            remain -= cur_weight

        return total_price if remain == 0 else -1
```
```java
class Solution {
    public double maxPrice(int[][] items, int capacity) {
        double totalPrice = 0;
        Arrays.sort(items, (a, b) -> b[0] * a[1] - a[0] * b[1]);
        int remain = capacity;
        int length = items.length;
        for (int i = 0; i < length && remain > 0; i++) {
            double price = items[i][0];
            int weight = items[i][1];
            int curWeight = Math.min(weight, remain);
            if (curWeight == weight) {
                totalPrice += price;
            } else {
                totalPrice += price / weight * curWeight;
            }
            remain -= curWeight;
        }
        return remain == 0 ? totalPrice : -1;
    }
}
```

- 时间复杂度：`O(nlogn)`，其中 `n` 是数组 `items` 的长度。对数组 `items` 排序的时间是 `O(nlogn)`，排序之后需要遍历数组 items 一次计算最大总价格，因此时间复杂度是 `O(nlogn)`
- 空间复杂度：`O(n)`，其中 `n` 是数组 `items` 的长度。由于数组 `items` 的元素是数组，因此排序的空间复杂度是 `O(n)`
