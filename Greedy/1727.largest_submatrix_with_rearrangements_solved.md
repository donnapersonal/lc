# 1727.重新排列后的最大子矩阵

题目链接：[1727.重新排列后的最大子矩阵](https://leetcode.cn/problems/largest-submatrix-with-rearrangements/)

## 题目大意

给你一个二进制矩阵 matrix ，它的大小为 m x n ，你可以将 matrix 中的 列 按任意顺序重新排列

请你返回最优方案下将 matrix 重新排列后，全是 1 的最大子矩阵面积

![alt text](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.icqltywv4.webp)

限制:
- m == matrix.length, n == matrix[i].length
- 1 <= m * n <= 10^5
- matrix[i][j] is either 0 or 1.

## 解题

```js
/**
 * @param {number[][]} matrix
 * @return {number}
 */
var largestSubmatrix = function(matrix) {
    const m = matrix.length;
    const n = matrix[0].length;

    const heights = new Array(n).fill(0);
    let res = 0;

    for (const row of matrix) {
        // Update consecutive 1s height for each column
        for (let j = 0; j < n; j++) {
            if (row[j] === 0) {
                heights[j] = 0;
            } else {
                heights[j]++;
            }
        }

        // Since columns can be rearranged, sort heights descending
        const sortedHeights = [...heights].sort((a, b) => b - a);

        // Try each width
        for (let i = 0; i < n; i++) {
            const width = i + 1;
            const height = sortedHeights[i];
            res = Math.max(res, width * height);
        }
    }

    return res;
};
```
```python
from typing import List

class Solution:
    def largestSubmatrix(self, matrix: List[List[int]]) -> int:
        m, n = len(matrix), len(matrix[0])
        heights = [0] * n
        res = 0

        for row in matrix:
            for j in range(n):
                if row[j] == 0:
                    heights[j] = 0
                else:
                    heights[j] += 1

            sorted_heights = sorted(heights, reverse=True)

            for width, height in enumerate(sorted_heights, start=1):
                res = max(res, width * height)

        return res
```

```java
import java.util.*;

class Solution {
    public int largestSubmatrix(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;

        int[] heights = new int[n];
        int res = 0;

        for (int i = 0; i < m; i++) {
            // Update consecutive 1s height for each column
            for (int j = 0; j < n; j++) {
                if (matrix[i][j] == 0) {
                    heights[j] = 0;
                } else {
                    heights[j]++;
                }
            }

            // Since columns can be rearranged, sort heights descending
            int[] sortedHeights = heights.clone();
            Arrays.sort(sortedHeights); // ascending order

            // Iterate from largest height to smallest height
            for (int j = n - 1; j >= 0; j--) {
                int height = sortedHeights[j];
                int width = n - j;
                res = Math.max(res, width * height);
            }
        }

        return res;
    }
}
```

- 时间复杂度：`O(mnlogn)`，其中 `m` 和 `n` 分别是 `matrix` 的行数和列数。瓶颈在排序上，有 `m` 行，每行都要跑一个 `O(nlogn)` 的排序
- 空间复杂度：`O(n)`
