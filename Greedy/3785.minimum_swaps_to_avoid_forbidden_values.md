# 3785.避免禁用值的最小交换次数

题目链接：[3785.避免禁用值的最小交换次数](https://leetcode.cn/problems/minimum-swaps-to-avoid-forbidden-values/)

## 题目大意

给你两个长度为 n 的整数数组 nums 和 forbidden

你可以执行以下操作任意次（包括零次）：
- 选择两个 不同 下标 i 和 j，然后交换 nums[i] 和 nums[j]
  
返回使得对于每个下标 i，nums[i] 不等于 forbidden[i] 所需的 最小 交换次数。如果无论如何都无法满足条件，返回 -1

```js
Example 1:
Input: nums = [1,2,3], forbidden = [3,2,1]
Output: 1
Explanation:
One optimal set of swaps:
Select indices i = 0 and j = 1 in nums and swap them, resulting in nums = [2, 1, 3].
After this swap, for every index i, nums[i] is not equal to forbidden[i].

Example 2:
Input: nums = [4,6,6,5], forbidden = [4,6,5,5]
Output: 2
Explanation:
One optimal set of swaps:
Select indices i = 0 and j = 2 in nums and swap them, resulting in nums = [6, 6, 4, 5].
Select indices i = 1 and j = 3 in nums and swap them, resulting in nums = [6, 5, 4, 6].
After these swaps, for every index i, nums[i] is not equal to forbidden[i].

Example 3:
Input: nums = [7,7], forbidden = [8,7]
Output: -1
Explanation:
It is not possible to make nums[i] different from forbidden[i] for all indices.

Example 4:
Input: nums = [1,2], forbidden = [2,1]
Output: 0
Explanation:
No swaps are required because nums[i] is already different from forbidden[i] for all indices, so the answer is 0.
```

限制：
- 1 <= n == nums.length == forbidden.length <= 10^5
- 1 <= nums[i], forbidden[i] <= 10^9

## 解题

核心思路
- 先判断是否无解
  - 对于某个数字 x，如果它在 nums 和 forbidden 中一共出现超过 n 次，那么一定会有某些位置无法避开 x
  - 因为一共有 n 个位置，如果 x 出现次数太多，就无法让所有 nums[i] != forbidden[i]
  - 所以直接返回 -1
- 找出当前冲突位置
  - 只关心这些位置：nums[i] == forbidden[i]
  - 因为这些位置是非法的，需要通过交换修复
  - s​​ame[x] 表示：有多少个冲突位置的值是 x
    例如：
    nums      = [1, 1, 2, 3]
    forbidden = [1, 2, 2, 4]
    冲突位置是：
    index 0: 1 == 1
    index 2: 2 == 2
    所以：same = {1: 1, 2: 1}

- 统计冲突总数和最多冲突值
  - k = same.total() -> k	总共有多少个冲突位置
  - mx = max(same.values(), default=0) -> mx	某个数字造成的最多冲突次数


**为什么答案是这个：return max((k + 1) // 2, mx)**

有两个下界
- 下界 1：每次交换最多修复 2 个冲突
  - 如果有 k 个冲突位置，一次交换最多让两个冲突位置都变合法
  - 所以至少需要：ceil(k / 2)
  - 代码写法是：(k + 1) // 2
- 下界 2：某个数字冲突太多时，需要至少 mx 次
  - 如果某个数字 x 出现了很多冲突，比如：same[x] = mx
  - 这些位置上的 nums[i] 都是 x，而 forbidden[i] 也都是 x
  - 每次交换最多只能解决一个这种“同值冲突”，所以至少需要：mx

所以最终答案
- 答案必须同时满足两个下界：max(ceil(k / 2), mx)
- 也就是：max((k + 1) // 2, mx)

```js
/**
 * @param {number[]} nums
 * @param {number[]} forbidden
 * @return {number}
 */
var minSwaps = function(nums, forbidden) {
    const n = nums.length;

    const total = new Map();

    for (const x of nums) {
        total.set(x, (total.get(x) || 0) + 1);
    }

    for (const x of forbidden) {
        total.set(x, (total.get(x) || 0) + 1);
    }

    for (const count of total.values()) {
        if (count > n) {
            return -1;
        }
    }

    const same = new Map();

    for (let i = 0; i < n; i++) {
        if (nums[i] === forbidden[i]) {
            same.set(nums[i], (same.get(nums[i]) || 0) + 1);
        }
    }

    let k = 0;
    let mx = 0;

    for (const count of same.values()) {
        k += count;
        mx = Math.max(mx, count);
    }

    return Math.max(Math.floor((k + 1) / 2), mx);
};
```
```python
class Solution:
    def minSwaps(self, nums: List[int], forbidden: List[int]) -> int:
        n = len(nums)
        total = Counter(nums) + Counter(forbidden)
        if any(c > n for c in total.values()):
            return -1

        same = Counter(x for x, y in zip(nums, forbidden) if x == y)
        k = same.total()
        mx = max(same.values(), default=0)
        return max((k + 1) // 2, mx)
```
```java
import java.util.*;

class Solution {
    public int minSwaps(int[] nums, int[] forbidden) {
        int n = nums.length;

        Map<Integer, Integer> total = new HashMap<>();

        for (int x : nums) {
            total.put(x, total.getOrDefault(x, 0) + 1);
        }

        for (int x : forbidden) {
            total.put(x, total.getOrDefault(x, 0) + 1);
        }

        for (int count : total.values()) {
            if (count > n) {
                return -1;
            }
        }

        Map<Integer, Integer> same = new HashMap<>();

        for (int i = 0; i < n; i++) {
            if (nums[i] == forbidden[i]) {
                same.put(nums[i], same.getOrDefault(nums[i], 0) + 1);
            }
        }

        int k = 0;
        int mx = 0;

        for (int count : same.values()) {
            k += count;
            mx = Math.max(mx, count);
        }

        return Math.max((k + 1) / 2, mx);
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 是 `nums` 的长度
- 空间复杂度：`O(n)`
