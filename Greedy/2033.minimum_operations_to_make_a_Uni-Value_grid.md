# 2033.获取单值网格的最小操作数

题目链接：[2033.获取单值网格的最小操作数](https://leetcode.cn/problems/minimum-operations-to-make-a-uni-value-grid/)

## 题目大意

给你一个大小为 m x n 的二维整数网格 grid 和一个整数 x 。每一次操作，你可以对 grid 中的任一元素 加 x 或 减 x 

单值网格 是全部元素都相等的网格

返回使网格化为单值网格所需的 最小 操作数。如果不能，返回 -1 

![example](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.8adw5f72pn.webp)

限制：
- m == mat.length, n == mat[i].length
- 1 <= m, n <= 500
- 1 <= mat[i][j] <= 10^5
- No two adjacent cells are equal.

## 解题

这是一个经典的：Flatten + Modulo Invariant + Median / L1 Distance 最小化

也可以叫：`转化成一维数组 + 同余判断 + 中位数贪心`

**核心方法**

题目允许每次对某个元素：`+ x 或 - x`，所以一个数经过任意次操作后，它对 `x` 的余数不会变

例如：

x = 3

4 % 3 = 1
4 + 3 = 7, 7 % 3 = 1
4 - 3 = 1, 1 % 3 = 1

所以如果所有数最后要变成同一个值，那么它们必须满足：`所有数 % x 相同`，否则一定无解，返回 -1

**为什么选中位数？**
- 通过同余检查后，所有数都可以变成同一个目标值
- 问题变成：`选择一个 target，使 sum(abs(num - target)) 最小
- 这是经典结论：`一组数到某个点的绝对距离之和最小，目标点应该选中位数`

所以：

nums.sort()
median = nums[len(nums) // 2]

然后把所有数变成 `median`

每个数需要的操作次数是：`abs(num - median) / x`

所以总操作数是：`sum(abs(num - median)) // x`

```js
/**
 * @param {number[][]} grid
 * @param {number} x
 * @return {number}
 */
var minOperations = function(grid, x) {
    const nums = [];
    const remainder = grid[0][0] % x;

    for (const row of grid) {
        for (const val of row) {
            if (val % x !== remainder) {
                return -1;
            }
            nums.push(val);
        }
    }

    nums.sort((a, b) => a - b);

    const median = nums[Math.floor(nums.length / 2)];

    let operations = 0;
    for (const val of nums) {
        operations += Math.abs(val - median) / x;
    }

    return operations;
};
```

class Solution:

        nums = []
        remainder = grid[0][0] % x

        for row in grid:

                nums.append(val)

        nums.sort()
        median = nums[len(nums) // 2]

        return sum(abs(val - median) for val in nums) // x

```java
import java.util.*;

class Solution {
    public int minOperations(int[][] grid, int x) {
        List<Integer> nums = new ArrayList<>();
        int remainder = grid[0][0] % x;

        for (int[] row : grid) {
            for (int val : row) {
                if (val % x != remainder) {
                    return -1;
                }
                nums.add(val);
            }
        }

        Collections.sort(nums);

        int median = nums.get(nums.size() / 2);

        int operations = 0;
        for (int val : nums) {
            operations += Math.abs(val - median) / x;
        }

        return operations;
    }
}
```

- 时间复杂度：`O(mnlog(mn))` 或 `O(mn)`，其中 `m` 和 `n` 分别是 `grid` 的行数和列数，瓶颈在排序上
- 空间复杂度：`O(mn)`
