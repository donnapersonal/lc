# 1536.排布二进制网格的最少交换次数

题目链接：[1536.排布二进制网格的最少交换次数](https://leetcode.cn/problems/minimum-swaps-to-arrange-a-binary-grid/)

## 题目大意

给你一个 n x n 的二进制网格 grid，每一次操作中，你可以选择网格的 相邻两行 进行交换

一个符合要求的网格需要满足主对角线以上的格子全部都是 0 

请你返回使网格满足要求的最少操作次数，如果无法使网格符合要求，请你返回 -1 

主对角线指的是从 (1, 1) 到 (n, n) 的这些格子

![example](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.8dxht765aj.webp)

限制：
- n == grid.length == grid[i].length
- 1 <= n <= 200
- grid[i][j] is either 0 or 1

## 解题

设 tailZeros[i] 为 grid[i] 的尾零个数。示例 1 的 tailZeros=[0,1,2]，我们要通过交换，让尾零个数变成 [≥2,≥1,≥0]

问题等价于：
- 交换 tailZeros 中的相邻元素，使得 tailZeros[i]≥n−1−i，至少要交换多少次？

这可以贪心地做：从左到右，找第一个满足 tailZeros[i]≥n−1 的数，交换到第一个位置；然后找第一个满足 tailZeros[i]≥n−2 的数，交换到第二个位置；依此类推

为什么要选最近的？假设下标 p 和 q 都可以换到下标 0。由于越靠后，tailZeros[i]≥n−1−i 这个约束越宽松，所以能换到下标 0，也能换到后面所有的下标，所以下标 p 和 q 能发挥的作用是一样的。既然一样，谁离下标 0 更近就选谁换过去

由于 tailZeros[n−1]≥0 一定成立，所以外层循环只需循环到 i=n−2

```js
/**
 * @param {number[][]} grid
 * @return {number}
 */
var minSwaps = function(grid) {
    const n = grid.length;

    const tailZeros = new Array(n).fill(n);

    // Count trailing zeros for each row.
    for (let i = 0; i < n; i++) {
        for (let j = n - 1; j >= 0; j--) {
            if (grid[i][j] === 1) {
                tailZeros[i] = n - 1 - j;
                break;
            }
        }
    }

    let res = 0;

    for (let i = 0; i < n - 1; i++) {
        const needZeros = n - 1 - i;

        let found = false;

        for (let j = i; j < n; j++) {
            if (tailZeros[j] >= needZeros) {
                res += j - i;

                // Equivalent to:
                // tail_zeros[i + 1:j + 1] = tail_zeros[i:j]
                const selected = tailZeros[j];
                for (let k = j; k > i; k--) {
                    tailZeros[k] = tailZeros[k - 1];
                }
                tailZeros[i] = selected;

                found = true;
                break;
            }
        }

        if (!found) {
            return -1;
        }
    }

    return res;
};
```

```python
class Solution:
    def minSwaps(self, grid: List[List[int]]) -> int:
        n = len(grid)

        tail_zeros = [n] * n
        for i in range(n):
            for j in range(n - 1, -1, -1):
                if grid[i][j]:
                    tail_zeros[i] = n - 1 - j
                    break

        res = 0

        for i in range(n - 1):
            need_zeros = n - 1 - i

            for j in range(i, n):
                if tail_zeros[j] >= need_zeros:
                    res += j - i
                    tail_zeros[i + 1:j + 1] = tail_zeros[i:j]
                    break
            else:
                return -1

        return res
```

```java
class Solution {
    public int minSwaps(int[][] grid) {
        int n = grid.length;

        int[] tailZeros = new int[n];

        // Initialize tailZeros with n.
        for (int i = 0; i < n; i++) {
            tailZeros[i] = n;
        }

        // Count trailing zeros for each row.
        for (int i = 0; i < n; i++) {
            for (int j = n - 1; j >= 0; j--) {
                if (grid[i][j] == 1) {
                    tailZeros[i] = n - 1 - j;
                    break;
                }
            }
        }

        int res = 0;

        for (int i = 0; i < n - 1; i++) {
            int needZeros = n - 1 - i;

            int target = -1;

            for (int j = i; j < n; j++) {
                if (tailZeros[j] >= needZeros) {
                    target = j;
                    break;
                }
            }

            if (target == -1) {
                return -1;
            }

            res += target - i;

            // Equivalent to moving tailZeros[target] to index i,
            // and shifting tailZeros[i...target-1] one step right.
            int selected = tailZeros[target];

            for (int k = target; k > i; k--) {
                tailZeros[k] = tailZeros[k - 1];
            }

            tailZeros[i] = selected;
        }

        return res;
    }
```
