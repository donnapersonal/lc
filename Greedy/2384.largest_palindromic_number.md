# 2384. 最大回文数字

题目链接：[2384. 最大回文数字](https://leetcode.cn/problems/largest-palindromic-number/)

## 题目大意

给你一个仅由数字（0 - 9）组成的字符串 num 

请你找出能够使用 num 中数字形成的 最大回文 整数，并以字符串形式返回。该整数不含 前导零 

注意：
- 你 无需 使用 num 中的所有数字，但你必须使用 至少 一个数字
- 数字可以重新排序

```js
Example 1:
Input: num = "444947137"
Output: "7449447"
Explanation: 
Use the digits "4449477" from "444947137" to form the palindromic integer "7449447".
It can be shown that "7449447" is the largest palindromic integer that can be formed.

Example 2:
Input: num = "00009"
Output: "9"
Explanation: 
It can be shown that "9" is the largest palindromic integer that can be formed.
Note that the integer returned should not contain leading zeroes.

```

限制：
- 1 <= num.length <= 10^5
- num consists of digits.

## 解题

一个回文串（如 998767899，123321）可以被分成两部分：

- 两边对应的部分（如 9987 和 7899，123 和 321），这两部分中的数字每种都出现偶数次
- 中间单独一个数字（如 6），这部分是可选的
  
因此令 cnt[i] 表示数字 i 出现的次数，我们先从 9 到 0 枚举第一部分中出现的数，再看是否还有剩下的数放进中间单独的数字即可。复杂度 O(n)

由于题目要求不能有前导零，因此本题有一定实现细节，详见参考代码的注释

```js
/**
 * @param {string} num
 * @return {string}
 */
var largestPalindromic = function(num) {
    const cnt = new Array(10).fill(0);

    for (const c of num) {
        cnt[c.charCodeAt(0) - '0'.charCodeAt(0)]++;
    }

    // ans represents the left half of the palindrome
    let ans = [];

    // Build the left half from larger digits to smaller digits
    for (let i = 9; i >= 0; i--) {
        // If we reach 0 and have not added any non-zero digit,
        // adding 0 would create a leading zero, so stop.
        if (i === 0 && ans.length === 0) {
            break;
        }

        // Digits on both sides of a palindrome must appear in pairs
        const t = Math.floor(cnt[i] / 2);

        for (let j = 0; j < t; j++) {
            ans.push(String(i));
        }

        cnt[i] -= t * 2;
    }

    // ans2 is the reversed left half
    const ans2 = [...ans].reverse();

    // Pick the largest remaining digit as the middle digit
    for (let i = 9; i >= 0; i--) {
        if (cnt[i] > 0) {
            ans.push(String(i));
            break;
        }
    }

    return ans.join('') + ans2.join('');
};
```

```python
class Solution:
    def largestPalindromic(self, num: str) -> str:
        cnt = [0] * 10

        for c in num:
            cnt[ord(c) - ord('0')] += 1

        # ans represents the left half of the palindrome
        ans = []

        # Build the left half from larger digits to smaller digits
        for i in range(9, -1, -1):
            # If we reach 0 and have not added any non-zero digit,
            # adding 0 would create a leading zero, so stop.
            if i == 0 and not ans:
                break

            # Digits on both sides of a palindrome must appear in pairs
            t = cnt[i] // 2

            for _ in range(t):
                ans.append(str(i))

            cnt[i] -= t * 2

        # ans2 is the reversed left half
        ans2 = ans[::-1]

        # Pick the largest remaining digit as the middle digit
        for i in range(9, -1, -1):
            if cnt[i] > 0:
                ans.append(str(i))
                break

        return ''.join(ans) + ''.join(ans2)
```

```java
class Solution {
    public String largestPalindromic(String num) {
        int[] cnt = new int[10];

        for (char c : num.toCharArray()) {
            cnt[c - '0']++;
        }

        // ans represents the left half of the palindrome
        StringBuilder ans = new StringBuilder();

        // Build the left half from larger digits to smaller digits
        for (int i = 9; i >= 0; i--) {
            // If we reach 0 and have not added any non-zero digit,
            // adding 0 would create a leading zero, so stop.
            if (i == 0 && ans.length() == 0) {
                break;
            }

            // Digits on both sides of a palindrome must appear in pairs
            int t = cnt[i] / 2;

            for (int j = 0; j < t; j++) {
                ans.append((char) (i + '0'));
            }

            cnt[i] -= t * 2;
        }

        // ans2 is the reversed left half
        StringBuilder ans2 = new StringBuilder(ans).reverse();

        // Pick the largest remaining digit as the middle digit
        for (int i = 9; i >= 0; i--) {
            if (cnt[i] > 0) {
                ans.append((char) (i + '0'));
                break;
            }
        }

        return ans.toString() + ans2.toString();
    }
}
```

设 n = len(num)。

Time Complexity: O(n + 10) = O(n)
Space Complexity: O(10 + n) = O(n)

其中：

cnt 数组大小固定为 10
ans 最多存储一半字符，所以整体输出空间是 O(n)
