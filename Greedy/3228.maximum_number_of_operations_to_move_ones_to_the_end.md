# 3228.将 1 移动到末尾的最大操作次数

题目链接：[3228.将 1 移动到末尾的最大操作次数](https://leetcode.cn/problems/maximum-number-of-operations-to-move-ones-to-the-end/)

## 题目大意

给你一个 二进制字符串 s

你可以对这个字符串执行 任意次 下述操作：
- 选择字符串中的任一下标 i（ i + 1 < s.length ），该下标满足 s[i] == '1' 且 s[i + 1] == '0'
- 将字符 s[i] 向 右移 直到它到达字符串的末端或另一个 '1'。例如，对于 s = "010010"，如果我们选择 i = 1，结果字符串将会是 s = "000110"
  
返回你能执行的 最大 操作次数

```js
Example 1:
Input: s = "1001101"
Output: 4
Explanation:
We can perform the following operations:
Choose index i = 0. The resulting string is s = "0011101".
Choose index i = 4. The resulting string is s = "0011011".
Choose index i = 3. The resulting string is s = "0010111".
Choose index i = 2. The resulting string is s = "0001111".

Example 2:
Input: s = "00111"
Output: 0
```

限制：
- 1 <= s.length <= 10^5
- s[i] is either '0' or '1'.

## 解题

把 1 当作车，想象有一条长为 n 的道路上有一些车

题意：把所有的车都开到最右边。例如 011010 最终要变成 000111

如果优先操作右边的（能移动的）车，那么这些车都只需操作一次：
  011010
→ 011001
→ 010011
→ 000111
​	
一共需要操作 3 次（注意一次操作可以让一辆车移动多次）

而如果优先操作左边的（能移动的）车，这会制造大量的「堵车」，每辆车的操作次数会更多

  011010
→ 010110
→ 001110
→ 001101
→ 001011
→ 000111
​	
一共需要操作 5 次

算法：
- 从左到右遍历 s，同时用一个变量 count1 维护遍历到的 1 的个数
- 如果 s[i] 是 1，把 count1 增加 1
- 如果 s[i] 是 0 且 s[i−1] 是 1，意味着找到了一段道路，可以让 i 左边的每辆车都操作一次，把答案增加 count1
​- 遍历结束，返回答案

```js
/**
 * @param {string} s
 * @return {number}
 */
var maxOperations = function(s) {
    let res = 0;
    let count1 = 0;

    for (let i = 0; i < s.length; i++) {
        const c = s[i];

        if (c === '1') {
            count1++;
        } else if (i > 0 && s[i - 1] === '1') {
            res += count1;
        }
    }

    return res;
};
```
```python
class Solution:
    def maxOperations(self, s: str) -> int:
        res = count1 = 0

        for i, c in enumerate(s):
            if c == '1':
                count1 += 1
            elif i > 0 and s[i - 1] == '1':
                res += count1

        return res
```
```java
class Solution {
    public int maxOperations(String s) {
        int res = 0;
        int count1 = 0;

        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);

            if (c == '1') {
                count1++;
            } else if (i > 0 && s.charAt(i - 1) == '1') {
                res += count1;
            }
        }

        return res;
    }
}
```

- 时间复杂度：`O(n)`，其中 `n` 是 `s` 的长度
- 空间复杂度：`O(1)`
