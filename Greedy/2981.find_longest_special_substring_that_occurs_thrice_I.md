# 2981.找出出现至少三次的最长特殊子字符串 I

题目链接：[2981.找出出现至少三次的最长特殊子字符串 I](https://leetcode.cn/problems/find-longest-special-substring-that-occurs-thrice-i/)

## 题目大意

给你一个仅由小写英文字母组成的字符串 s 

如果一个字符串仅由单一字符组成，那么它被称为 特殊 字符串。例如，字符串 "abc" 不是特殊字符串，而字符串 "ddd"、"zz" 和 "f" 是特殊字符串

返回在 s 中出现 至少三次 的 最长特殊子字符串 的长度，如果不存在出现至少三次的特殊子字符串，则返回 -1 

子字符串 是字符串中的一个连续 非空 字符序列

```js
Example 1:
Input: s = "aaaa"
Output: 2
Explanation: The longest special substring which occurs thrice is "aa": substrings "aaaa", "aaaa", and "aaaa".
It can be shown that the maximum length achievable is 2.

Example 2:
Input: s = "abcdef"
Output: -1
Explanation: There exists no special substring which occurs at least thrice. Hence return -1.

Example 3:
Input: s = "abcaba"
Output: 1
Explanation: The longest special substring which occurs thrice is "a": substrings "abcaba", "abcaba", and "abcaba".
It can be shown that the maximum length achievable is 1.
```

限制:
- 3 <= s.length <= 50
- s consists of only lowercase English letters.

## 解题

核心思路: 特殊子串只可能来自某个字符的连续段

例如：s = "aaabbbaaaa"
- 字符 'a' 的连续段长度是：3, 4
- 字符 'b' 的连续段长度是：3

所以只需要统计每个字符的`连续段长度`

**为什么只需要维护前三大连续段？**

对于同一个字符，假设它的前三大连续段长度是：first >= second >= third

答案只可能来自 3 种情况
- 情况 1：三个出现都来自最长段
  - 如果最长连续段长度是 first，那么长度为 L 的特殊子串在这个段里可以出现：first - L + 1 次
  - 要至少出现 3 次：first - L + 1 >= 3
  - 所以：L <= first - 2
  - 对应候选答案：first - 2
- 情况 2：两个出现来自最长段，一个来自第二长段
  - 最长段至少贡献 2 次，所以：L <= first - 1
  - 第二长段至少贡献 1 次，所以：L <= second
  - 因此候选答案是：min(first - 1, second)
- 情况 3：三个出现分别来自前三个连续段
  - 三个连续段各贡献一次，那么长度最大就是第三大连续段：third
  
**算法步骤**
- 用 top3[26][3] 记录每个字符的前三大连续段长度
- 扫描字符串，统计当前连续段长度 cnt
- 每当一个连续段结束时，把 cnt 插入到对应字符的前三大长度中
- 最后对每个字符计算三种候选答案：
  - first - 2
  - min(first - 1, second)
  - third
  - 取最大值
- 如果最大值为 0，说明不存在答案，返回 -1

```js
/**
 * @param {string} s
 * @return {number}
 */
var maximumLength = function(s) {
    const top3 = Array.from({ length: 26 }, () => [0, 0, 0]);

    const n = s.length;
    let cnt = 0;

    for (let i = 0; i < n; i++) {
        cnt++;

        if (i === n - 1 || s[i] !== s[i + 1]) {
            const idx = s.charCodeAt(i) - 'a'.charCodeAt(0);

            if (cnt > top3[idx][0]) {
                top3[idx][2] = top3[idx][1];
                top3[idx][1] = top3[idx][0];
                top3[idx][0] = cnt;
            } else if (cnt > top3[idx][1]) {
                top3[idx][2] = top3[idx][1];
                top3[idx][1] = cnt;
            } else if (cnt > top3[idx][2]) {
                top3[idx][2] = cnt;
            }

            cnt = 0;
        }
    }

    let res = 0;

    for (const [first, second, third] of top3) {
        res = Math.max(
            res,
            first - 2,
            Math.min(first - 1, second),
            third
        );
    }

    return res > 0 ? res : -1;
};
```
```python
class Solution:
    def maximumLength(self, s: str) -> int:
        # top3[i] stores the top 3 run lengths for character chr(i + ord('a'))
        top3 = [[0, 0, 0] for _ in range(26)]

        n = len(s)
        cnt = 0

        for i, ch in enumerate(s):
            cnt += 1

            # End of a consecutive group
            if i == n - 1 or s[i] != s[i + 1]:
                idx = ord(ch) - ord('a')

                # Insert cnt into top3[idx]
                if cnt > top3[idx][0]:
                    top3[idx][2] = top3[idx][1]
                    top3[idx][1] = top3[idx][0]
                    top3[idx][0] = cnt
                elif cnt > top3[idx][1]:
                    top3[idx][2] = top3[idx][1]
                    top3[idx][1] = cnt
                elif cnt > top3[idx][2]:
                    top3[idx][2] = cnt

                cnt = 0

        res = 0

        for first, second, third in top3:
            res = max(
                res,
                first - 2,
                min(first - 1, second),
                third
            )

        return res if res > 0 else -1
```
```java
class Solution {
    public int maximumLength(String s) {
        int[][] top3 = new int[26][3];

        int n = s.length();
        int cnt = 0;

        for (int i = 0; i < n; i++) {
            cnt++;

            if (i == n - 1 || s.charAt(i) != s.charAt(i + 1)) {
                int idx = s.charAt(i) - 'a';

                if (cnt > top3[idx][0]) {
                    top3[idx][2] = top3[idx][1];
                    top3[idx][1] = top3[idx][0];
                    top3[idx][0] = cnt;
                } else if (cnt > top3[idx][1]) {
                    top3[idx][2] = top3[idx][1];
                    top3[idx][1] = cnt;
                } else if (cnt > top3[idx][2]) {
                    top3[idx][2] = cnt;
                }

                cnt = 0;
            }
        }

        int res = 0;

        for (int i = 0; i < 26; i++) {
            int first = top3[i][0];
            int second = top3[i][1];
            int third = top3[i][2];

            res = Math.max(res, first - 2);
            res = Math.max(res, Math.min(first - 1, second));
            res = Math.max(res, third);
        }

        return res > 0 ? res : -1;
    }
}
```

- 时间复杂度：`O(n)`，设 `n = len(s)`
  - 扫描字符串一次：`O(n)`
  - 每个连续段更新 top3：`O(1)`
  - 最后遍历 `26` 个字符：`O(26)`，可视为 `O(1)`
  - 所以总时间复杂度是：`O(n)`
- 空间复杂度：`O(1)`
  - 因为 `top3` 固定是：`26 * 3`
  - 只和字符集大小有关，不随 `n` 增长
  - 所以空间复杂度是：`O(1)`
