# 205.Isomorphic Strings

难度: Easy
标签: String、HashMap

![205](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.1ow3q1fb9v.webp)

限制：
- 1 <= s.length <= 5 * 10^4
- t.length == s.length
- s and t consist of any valid ascii character.

## 解题

### 暴力

对于每一对位置 i 和 j，如果 s[i] == s[j]，则 t[i] 必须等于 t[j]

同样，如果 t[i] == t[j]，则 s[i] 必须等于 s[j]

这检查两个字符串是否具有相同的字符模式

```python
class Solution:
    def isIsomorphic(self, s: str, t: str) -> bool:
        if len(s) != len(t):
            return False

        n = len(s)

        for i in range(n):
            for j in range(n):
                if s[i] == s[j] and t[i] != t[j]:
                    return False

                if t[i] == t[j] and s[i] != s[j]:
                    return False

        return True
```

n = s.length()

- 时间复杂度: `O(n²)`
- 空间复杂度: `O(1)`

暴力搜索法会比较每一对位置

例如：

s = "paper"
t = "title"

它会反复检查相同的字符是否出现在不同的位置

这是不必要的

### 最优解：HashMap

**我们需要维护什么？**

在从左向右扫描的过程中，需要知道：
- 字符串 s 中的该字符是否已经映射到了某个字符？
- 字符串 t 中的该字符是否已经由某个字符映射而来？

因此，使用：
- 从 s 的字符到 t 的字符的映射
- 从 t 的字符到 s 的字符的映射

核心思想:
- 同时扫描两个字符串
- 在每个索引位置：sChar = s[i] / tChar = t[i]
- 检查以下两个条件：
  - 如果 sChar 之前已建立映射，它必须映射到 tChar
  - 如果 tChar 之前已建立映射，它必须映射回 sChar
- 如果任一条件不满足，则返回 false
- 否则，保存该映射关系

处理完索引 `i` 之后：从 `s[0...i]` 到 `t[0...i]` 的所有映射关系均保持一致且为一一对应

```python
class Solution:
    def isIsomorphic(self, s: str, t: str) -> bool:
        if len(s) != len(t):
            return False
        
        sTot = {}
        tTos = {}
        for ch_s, ch_t in zip(s, t):
            if ch_s in sTot and sTot[ch_s] != ch_t:
                return False
            
            if ch_t in tTos and tTos[ch_t] != ch_s:
                return False
            
            sTot[ch_s] = ch_t
            tTos[ch_t] = ch_s
        
        return True
```
```java
class Solution {
    public boolean isIsomorphic(String s, String t) {
        if (s.length() != t.length()) {
            return false;
        }

        Map<Character, Character> sTot = new HashMap<>();
        Map<Character, Character> tTos = new HashMap<>();
        for (int i = 0; i < s.length(); i++) {
            char sChar = s.charAt(i);
            char tChar = t.charAt(i);

            if (sTot.containsKey(sChar) && sTot.get(sChar) != tChar) {
                return false;
            }

            if (tTos.containsKey(tChar) && tTos.get(tChar) != sChar) {
                return false;
            }

            sTot.put(sChar, tChar);
            tTos.put(tChar, sChar);
        }

        return true;
    }
}
```

- 时间复杂度：`O(n)`，n = s.length()
- 空间复杂度：`O(k)`，其中 k 为不同字符的数量

更一般地，如果字符集固定为 ASCII：`O(1)`，因为字符的最大数量是有限的
