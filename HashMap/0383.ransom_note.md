# 383.Ransom Note

难度: Easy
标签: String、HashMap

![383](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.9ddmrvdfvu.webp)

限制：
- 1 <= nums.length <= 10^4
- -10^5 <= nums[i] <= 10^5
- 0 <= left <= right < nums.length
- At most 10^4 calls will be made to sumRange.

## 解题

### 暴力

对于赎金备注中的每个字符，在杂志中查找对应的未使用字符

如果找到，则将该字符标记为已使用

如果未找到，则返回 false

```python
class Solution:
    def canConstruct(self, ransomNote: str, magazine: str) -> bool:
        used = [False] * len(magazine)

        for ch in ransomNote:
            found = False

            for i in range(len(magazine)):
                if not used[i] and magazine[i] == ch:
                    used[i] = True
                    found = True
                    break

            if not found:
                return False

        return True
```

令：m = ransomNote.length(), n = magazine.length()

- 时间复杂度：`O(m * n)`
- 空间复杂度：`O(n)`

因为使用了一个已用数组来存储杂志上的内容

对于 `ransomNote` 中的每个字符，如果采用暴力法，就需要重新扫描 `magazine`

示例：

ransomNote = "aaa"
magazine = "aaab"

对于每一个 'a'，我们都要反复在 `magazine` 中进行查找，这其实是不必要的

### 最优解：HashMap

**需要维护什么信息呢？**

我们只需要知道：`magazine` 中每个字符各出现了多少次？

然后，对于 `ransomNote` 中的每个字符，我们消耗掉一个可用的计数

因此，可以使用：`HashMap / Counter` 或者，如果输入仅包含小写英文字母：`int[26]`

常见的两种最优解法如下：
- HashMap / 计数器 (Counter)
- 固定大小的频率数组

针对这题，由于字符通常为小写英文字母，因此使用频率数组的解法最为简洁且高效

核心思路
- 统计 `magazine` 中每个字符的出现次数
- 遍历 `ransomNote`
- 对于每个字符，将其对应的计数减 1
- 若任一计数变为负数，则返回 `false`
- 若所有字符均能成功匹配（即计数未降至负数），则返回 `true`
  

在遍历 `ransomNote` 的过程中：`count[ch]` 表示 `magazine` 中尚可使用的字符 `ch` 的剩余数量。若：`count[ch] < 0` 则说明 `ransomNote` 所需的字符 `ch` 数量超过了 `magazine` 提供的数量

```python
# HashMap Version
class Solution:
    def canConstruct(self, ransomNote: str, magazine: str) -> bool:
        if len(ransomNote) > len(magazine):
            return False

        magazine_count = Counter(magazine)
        for ch in ransomNote:
            if ch not in magazine_count or magazine_count[ch] <= 0:
                return False
            magazine_count[ch] -= 1
        
        return True
        
# Frequency Array Version
class Solution:
    def canConstruct(self, ransomNote: str, magazine: str) -> bool:
        count = [0] * 26

        for ch in magazine:
            index = ord(ch) - ord('a')
            count[index] += 1

        for ch in ransomNote:
            index = ord(ch) - ord('a')
            count[index] -= 1

            if count[index] < 0:
                return False

        return True
```
```java
// HashMap
class Solution {
    public boolean canConstruct(String ransomNote, String magazine) {
        if (ransomNote.length() > magazine.length()) {
            return false;
        }
        Map<Character, Integer> magazineCount = new HashMap<>();
        for (int i = 0; i < magazine.length(); i++) {
            char ch = magazine.charAt(i);
            magazineCount.put(ch, magazineCount.getOrDefault(ch, 0) + 1);
        }
        for (int i = 0; i < ransomNote.length(); i ++) {
            char ch = ransomNote.charAt(i);
            if (!magazineCount.containsKey(ch) || magazineCount.get(ch) <= 0) {
                return false;
            }

            magazineCount.put(ch, magazineCount.get(ch) - 1);
        }
        return true;
    }
}

// Frequency Array Version
class Solution {
    public boolean canConstruct(String ransomNote, String magazine) {
        int[] count = new int[26];

        for (int i = 0; i < magazine.length(); i++) {
            char ch = magazine.charAt(i);
            int index = ch - 'a';
            count[index]++;
        }

        for (int i = 0; i < ransomNote.length(); i++) {
            char ch = ransomNote.charAt(i);
            int index = ch - 'a';
            count[index]--;

            if (count[index] < 0) {
                return false;
            }
        }

        return true;
    }
}
```

哈希表版本
- 时间复杂度：`O(m + n)`
- 空间复杂度：`O(k)`, 其中 `k` 为唯一字符的数量

频率数组版本
- 时间复杂度：`O(m + n)`
- 空间复杂度：`O(1)`，因为数组大小固定为 `26`
