# 88.Merge Sorted Array

难度：Easy  
标签: Array、Two Pointers

![88](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.7axtvjck05.webp)

Constraints:
- nums1.length == m + n
- nums2.length == n
- 0 <= m, n <= 200
- 1 <= m + n <= 200
- -10^9 <= nums1[i], nums2[j] <= 10^9

## 解题

### Brute Force
> `双指针 + 额外数组 []`
> - 定义临时数组 `sorted = []`，指针 `p1` 指向 `nums1`，`p2` 指向 `nums2`
> - 比较两个元素，将较小者加入 `sorted`
> - 将剩余元素加入 `sorted`
> - 将 `sorted` 复制回 `nums1`
>
> 时间复杂度：`O(m+n)`，空间复杂度：`O(m+n)`
> - 遍历两个数组：`O(m+n)`
> - 复制回原数组：`O(m+n)`
> - `sorted` 数组存储了全部 `m+n` 个元素，空间占用为 `O(m+n)`
>
> 瓶颈：
> - 该解法不包含冗余计算，时间复杂度已达到最优
> - 额外的结果数组需要 `O(m+n)` 的空间

```python
class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        sorted = []
        p1 = 0
        p2 = 0

        while p1 < m and p2 < n:
            if nums1[p1] <= nums2[p2]:
                sorted.append(nums1[p1])
                p1 += 1
            else:
                sorted.append(nums2[p2])
                p2 += 1

        while p1 < m:
            sorted.append(nums1[p1])
            p1 += 1

        while p2 < n:
            sorted.append(nums2[p2])
            p2 += 1

        nums1[:] = sorted
```

### 最优解
> - 无需创建新数组，可以直接从右向左填充 `nums1`
> - `nums1` 末尾已预留了足够的空间
> - 由于两个数组均已排序，最大元素位于末尾；我们可以从两个数组的末尾开始比较，将最大值放置在 `nums1` 的末尾
> - 这样可以避免覆盖 `nums1` 中尚未处理的元素

**解法**
> - 使用 3 个指针：
>   - `p1 = m - 1` -> 指向 `nums1` 的最后一个有效元素
>   - `p2 = n - 1` -> 指向 `nums2` 的最后一个元素
>   - `k = m + n - 1` -> 指向 `nums1` 的最后一个位置
> - 每次比较 `nums1[p1]` 和 `nums2[p2]`，将较大者放入 `nums1[k]`，并移动相应的指针
> - 移动 `k` 指针
> - 重复上述步骤，直到处理完 `nums2`
>
> 为什么只检查 `p2 >= 0`?
> - 如果 `nums2` 已经处理完，`nums1` 剩余元素本来就在正确位置
> - 如果 `nums1` 先处理完，仍然需要把 `nums2` 剩余元素复制进来
> 
> 两个数组未处理部分的最大元素一定是：`max(nums1[p1], nums2[p2])`，把它放到当前最后的空位 `k`，这个位置必然正确，不断重复，最终整个数组有序

```python
class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        # p1 指向 nums1 最后一个有效元素
        # p2 指向 nums2 最后一个元素
        # k 指向 nums1 最后一个可以写入的位置
        p1, p2, k = m-1, n-1, m+n-1

        # 若两个数组都有元素
        while p1 >= 0 and p2 >= 0:
            if nums1[p1] > nums2[p2]:
                nums1[k] = nums1[p1]
                p1 -= 1
            else:
                nums1[k] = nums2[p2]
                p2 -= 1
            
            k -= 1
        
        # 判断 nums2, 如果 `nums1` 先处理完，仍然需要把 `nums2` 剩余元素复制进来
        while p2 >= 0:
            nums1[k] = nums2[p2]
            p2 -= 1
            k -= 1
```
```java
class Solution {
    public void merge(int[] nums1, int m, int[] nums2, int n) {
        int p1 = m-1;
        int p2 = n-1;
        int k = m+n - 1;
        while (p1 >= 0 && p2 >= 0) {
            if (nums1[p1] > nums2[p2]) {
                nums1[k] = nums1[p1];
                p1 --;
            } else {
                nums1[k] = nums2[p2];
                p2 --;
            }
            k --;
        }
        while (p2 >= 0) {
            nums1[k] = nums2[p2];
            p2 -= 1;
            k -= 1;
        }
    }
}
```

**Complexity**
> - 时间复杂度：`O(m + n)`，最坏情况下，每个元素最多被处理一次
> - 空间复杂度：`O(1)`，仅需使用 3 个指针
