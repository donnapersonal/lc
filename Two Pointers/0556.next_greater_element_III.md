# 556.下一个更大元素 III

题目链接：[556.下一个更大元素 III](https://leetcode.cn/problems/next-greater-element-iii/)

## 题目大意

给你一个正整数 n ，请你找出符合条件的最小整数，其由重新排列 n 中存在的每位数字组成，并且其值大于 n 。如果不存在这样的正整数，则返回 -1 

注意 ，返回的整数应当是一个 32 位整数 ，如果存在满足题意的答案，但不是 32 位整数 ，同样返回 -1 

```js
Example 1:
Input: n = 12
Output: 21

Example 2:
Input: n = 21
Output: -1
```

限制：
- 1 <= n <= 2^31 - 1

## 解题

把 n 转换成字符串（字符数组），那么本题实际上是在求字符数组的 31. 下一个排列，当不存在下一个排列时返回 −1

```js
/**
 * @param {number} n
 * @return {number}
 */
var nextGreaterElement = function(n) {
    const digits = String(n).split('');

    // Step 1: find the first digit from right that is smaller than its next digit
    let i = digits.length - 2;
    while (i >= 0 && digits[i] >= digits[i + 1]) {
        i--;
    }

    // If no such digit exists, digits are in descending order
    if (i < 0) {
        return -1;
    }

    // Step 2: find the smallest digit on the right that is greater than digits[i]
    let j = digits.length - 1;
    while (digits[j] <= digits[i]) {
        j--;
    }

    // Step 3: swap them
    [digits[i], digits[j]] = [digits[j], digits[i]];

    // Step 4: reverse the suffix to make it the smallest possible
    let left = i + 1;
    let right = digits.length - 1;

    while (left < right) {
        [digits[left], digits[right]] = [digits[right], digits[left]];
        left++;
        right--;
    }

    const res = Number(digits.join(''));

    return res <= 2147483647 ? res : -1;
};
```

```python
class Solution:
    def nextGreaterElement(self, n: int) -> int:
        digits = list(str(n))

        # Step 1: find the first digit from right that is smaller than its next digit
        i = len(digits) - 2

            i -= 1

        # If no such digit exists, the digits are in descending order
        if i < 0:
            return -1

        # Step 2: find the smallest digit on the right that is greater than digits[i]
        j = len(digits) - 1
        while digits[j] <= digits[i]:
            j -= 1

        # Step 3: swap them
        digits[i], digits[j] = digits[j], digits[i]

        # Step 4: reverse the suffix to make it the smallest possible
        digits[i + 1:] = reversed(digits[i + 1:])

        res = int(''.join(digits))

        return res if res <= 2 ** 31 - 1 else -1
```

```java
class Solution {
    public int nextGreaterElement(int n) {
        char[] digits = String.valueOf(n).toCharArray();

        // Step 1: find the first digit from right that is smaller than its next digit
        int i = digits.length - 2;
        while (i >= 0 && digits[i] >= digits[i + 1]) {
            i--;
        }

        // If no such digit exists, digits are in descending order
        if (i < 0) {
            return -1;
        }

        // Step 2: find the smallest digit on the right that is greater than digits[i]
        int j = digits.length - 1;
        while (digits[j] <= digits[i]) {
            j--;
        }

        // Step 3: swap them
        swap(digits, i, j);

        // Step 4: reverse the suffix to make it the smallest possible
        reverse(digits, i + 1, digits.length - 1);

        long res = Long.parseLong(new String(digits));

        return res <= Integer.MAX_VALUE ? (int) res : -1;
    }

    private void swap(char[] digits, int i, int j) {
        char temp = digits[i];
        digits[i] = digits[j];
        digits[j] = temp;
    }

    private void reverse(char[] digits, int left, int right) {
        while (left < right) {
            swap(digits, left, right);
            left++;
            right--;
        }
    }
}
```
