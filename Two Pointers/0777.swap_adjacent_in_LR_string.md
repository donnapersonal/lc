# 777.在 LR 字符串中交换相邻字符

题目链接：[777.在 LR 字符串中交换相邻字符](https://leetcode.cn/problems/swap-adjacent-in-lr-string/)

## 题目大意

在一个由 'L' , 'R' 和 'X' 三个字符组成的字符串（例如"RXXLRXRXL"）中进行移动操作。一次移动操作指用一个 "LX" 替换一个 "XL"，或者用一个 "XR" 替换一个 "RX"。现给定起始字符串 start 和结束字符串 result，请编写代码，当且仅当存在一系列移动操作使得 start 可以转换成 result 时，返回 True

```js
Example 1:
Input: start = "RXXLRXRXL", result = "XRLXXRRLX"
Output: true
Explanation: We can transform start to result following these steps:
RXXLRXRXL ->
XRXLRXRXL ->
XRLXRXRXL ->
XRLXXRRXL ->
XRLXXRRLX

Example 2:
Input: start = "X", result = "L"
Output: false
```

限制：
- 1 <= start.length <= 10^4
- start.length == result.length
- Both start and result will only consist of characters in 'L', 'R', and 'X'.

## 解题

**双指针**

根据题意，我们每次移动要么是将 `XL` 变为 `LX`，要么是将 `RX` 变为 `XR`，而该两者操作可分别看做将 `L` 越过多个 `X` 向左移动，将 `R` 越过多个 `X` 向右移动

因此在 `start` 和 `end` 中序号相同的 `L` 和 `R` 必然满足坐标性质：
- 序号相同的 `L`: `start` 的下标不小于 `end` 的下标（即 `L` 不能往右移动）
- 序号相同的 `R`: `start` 的下标不大于 `end` 的下标（即 `R` 不能往左移动）
  
其中「序号」是指在 `LR` 字符串中出现的相对顺序

```js
/**
 * @param {string} start
 * @param {string} result
 * @return {boolean}
 */
var canTransform = function(start, end) {
    // Safety check: if lengths differ, transformation is impossible.
    if (start.length !== end.length) {
        return false;
    }

    const n = start.length;
    let i = 0;
    let j = 0;

    while (i < n || j < n) {
        // Skip X in start because X is just an empty position.
        while (i < n && start[i] === 'X') {
            i++;
        }

        // Skip X in end for the same reason.
        while (j < n && end[j] === 'X') {
            j++;
        }

        // If one string has no more L/R characters,
        // the other one must also have no more L/R characters.
        if (i === n || j === n) {
            return i === n && j === n;
        }

        // The current non-X characters must be the same.
        // Otherwise, the relative order of L and R changed.
        if (start[i] !== end[j]) {
            return false;
        }

        // L can only move left.
        // If L starts at index i and ends at index j,
        // then j must be <= i.
        if (start[i] === 'L' && i < j) {
            return false;
        }

        // R can only move right.
        // If R starts at index i and ends at index j,
        // then j must be >= i.
        if (start[i] === 'R' && i > j) {
            return false;
        }

        // Move to the next non-X character.
        i++;
        j++;
    }

    return true;
```

```java
class Solution {
    public boolean canTransform(String start, String end) {
        if (start.length() != end.length()) {
            return false;
        }

        int n = start.length();
        int i = 0;
        int j = 0;

        while (i < n || j < n) {
            // Skip X in start.
            while (i < n && start.charAt(i) == 'X') {
                i++;
            }

            // Skip X in end.
            while (j < n && end.charAt(j) == 'X') {
                j++;
            }

            // If one string has no more non-X characters,
            // the other one must also be finished.
            if (i == n || j == n) {
                return i == n && j == n;
            }

            char c1 = start.charAt(i);
            char c2 = end.charAt(j);

            // Non-X characters must appear in the same relative order.
            if (c1 != c2) {
                return false;
            }

            // L can only move left.
            // Therefore its target index j cannot be greater than its start index i.
            if (c1 == 'L' && i < j) {
                return false;
            }

            // R can only move right.
            // Therefore its target index j cannot be smaller than its start index i.
            if (c1 == 'R' && i > j) {
                return false;
            }

            i++;
            j++;
        }

        return true;
    }
}
```
