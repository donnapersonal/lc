# 1861.旋转盒子

题目链接：[1861.旋转盒子](https://leetcode.cn/problems/rotating-the-box/)

## 题目大意

给你一个 m x n 的字符矩阵 boxGrid ，它表示一个箱子的侧视图。箱子的每一个格子可能为：

- '#' 表示石头
- '*' 表示固定的障碍物
- '.' 表示空位置
  
这个箱子被 顺时针旋转 90 度 ，由于重力原因，部分石头的位置会发生改变。每个石头会垂直掉落，直到它遇到障碍物，另一个石头或者箱子的底部。重力 不会 影响障碍物的位置，同时箱子旋转不会产生惯性 ，也就是说石头的水平位置不会发生改变

题目保证初始时 boxGrid 中的石头要么在一个障碍物上，要么在另一个石头上，要么在箱子的底部

请你返回一个 n x m 的矩阵，表示按照上述旋转后，箱子内的结果

![example2](https://github.com/donnapersonal/picx-images-hosting/raw/master/image.6bhpb51lej.webp)

限制：
- m == boxGrid.length, n == boxGrid[i].length
- 1 <= m, n <= 500
- boxGrid[i][j] is either '#', '*', or '.'.

## 解题

对于每一行 row，倒着遍历，我们可以直接确定每个石头落入的位置：

- 如果 row[j] 是障碍物，那么它左边最近的石头，在旋转后掉落到 row[j−1]。我们用一个变量 k 维护石头掉落后的位置，如果 row[j] 是障碍物，那么更新 k=j−1。注：如果 row[j] 左边最近的不是石头而是障碍物，那么 k 会继续更新，无需担心石头落到错误的位置
- 如果 row[j] 是石头，那么它掉落到 row[k]。然后把 k 减一，表示左边下一块石头掉落后的位置

```js
/**
 * @param {character[][]} boxGrid
 * @return {character[][]}
 */
var rotateTheBox = function(boxGrid) {
    const m = boxGrid.length, n = boxGrid[0].length;
    const res = Array.from({ length: n }, () => Array(m).fill('.'));

    for (let i = 0; i < m; i++) {
        const row = boxGrid[i];
        let k = n - 1;
        for (let j = n - 1; j >= 0; j--) {
            if (row[j] === '*') { // 障碍物
                res[j][m - 1 - i] = '*';
                k = j - 1; // 障碍物左边最近的石头，在旋转后掉落到 j-1
            } else if (row[j] === '#') { // 石头
                res[k][m - 1 - i] = '#'; // 旋转后，石头掉落到 k
                k--;
            }
        }
    }

    return res;
};
```

class Solution:
    def rotateTheBox(self, boxGrid: list[list[str]]) -> list[list[str]]:
        m, n = len(boxGrid), len(boxGrid[0])
        res = [['.'] * m for _ in range(n)]

        for i, row in enumerate(boxGrid):
            k = n - 1
            for j in range(n - 1, -1, -1):
                if row[j] == '*':  # 障碍物
                    res[j][-1 - i] = '*'
                    k = j - 1  # 障碍物左边最近的石头，在旋转后掉落到 j-1
                elif row[j] == '#':  # 石头
                    res[k][-1 - i] = '#'  # 旋转后，石头掉落到 k
                    k -= 1

        return res

```java
class Solution {
    public char[][] rotateTheBox(char[][] boxGrid) {
        int m = boxGrid.length;
        int n = boxGrid[0].length;
        char[][] res = new char[n][m];
        for (char[] row : res) {
            Arrays.fill(row, '.');
        }

        for (int i = 0; i < m; i++) {
            char[] row = boxGrid[i];
            int k = n - 1;
            for (int j = n - 1; j >= 0; j--) {
                if (row[j] == '*') { // 障碍物
                    res[j][m - 1 - i] = '*';
                    k = j - 1; // 障碍物左边最近的石头，在旋转后掉落到 j-1
                } else if (row[j] == '#') { // 石头
                    res[k][m - 1 - i] = '#'; // 旋转后，石头掉落到 k
                    k--;
                }
            }
        }

        return res;
    }
}
```

- 时间复杂度：`O(mn)`，其中 `m` 和 `n` 分别是 `boxGrid` 的行数和列数
- 空间复杂度：`O(1)`，返回值不计入
